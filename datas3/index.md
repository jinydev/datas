# 빅데이터 분석 프로그래밍 (Big Data Analysis Programming)

안녕하세요. **빅데이터 분석 프로그래밍** 강좌에 오신 것을 환영합니다.
본 강좌는 데이터 분석의 기초부터 시작하여, 파이썬을 활용한 데이터 시각화, 수치 해석, 그리고 머신러닝의 기초까지 다루는 종합 과정입니다.

## 1. 강의 목표
- 파이썬 프로그래밍 언어를 이해하고, 데이터 분석을 위한 기초 문법을 습득합니다.
- **Matplotlib**을 활용하여 데이터를 시각적으로 표현하는 능력을 기릅니다.
- **Numpy**와 **Pandas**를 사용하여 대용량 데이터를 효율적으로 처리하고 분석합니다.
- **Scikit-learn**을 통해 머신러닝 모델을 학습하고 평가하는 분석 프로젝트를 수행합니다.

## 2. 강의 특징
- **시각화 중심 학습 (Visual Learning)**: 이론보다는 눈에 보이는 그래프와 도표를 먼저 학습하여 흥미를 유발합니다.
- **게임화 요소 (Gamification)**: 추상적인 숫자가 아닌, 게임 캐릭터나 아이템 확률 등의 예제를 활용하여 쉽게 이해합니다.
- **실습 위주 (Hands-on)**: "백문이 불여일견", 코드를 직접 실행하며 데이터를 다루는 법을 익힙니다.
