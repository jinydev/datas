### 데이터프레임 합 계산

```python
# 사고(건)    2433

# 부상(명)    4899
```

`df.sum(axis=0)`로 축 `axis=0`에 따라 이동(위에서 아래로 수직으로)하며 합을 구한다. 코드 `df.sum()`과 같으며 모든 열의 합을 구할 수 있다.

### 데이터프레임 조건 검색 (Boolean Indexing)

데이터프레임 원소의 비교 논리 값은 다음과 같이 반환받을 수 있다. 원소 값이 10 초과인 자료가 있는 부분은 `True`가 표시된다.

```python
df2 > 10
# (결과 생략: 10보다 큰 값은 True, 아니면 False인 데이터프레임 반환)
```

위의 불리언 인덱싱 조건을 `df2[조건]` 내부에 기술하면 `True`인 원소만 표시되고 `False`인 원소는 `NaN`이 표시된다.

```python
df2[df2 > 10]
```
| | 사고(건) | 사망(명) | 부상(명) | 재해자수 |
| :--- | :--- | :--- | :--- | :--- |
| **2016년1월** | 192.0 | NaN | 387.0 | 392.0 |
| **2016년2월** | 174.0 | NaN | 328.0 | 334.0 |
| ... | ... | ... | ... | ... |
| **총계** | 2433.0 | 98.0 | 4899.0 | 4997.0 |

다음 `df['사망(명)'] == 7`은 열 이름 '사망(명)'이 7인 논리 값의 시리즈가 반환된다.

```python
df['사망(명)'] == 7
# 2016년1월      False
# 2016년2월      False
# 2016년3월      True
# ...
# Name: 사망(명), dtype: bool
```

함수 `Series.isin([value0, value1, value2, ...])`은 시리즈의 각 요소가 리스트 값에 포함되는지 확인하여 논리 값 시리즈를 반환한다.
다음 `df['사망(명)'].isin([7])`은 열 이름 '사망(명)'이 7인 논리 값의 시리즈가 반환된다.

```python
df['사망(명)'].isin([7])
# 2016년1월      False
# ...
# 2016년3월      True
# ...
# Name: 사망(명), dtype: bool
```

```python
df.query('A == "D clmns"')
```

다음의 그리고 연산자 `&`를 사용할 수 있다.

```python

```

다음의 `in` 연산자도 사용할 수 있다.

```python
df.query("E in ['python', 'go']")
```

`df.query()` 내부의 질의 문자열 내부에서 변수를 사용하려면 `@변수명`을 사용한다.

`df.query()` 내부의 질의 문자열에 f-string 사용이 가능하다. 그러므로 문자열 내부에서 변수 `num`을 `{num}`으로 사용이 가능하다.
