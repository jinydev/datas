### pydataset 소개

#### 패키지 pydataset 개요

`pydataset`은 파이썬에서 사용할 수 있는 예제 데이터셋(dataset)을 쉽게 메모리에 저장하고 활용할 수 있도록 제공되는 패키지이다. 이 패키지는 다양한 테스트용 데이터셋을 포함하고 있어, 데이터 분석이나 시각화 연습 등 다양한 용도로 활용될 수 있다. 이러한 데이터셋을 장난감 데이터(toy dataset)라고도 부른다.

`pydataset`의 주요 특징은 다음과 같다.

*   **다양한 데이터셋 제공**: `pydataset` 패키지는 R 패키지인 `datasets`에 포함된 다양한 데이터셋을 파이썬에서 사용할 수 있도록 변환한 것으로, 빠르게 샘플 데이터를 활용할 수 있다.
*   **간편한 데이터 로드**: 패키지를 설치하고 로드한 후, `data()` 함수를 사용하여 데이터셋을 쉽게 로드할 수 있다.
*   **표준화된 데이터 형식**: 로드한 데이터는 `pandas`의 DataFrame 형태로 반환되어 데이터 분석 및 조작이 용이하다.

![Image](img/파이썬으로_배우는_데이터분석입문8_1.jpeg)

`pydataset` 라이브러리를 사용하면 CSV 파일이나 `pandas`의 `read_csv` 메소드를 사용하여 JSON, Excel과 같은 다른 파일을 읽을 필요가 없어, 데이터 분석, 시각화 및 기계 학습을 더 쉽게 탐색할 수 있는 장점이 있다.

#### 데이터프레임의 간편한 데이터 시각화

패키지 `seaborn`을 활용해 간편히 데이터 시각화가 가능하다. 다음은 `sns.barplot()`을 사용해 세로 막대그래프를 그릴 수 있다.

```python
import seaborn as sns
import matplotlib.pyplot as plt
plt.rcParams['font.family'] = 'Malgun Gothic'
plt.rcParams['axes.unicode_minus'] = False

sns.barplot(traffic)
plt.show()
```

다음 `traffic[['사망자수(명)']].T` 코드는 '사망자수(명)'의 행과 열을 바꾼 데이터프레임을 반환한다. 위를 `sns.barplot(traffic[['사망자수(명)']].T)`의 인자로 사용하면 '사망자수(명)'를 시도 지역별로 막대그래프를 그릴 수 있다.

![Image](img/파이썬으로_배우는_데이터분석입문8_25.jpeg)

```python
import seaborn as sns
import matplotlib.pyplot as plt

sns.barplot(traffic[['사망자수(명)']].T)
plt.show()
```

다음 `sns.barplot(traffic[[traffic.columns[0]]].T)` 코드는 `traffic.columns[0]`인 '사고건수(건)'의 행과 열을 바꿔 '사고건수(건)'를 시도 지역별로 막대그래프를 그릴 수 있다.

