### 1.1 구글 코랩 개요             



#### Colab 개요 

구글 코랩(Colab)은 구글이 제공하는 클라우드 기반의 주피터 노트북( notebook) 환경을 말합니다.

> 주피터 노트북이란?

주로 파이썬을 기반으로 한 데이터 분석, 머신러닝, WAY 등의 작업을 수행할 때 그 진가를 발휘한다. 

코랩은 브라우저에서 실행되며, 무료로 사용할 수 있는 가상 머 신을 제공하여 사용자가 별도로 서버를 구축하지 않고도 고성능의 하드웨어 자원을 활용할 수 있게 해준다. 

물론, 고성능 하드웨어를 오랜 기간 사용하려면 유료이나 단기간 사용한다면 무료 이다.



다음은 코랩의 주요 특징과 장점이다.

- 무료 GPU 지원: DHS Pee 사용할 수 있는 GPUS 제공하여 Bald 모델 학습 등에 유용

- 클라우드 기반    = 작업은 구글의 클라우드 서버에서 이루어지기 때문에 사용자의 로컬 환경에 구애 를 받지 않고 작업 수행 가능 

- 주피터 노트북 통합: DHS 주피터 LEST} 호환되어 있어 코드와 문서를 함께 작성하고 시각화

- 파이썬 라이브러리 지원: DHS 주요 데이터 과학 및 머신러닝 라이브러리들을 기본적으로 지원하며, 필요 시 추가 라이브러리 설치 가능

- 구글 드라이브 연동: 구글 드라이브와 연동되어 노트북 및 데이터를 손쉽게 저장하고 공유 가능 

- 다중 언어 지원: 주로 파이썬을 사용하지만, 다양한 언어를 지원하며 특정 셀에서 다른 언어 실행 가능

- 데이터 시각화: Matplotlib, Seabor, Plotly 등 다양한 시각화 라이브러리를 활용하여 데이터를 시각적 으로 분석 가능



코랩을 사용하면 사용자는 개인적인 환경설정이나 고성능 하드웨어의 필요성에 구애받지 않고, 즉시 클라우드 기반의 머신러닝 또는 데이터 분석 프로젝트를 시작할 수 있는 장점으로 활용도 가 높다.

### 2.1 비주얼 스튜디오 코드 개요 

#### vs code 개요 

비주얼 스튜디오 코드(visual Studio Code, 이하 vs code)는 마이크로소프트에서 제공 하는 대부분의 프로그래밍 언어를 위한 개발환경을 지원하고 가볍고 확장 가능한 통합개발 환경(IDE)이다. 

> IDE란?



주로 다양한 소프트웨어 개발환경으로 사용되며, 많은 언어와 프레임워크 (framework)를 지원한다.

전문 텍스트 에디터로 2016년에 정식판이 처음으로 발표된 이후, 매우 빠른 버전 개발로 현재 호평받는 범용 코드 편집기이자 통합개발환경으로 발전되었다.



![image-20260212215436433](/Users/hojin8/Library/Application Support/typora-user-images/image-20260212215436433.png)








다음은 vs code의 주요 특징 및 장점이다.



무료 및 오픈 소스: vs code는 무료로 사용할 수 있으며 오픈소스로 개발되어 커뮤니티에서 지속적 으로 발전

+ 가볍고 빠른 실행: 다른 대부분의 125에 비해 가벼우면서 빠르게 실행
+ 다양한 언어 지원: 다양한 프로그래밍 언어를 지원하며, 확장 기능을 통해 필요한 의 지원 추가 가능 
+ 인텔리센스(18066066): 코드 작성 중 자동 완성 및 코드 제안을 제공하여 개발 생산성을 높일 수 있음
+ 통합 터미널: 내장된 터미널을 통해 프로젝트를 관리하고 명령을 실행할 수 있음 
+ Git 통합: 내장된 Git 지원으로 AA 코드의 버전 관리 및 협업이 용이
+ 테마 및 아이콘 설정: 다양한 테마와 아이콘을 지원하여 개발자의 취향에 맞게 환경을 설정할 수 있음
+ 확장성: 확장 기능을 통해 플러그인을 추가하고, 필요한 도구나 기능을 손쉽게 확장할 수 있음
+ 통합 디버깅: 내장된 디버깅 기능으로 코드를 효과적으로 디버깅할 수 있음
+ 다중 플랫폼 지원: Windows, macOs, Linux 등 다양한 플랫폼에서 동작하며, 개발 환경을 효율적으로 이관할 수 있음 
+ 데이터 사이언스 및 머신러닝용 기능에 대한 지원도 강력하며, 주피터 노트북 클라이언트를 자체 내장 하고 있어 바로 이용할 수 있으며, 변수탐색기등 편리한 기능을 다수 제공 



개발자들은 자신의 환경에 맞게 플러그인과 설정을 추가하여 보다 효율적인 작업을 수행할 수 있다.
