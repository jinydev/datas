**매개변수:**
*   `data`: ndarray(구조적 또는 동종), iterable, dict 또는 DataFrame
    *   Dict에는 Series, 배열, 상수, 데이터 클래스 또는 목록과 같은 객체가 포함될 수 있다.
    *   데이터가 dict인 경우, 열 순서는 삽입 순서를 따른다.
*   `index`: 인덱스 또는 배열과 유사한 자료
    *   결과 프레임에 사용할 인덱스. 입력 데이터에 인덱스 정보가 없고 제공된 인덱스가 없는 경우 기본적으로 `RangeIndex`가 사용된다.
*   `columns`: 인덱스 또는 배열과 유사한 자료
    *   데이터에 열 레이블이 없을 때 결과 프레임에 사용할 열 레이블. 기본값은 `RangeIndex(0, 1, 2, ..., n)`.
*   `dtype`: 데이터 유형
    *   강제할 데이터 유형. 단일 dtype만 허용됨. None인 경우 추론함.
*   `copy`: bool 또는 None
    *   입력에서 데이터를 복사할지 여부.

| **강수희** | 18 | 27 | 10 | 20 |
| **유한빈** | 12 | 22 | 9 | 20 |

다음으로 행 슬라이싱한 데이터프레임을 반환받을 수 있다.
```python
pf.iloc[2:]
```
| | 중간 | 기말 | 과제 | 출석 |
| :--- | :--- | :--- | :--- | :--- |
| **홍소희** | 17 | 17 | 10 | 19 |
| **유한빈** | 12 | 22 | 9 | 20 |
| **신수빈** | 22 | 34 | 8 | 16 |

### 열 참조

앞 부분을 `:`로 하고 뒤에 열 첨자로 열만 참조할 수 있다.
```python
pf.iloc[:, 2]
# 윤일형     8
# 강수희    10
# 홍소희    10
# 유한빈     9
# 신수빈     8
# Name: 과제, dtype: int64
```

다음 `[]` 내부에 `[2]`으로 첨자 2열로 구성된 데이터프레임을 반환받는다.
```python
pf.iloc[:, [2]]
```
| | 과제 |
| :--- | :--- |
| **윤일형** | 8 |
| **강수희** | 10 |
| **홍소희** | 10 |
| **유한빈** | 9 |
| **신수빈** | 8 |

모든 행에 대해 열을 슬라이싱한 결과이다.
```python
pf.iloc[:, 1:]
```
| | 기말 | 과제 | 출석 |
| :--- | :--- | :--- | :--- |
| **윤일형** | 35 | 8 | 18 |
| **강수희** | 27 | 10 | 20 |
| **홍소희** | 17 | 10 | 19 |
| **유한빈** | 22 | 9 | 20 |
| **신수빈** | 34 | 8 | 16 |

### 행과 열 참조

`pf.iloc[1, 1]`으로 첨자 1, 1에 해당하는 값을 참조한다.
```python
pf.iloc[1, 1]
# 27
```

다음은 행과 열을 슬라이싱한 결과이다.
```python
pf.iloc[1:3, 1:]
```
| | 기말 | 과제 | 출석 |
| :--- | :--- | :--- | :--- |
| **강수희** | 27 | 10 | 20 |
| **홍소희** | 17 | 10 | 19 |

다음은 전체 행에 대해 첨자 1열 이후 모든 목록과 열 첨자를 슬라이싱한 결과이다.
```python
pf.iloc[:, 1:]
```
| | 기말 | 과제 | 출석 |
| :--- | :--- | :--- | :--- |
| **윤일형** | 35 | 8 | 18 |
| **강수희** | 27 | 10 | 20 |
| **홍소희** | 17 | 10 | 19 |
| **유한빈** | 22 | 9 | 20 |
| **신수빈** | 34 | 8 | 16 |

다음은 행 첨자 목록과 열 첨자 목록으로 부분 데이터프레임을 반환받는 코드이다.
```python
pf.iloc[[1, 3, 4], 1:]
```

| :--- | :--- | :--- | :--- |

다음은 행 첨자 목록과 열 첨자 목록으로 부분 데이터프레임을 반환받는 코드이다.
```python
pf.iloc[[1, 3, 4], [0, 1]]
```

| | 중간 | 기말 |
| :--- | :--- | :--- |

### `at['행명', '열명']`, `iat[i, j]` 참조

`at`와 `iat`는 `loc` 및 `iloc`과 비슷하게 레이블 기반/정수 인덱스 기반으로 참조하는 방식으로 단 하나의 원소를 참조한다. `at[]`과 `iat[]`은 하나의 값을 참조하므로 속도가 빠른 장점이 있다.

```python
pf.at['윤일형', '과제']
# 8
pf.at['유한빈', '기말']
# 22
```

다음처럼 행 레이블을 `pf.index[i]`로 참조할 수 있다.
```python
pf.at[pf.index[1], '출석']
# 20
```

또한, 다음처럼 열 이름을 `pf.columns[j]`로 참조할 수 있다.
```python
pf.at[pf.index[3], pf.columns[2]]
# 9
```

`iat[i, j]`는 정수 첨자 기반의 참조 방식으로 단 하나의 원소를 참조한다.
```python
pf.iat[1, 2]
# 10
pf.iat[2, 3]
# 19
```
