### 산점도

#### 산점도 개요와 함수 plot()

산점도(산포도, scatter plot)는 두 연속형 변수 좌표상의 점들을 표시해 이 두 변수 사이의 관계를 나타내는 그래프이다.
x좌표와 y좌표 (x, y)의 값이 (3, 7), (5, 10)인 2개의 점을 그리는 간단한 산점도를 그려보자. 첫 번째 인자는 두 점의 x 값만 모은 리스트이며, 두 번째가 y값 리스트인 것에 주의하자.

*   `scatter(x, y, ...)`
    *   인자: `x = [x1, x2, x3, ...]`, `y = [y1, y2, y3, ...]`
    *   좌표 `(x1, y1)`, `(x2, y2)`, `(x3, y3)` ... 점 그리기

다음 코드처럼 함수 `scatter`의 인자로 `[3, 5]`, `[7, 10]` 등을 기술해 실행한다.

`%config InlineBackend.figure_format = 'retina'`는 `matplotlib`의 그림을 선명하게 그리기 위한 설정이다.

```python
import matplotlib.pyplot as plt
# %config InlineBackend.figure_format = 'retina'
plt.scatter([3, 5], [7, 10]) # x좌표와 y좌표가 (3, 7), (5, 10)인 2개의 점 산점도
plt.show()
```

![Image](img/파이썬으로_배우는_데이터분석입문9_3.jpeg)

함수 `scatter(x, y)`는 적어도 인자는 2개여야 한다. 다음 함수 `range()` 등을 사용하면 수의 시퀀스로 간단히 산점도를 그릴 수 있다.

```python
plt.scatter(range(1, 5), range(5, 9))
plt.show()
```
x좌표와 y좌표 (x, y)의 값이 (1, 5), (2, 6), (3, 7), (4, 8)인 4개의 점을 그리는 간단한 산점도이다.

다음은 x와 y 모두 시퀀스 인자를 사용한 코드이다. 5개의 좌표 (3, 13), (4, 14), (5, 15), (6, 16), (7, 17)을 그린 산점도이다.

```python
plt.scatter(range(3, 8), range(13, 18))
plt.show()
```

### 선 그리기

#### 선 그리기 함수 plot()

함수 `plot()`으로는 간단히 지정한 좌표를 지나는 선을 그릴 수 있다. 선 종류는 인자 `linestyle`로 조절한다.
*   `plot([x], y, linestyle='--', ...)`
    *   인자: `linestyle='--'`: 점선(dashed line) 표시

다음은 인자 하나인 y축의 값을 지정한 연결선 그림이다. y축 자료가 6개 이므로 지정하지 않은 x축은 기본으로 0에서 5까지가 된다. 다음 왼쪽 결과에서 선 종류가 기본인 실선이다.

```python
import matplotlib.pyplot as plt
plt.plot(range(10, 16))
plt.show()
```

다음은 x값 -5에서 5까지의 곡선 $y = x^2$을 그린 그림이다. x축에서 이웃한 숫자의 거리를 0.1로 지정해서 그린다. 0.1을 더 작게 하면 곡선이 더 부드러워진다. 실수의 시퀀스를 위해 `numpy`의 `arange()` 함수를 활용한다. 인자 `ls`는 `linestyle`과 같으며 `:`은 점선(dotted line) 이다.

```python
import numpy as np
x = np.arange(-5, 5.1, 0.1)
plt.plot(x, x**2, ls=":")
plt.xlabel('$x$') # 레이텍(LaTex) 형식의 수식 표현
plt.ylabel('$y = x^2$')
plt.show()
```

### 막대 그래프

#### 막대 그래프 함수 bar()와 barh()

막대 그래프는 이산형 자료의 값을 직사각형 막대를 사용하여 데이터를 시각화하는 방법이다.
함수 `bar()`를 사용하여 막대 그래프를 그려보자.

다음 코드에서 점수를 나타내는 막대의 세로 높이를 `point`, 대응되는 점수의 해당 학생 이름을 `name`에 저장한다. 함수 `bar(name, ...)`의 첫 인자는 x축에 표시되는 학생 이름인 `name`이다. 인자 `color`에는 각 막대의 색상을 지정할 수 있다. 다음 결과에서 보듯이 함수 `bar`는 세로 막대 방향으로 그려진다.

```python
import matplotlib.pyplot as plt
point = [99, 65, 79, 80, 90]
name = ["영희", "철수", "윤철", "미현", "현수"]

plt.bar(name, point, color = ["blue", "chartreuse", "purple", "grey", "orange"])
plt.title('데이터분석 점수')
plt.xlabel('이름')
plt.ylabel('점수')
plt.show()
```

![Image](img/파이썬으로_배우는_데이터분석입문9_15.jpeg)

함수 `barh`로 막대 그래프를 가로로 그려보자. 인자 `height=.6`으로 막대의 너비를 조절할 수 있다.

```python
import matplotlib.pyplot as plt
point = [99, 65, 79, 80, 90]
name = ["영희", "철수", "윤철", "미현", "현수"]

plt.barh(name, point, height = .6, color = ["blue", "chartreuse", "purple", "grey", "orange"])
plt.title('데이터분석 점수')
plt.xlabel('점수')
plt.ylabel('이름')
plt.show()
```

다음은 막대 그래프에 빈도수 값을 텍스트로 표시하는 예제이다.
`ax.patches`는 막대 그래프의 막대들을 의미하며, `rect.get_x()`, `rect.get_width()`, `rect.get_height()`를 통해 막대의 위치와 높이(빈도수)를 알아낼 수 있다.
이를 이용해 `ax.text()`로 막대 위에 빈도수를 표시한다.

```python
ax = sns.countplot(data=mtc, x='gear')

# ax.patches는 막대 그래프의 막대들을 담고 있는 리스트
for rect in ax.patches:
    # (x, y) 좌표 계산: 막대의 중심 x좌표, 막대의 높이 y좌표
    x = rect.get_x() + rect.get_width() / 2
    y = rect.get_height()
    
    # 텍스트 표시: y + 0.1 위치에 높이(빈도수)를 문자열로 표시
    # ha='center': 수평 중앙 정렬
    ax.text(x, y + 0.1, str(int(y)), ha='center')

plt.ylim(0, 18) # y축 범위 설정
plt.title("기어(gear) 빈도수")
plt.show()
```

![Image](img/파이썬으로_배우는_데이터분석입문9_21.jpeg)

다음 코드에서 객체 `ax.patches`는 3개의 직사각형 정보인 `Rectangle` 객체를 갖고 있다.

또한, `rect.get_height()`이 리스트 `[15, 12, 5]`를 반환한다.

```python

    height = rect.get_height() # 높이 [15, 12, 5]
    ax.text(x = rect.get_x() + rect.get_width() / 2, 
            y = height + 0.1, 
            s = int(height), 
            ha='center', size=15)

ax.set_ylim(0, 18)
plt.title("기어(gear) 빈도수")
plt.show()
```

막대컨테이너(BarContainer) 객체인 변수 `patches`에서 `patches[i].set_facecolor()`로 각 막대의 색상을 지정하면 막대마다 서로 다른 색상을 지정할 수 있다.
`plt.cm.viridis(n[i]/max(n))`을 지정하면 여러 색상을 갖고 있는 색상맵 `viridis`에서 값의 비율 `n[i]/max(n)`에 따라 다른 색상을 반환 받을 수 있다.
다음 코드로 색상을 빈도수에 따라 색상맵 `viridis`를 달리 지정해 `hist()`를 그릴 수 있다. 빈도 수가 작은 값은 보라색으로 큰 값은 노란색이 되는 것을 알 수 있다.

```python
n, bins, patches = plt.hist(iris['Sepal.Length'], bins = 20, color='yellowgreen', edgecolor='white')

for i in range(len(patches)):
    patches[i].set_facecolor(plt.cm.viridis(n[i]/max(n)))

plt.show()
```

상관계수(Correlation coefficient)는 상관관계 분석에서 두 변수 간에 선형 관계의 강도를 계량화한 값이다. 상관계수는 -1에서 1까지의 실수 값으로 다음의 선형관계를 나타낸다.

*   양수 값은 양의 상관관계를 나타내며 두 변수가 함께 증가하는 경향이 있다.
*   음수 값은 음의 상관관계를 나타내며 다른 변수 값이 감소할 때 한 변수 값은 증가하는 경향이 있다.

데이터프레임의 메소드 `corr()`으로 열의 값이 수인 변수 간의 상관관계의 데이터프레임을 얻을 수 있다. 다음 상관관계 계수 결과로 대각선은 1이며 대각선을 기준으로 대칭인 것을 알 수 있다. 꽃잎 너비인 `petal_width`와 꽃잎 길이인 `petal_length`의 상관관계는 0.963으로 관계가 높다는 것을 알 수 있다.

```python
# 상관관계 계수 계산
import pandas as pd
pd.set_option('display.precision', 3)
corr_iris = iris[iris.columns[:-1].to_list()].corr()
corr_iris
```

여러 개의 변수로 만들어진 상관관계 계수를 색깔로 파악할 수 있다. 값에 따라 색깔을 달리하면 변수들의 관계를 쉽게 알 수 있다. `seaborn`의 `heatmap` 함수를 이용하면 상관관계 계수를 히트맵으로 쉽게 만들 수 있다.

```python
sns.heatmap(corr_iris, annot=True)
plt.show()
```

그림 우측의 색상 막대(color bar)에서 보듯이 상관계수가 1에 가까우면 밝아지고 -1에 가까우면 어두워진다.
위 히트맵을 보기 좋게 좌측 하단의 삼각 행렬만 남도록 수정해 보자. 이를 위해 먼저 필요한 마스크(mask)를 만들자. 다음 코드로 `corr_iris`와 같은 모양인 4 x 4가 모두 0으로 채워진 2차원 행렬 `mask`를 생성한다.

```python
# mask 만들기
import numpy as np
mask = np.zeros_like(corr_iris)
mask
```
(배열 출력 생략)

![Image](img/파이썬으로_배우는_데이터분석입문9_49.jpeg)

다음 `numpy` 배열의 오른쪽 위 인덱스(indices)를 구하는 `np.triu_indices_from()`을 활용해 `mask`의 오른쪽 위 대각 행렬을 모두 1로 수정한 `mask`를 생성한다.

```python
# 오른쪽 위 대각 부분을 1로 바꾸기
mask[np.triu_indices_from(mask)] = 1
mask
```

상관계수로 만든 히트맵에서 대각선은 상관계수가 항상 1이 되며 오른쪽 위는 왼쪽 아래와 값이 같다. 그러므로 이 부분을 모두 제거하려면 `mask`와 상관행렬의 첫 번째 행과 마지막 열을 제거해야 한다. 이를 위해 `mask[1:, :-1]`로 행과 열에 슬라이싱을 사용하여 배열 `mask`의 일부를 선택한다. 다음 코드로 3행 3열로 줄어든 새로운 마스크 `mask_new`를 만들자.

```python
mask_new = mask[1:, :-1] # mask 첫 번째 행, 마지막 열 제거
# mask_new 출력 생략
```

마찬가지로, 다음 코드로 상관관계 계수 데이터프레임의 첫 번째 행과 마지막 열을 제거한 새로운 데이터프레임 `corr_iris_new`를 만들자.

```python
# 상관관계 계수 dataframe 첫 번째 행, 마지막 열 제거
corr_iris_new = corr_iris.iloc[1:, :-1]
# corr_iris_new 출력 생략
```

이제, 인자 `mask=mask_new`를 사용하고 인자 `cmap='coolwarm'`과 `cbar_kws = {'shrink': .5}` 등 몇 가지를 추가해 히트맵을 보기 좋게 수정하자.

![Image](img/파이썬으로_배우는_데이터분석입문9_50.jpeg)

서로 다른 변수 간의 상관관계 계수를 쉽게 파악할 수 있다. 인자 `cmap='coolwarm'`을 지정해 양의 상관관계는 붉은 색으로 음의 상관관계는 파란 색으로 표시할 수 있다.

```python
sns.heatmap(corr_iris_new, annot=True, cmap='coolwarm', mask=mask_new,
            annot_kws = {"size": 10}, vmin=-1, vmax=1,
            cbar_kws = {'shrink': .5}, # 범례 크기 줄이기
            linewidths = .5)           # 경계 구분선 추가
plt.show()
```

`sns.pairplot(iris, hue='species')`로 품종 `species`에 따른 4개 변수 간의 상관관계 산점도를 시각화할 수 있다. 대각선에는 품종 3개의 밀도 함수를 그려 준다. 꽃잎 너비인 `petal_width`와 꽃잎 길이인 `petal_length`의 밀도 함수를 살펴보면 3개의 품종이 잘 구분되는 것을 볼 수 있다.

```python
sns.pairplot(iris, hue='species')
plt.show()
```

```python
sns.pairplot(iris, hue="species", palette="husl", markers=["o", "s", "D"])
```

상관관계 그림을 좀 더 수정하려면 `sns.pairplot()`의 반환 객체인 `PairGrid`를 사용한다.
반환 객체인 `PairGrid` 객체 `g`를 사용한 다음 `g.map_lower(sns.kdeplot, ...)` 호출로 좌측 하단을 산점도 대신 밀도 추정 그림으로 바꿀 수도 있다.

```python
g = sns.pairplot(iris, hue="species", palette="husl", height=2.0, markers=["o", "s", "D"])
g.map_lower(sns.kdeplot, levels=4, color=".2")
plt.show()
```
