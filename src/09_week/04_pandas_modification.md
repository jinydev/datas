# 9주차 4강: 데이터 수정 및 정렬 (Modification)

> **학습목표**: 데이터프레임에 새로운 열을 추가하거나 기존 값을 수정하고, 원하는 순서대로 정렬하는 방법을 익힙니다.

## 9.4.1. 열(Column) 추가 및 수정


<br>

---

<br>

### 1. 기본 문법: `df['새이름'] = 값`
딕셔너리에 값을 추가하는 것과 똑같습니다.
*   기존에 있는 이름이면 -> **수정 (Update)**
*   없는 이름이면 -> **추가 (Add)**

```python
import pandas as pd

df = pd.DataFrame({'A': [1, 2, 3], 'B': [4, 5, 6]})

# 새로운 열 'Sum' 추가 (A + B)
df['Sum'] = df['A'] + df['B']
print(df)
#    A  B  Sum
# 0  1  4    5
# 1  2  5    7
# 2  3  6    9
```


<br>

---

<br>

### 2. 함수 체이닝: `assign()`
기존 데이터프레임을 건드리지 않고(Immutable), 새로운 열이 추가된 **새 데이터프레임을 반환**합니다.

```python
df_new = df.assign(Multi = df['A'] * df['B'])
```

<br>

---

<br>

## 9.4.2. 데이터 삭제: `drop()`

저번 시간에 잠깐 봤지만, 다시 한번 확실히 짚고 넘어갑시다.

*   **행 삭제**: `axis=0` (기본값) -> 인덱스 이름 지정
*   **열 삭제**: `axis=1` -> 컬럼 이름 지정

> **중요**: `drop`은 원본을 바꾸지 않고 삭제된 복사본을 줍니다.
> 원본을 바로 고치려면 `inplace=True`를 써야 하지만, **실수를 방지하기 위해 덮어쓰기(`df = df.drop(...)`) 방식을 권장합니다.**

```python
# 'Sum' 열 삭제해서 다시 저장
df = df.drop('Sum', axis=1)
```

<br>

---

<br>

## 9.4.3. 데이터 정렬: `sort_values()`

데이터를 크기순으로 나열합니다.

*   `by`: 정렬 기준 컬럼
*   `ascending`: `True`(오름차순, 기본값), `False`(내림차순)

```python
# B열 기준 내림차순 정렬
sorted_df = df.sort_values(by='B', ascending=False)
```


<br>

---

<br>

### 인덱스 정렬: `sort_index()`
값(Value)이 아니라, 행 번호(Index)를 기준으로 다시 정렬하고 싶을 때 씁니다.

```python
# 인덱스가 뒤죽박죽일 때 다시 0, 1, 2... 순서로
final_df = sorted_df.sort_index()
```

## 정리 (Summary)

이 강의에서 배운 핵심 내용을 요약해 봅시다.

*   **[핵심 1]**: `df['컬럼명'] = 값` 형태로 매우 직관적으로 데이터를 추가하거나 수정할 수 있습니다.
*   **[핵심 2]**: `drop()`을 사용할 때는 `axis` 방향(0:행, 1:열)을 잘 확인해야 하며, 원본 덮어쓰기에 쥐의하세요.
*   **[핵심 3]**: `sort_values()`는 값을 기준으로, `sort_index()`는 인덱스를 기준으로 데이터를 정렬합니다.
