# 파이썬으로 배우는 데이터 분석 입문 - Part 6

## 학습목표
- **판다스 및 시리즈 개요**: 데이터 분석의 핵심 도구인 Pandas 라이브러리를 이해하고, 1차원 데이터 구조인 시리즈(Series)의 생성과 인덱싱 기법을 익힌다.
- **데이터프레임의 생성과 활용**: 2차원 표 형태인 데이터프레임(DataFrame)의 구조를 파악하고, 여러 리스트/딕셔너리 자료형과 날짜 범위(`date_range`)를 활용한 객체 생성법을 학습한다.
- **다양한 참조 방식 활용**: 데이터프레임 내에서 열과 행을 자유자재로 다룰 수 있는 라벨 중심 참조(`loc`), 인덱스/위치 번호 중심 참조(`iloc`), 그리고 스칼라 값 고속 추출(`at`, `iat`) 메서드 및 활용 방법을 체득한다.

## SECTION 01. pandas 개요

### [01. 판다스 소개](./01_판다스_소개)
파이썬 데이터 분석의 핵심 모듈인 Pandas에 대해 알아보고, 데이터를 1차원 시리즈와 2차원 데이터프레임으로 표현하는 기초적인 내용을 살펴봅니다.

### [02. 시리즈 개요](./02_시리즈_개요)
데이터의 인덱스와 값(values)을 포함하여 1차원 배열로 이루어진 Pandas `Series` 객체의 구조와 특징을 배웁니다.

### [03. 데이터프레임 개요](./03_데이터프레임_개요)
레이블이 지정된 행과 열 축이 존재하는 대표적인 2차원 테이블 형식의 자료 구조 `DataFrame`의 장점과 지원 기능에 대해 익힙니다.

### [04. Series 생성과 활용](./04_Series_생성과_활용)
ndarray, 리스트 등을 사용해 `Series`를 생성하는 여러 방법을 살펴보고, 내부 속성을 통해 다양한 방식의 인덱싱 및 슬라이싱 기법을 다룹니다.

### [05. Series 활용 함수](./05_Series_활용_함수)
시리즈의 값을 기준으로 오름차순 또는 내림차순 정렬하거나, 기초 통계 데이터 및 빈도수(count, value_counts 등)를 집계하는 활용 방법을 알아봅니다.

## SECTION 02. 데이터프레임 생성

Pandas의 `DataFrame()` 클래스를 사용할 때 내부적으로 어떠한 인자(data, index, columns, dtype)가 입력될 수 있는지 파악합니다.

### [07. 2차원 행렬의 테이블 데이터 사용](./07_2차원_행렬의_테이블_데이터_사용)
이중 리스트나 Numpy의 2차원 난수 행렬 배열 자료 등을 `DataFrame`으로 곧바로 변환하는 기본 생성법을 배웁니다.

### [08. 딕셔너리의 테이블 데이터 사용](./08_딕셔너리의_테이블_데이터_사용)
파이썬 딕셔너리(`dict`) 형태의 데이터들을 `DataFrame`으로 변환하며 각 Key가 열(Column)로 변환되는 과정을 이해합니다.

### [09. 여러 Series로 DataFrame 생성](./09_여러_Series로_DataFrame_생성)

### [10. 여러 딕셔너리의 목록 데이터 사용](./10_여러_딕셔너리의_목록_데이터_사용)
딕셔너리들을 원소로 담은 파이썬 목록을 바로 DataFrame의 각 행 데이터로 입력하는 실무적인 사용법을 파악합니다.

### [11. dict의 값으로 Series 사용](./11_dict의_값으로_Series_사용)
Series와 dict를 조합할 때, 결측값 처리 원리를 포함해 Series의 Index가 DataFrame의 행 이름(레이블)과 어떻게 매핑되는지 이해합니다.

### [12. 열과 전체 자료형 속성 dtypes](./12_열과_전체_자료형_속성_dtypes)
DataFrame의 전체 데이터들이 메모리상에 어떤 판다스/Numpy 자료형태로 올라와 있는지 확인하는 `dtypes` 속성을 알아봅니다.

### [13. 여러 날짜 생성을 위한 함수 date_range()](./13_여러_날짜_생성을_위한_함수_date_range)
`date_range()` 함수를 활용하여 시계열 예측과 분석에서 빈번히 사용되는 규칙적인 간격(일, 주, 월 등)의 DatetimeIndex를 매우 간단하게 만들어 내는 기법을 배웁니다.

### [14. 날짜를 포함한 데이터프레임](./14_날짜를_포함한_데이터프레임)
`date_range()`로 계산해둔 일자 목록을 데이터프레임의 인덱스로 곧바로 활용하여, 날짜 기반의 테이블 데이터를 생성합니다.

## SECTION 03. DataFrame의 다양한 참조

### [15. 열 참조](./15_열_참조)
생성된 데이터프레임에서 이름이 부착된 단일 열(`Series` 형태)이나 복수 열(`DataFrame` 형태)을 쉽고 간편하게 추출하여 가져오는 방법을 연습합니다.

### [16. 조건 참조](./16_조건_참조)
파이썬 논리 및 관계 연산자(Boolean Indexing)를 사용해 데이터를 조건부로 필터링하고 걸러내는 고급 데이터 분석의 뼈대를 익힙니다.

### [17. loc[index]로 행 참조](./17_loc_index로_행_참조)
문자열 레이블(Index) 이름 자체로 원하는 데이터의 행 범위를 참조하거나 반환받을 수 있도록 해주는 `loc` 메서드를 집중적으로 알아봅니다.

### [18. loc[index[i]]로 행 참조](./18_loc_index_i로_행_참조)
직접 레이블 문자열을 입력하지 않고 `index` 배열의 위치를 혼합 활용하여 `loc`의 행 접근성을 극대화하는 방법을 배웁니다.

### [19. 열 참조 (loc 사용)](./19_열_참조_loc)
`loc`의 행 조건문을 전체 `:`로 설정하고 오로지 지정한 문자열 열(Column) 레이블 이름으로 슬라이싱하여 다수 열의 데이터를 참조합니다.

### [20. loc[:, columns[i]]로 열 참조](./20_loc_columns_i로_열_참조)
직접적인 문자 이름 대신 DataFrame의 내부 속성인 `columns` 배열을 통해 순번으로 접근 가능한 `loc`의 추가 활용 방안을 학습합니다.

### [21. 행과 열 참조](./21_행과_열_참조)
행 슬라이스 조건과 열 슬라이스 조건을 `loc[]`에 구별 자릿수로 함께 넣어 2차원의 원하는 사각형 부분집합 데이터만을 단번에 참조해 옵니다.

### [22. df.iloc[첨자] 참조](./22_df_iloc_첨자_참조)
레이블(이름) 규칙과는 무관하게 데이터프레임을 파이썬 내부 배열의 메모리상 위치 기준(순번 정수 인덱스 첨자)으로 접근해 다루는 `iloc`와 빠르고 가벼운 단일 값 추출 기법 `at`, `iat`을 소개합니다.
