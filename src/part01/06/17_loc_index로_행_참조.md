## 03. loc[index]로 행 참조

`pf.loc['윤일형']`으로 '윤일형' 행을 참조할 수 있다. 반환 값은 `Series`이다.

```python
pf.loc['윤일형']
```

다음 코드 `pf.loc[['윤일형']]`으로 리스트로 만들면 '윤일형' 행의 데이터프레임을 참조할 수 참조할 수 있다. 반환 값은 데이터프레임이다.

```python
pf.loc[['윤일형']]
```

행 명의 목록으로 여러 행을 데이터프레임으로 반환받을 수 있다.

```python
pf.loc[['홍소희', '윤일형']]
```

다음은 슬라이싱으로 끝이 포함된 범위의 여러 행을 여러 행을 참조할 수 있다.

```python
pf.loc['윤일형':'유한빈']
```

데이터프레임 `pf[조건]`으로 논리 인덱싱이 가능하다. 다음 코드는 '중간'이 20 초과인 행을 가진 데이터프레임을 데이터프레임을 반환한다.

```python
pf.loc[pf.중간 > 20]
```

다음 코드로는 '중간'이 10 초과이며 '과제'가 10인 행을 가진 데이터프레임을 반환받는다.

```python
pf.loc[(pf.중간 > 10) & (pf.과제 == 10)]
```
