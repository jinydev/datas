## 03. 튜플

### 튜플 개요와 활용

파이썬의 튜플(tuple)은 리스트와 유사하지만, 한 번 생성되면 변경할 수 없는(immutable) 시퀀스 자료형이다. 튜플은 괄호 `( )`로 표현되며, 각 요소는 쉼표 `,`로 구분된다. 튜플은 다음과 같은 특징을 갖는다.

- **불변성(immutable)**: 한 번 생성된 튜플은 수정할 수 없기 때문에, 데이터 무결성을 보장하고 의도치 않은 변경을 방지
- **성능**: 리스트보다 메모리 사용이 적고, 읽기 속도가 빠름
- **해시 가능성(Hashable)**: 튜플은 딕셔너리의 키나 집합의 원소로 사용될 수 있으며, 이를 통해 유용한 데이터 구조를 만들 수 있음

괄호를 둘러싸며 생략할 수 있다.

```python
tp1 = 1, 2, 3
tp2 = (10, 20, 30)
print(tp1, tp2)
```
**출력:**
```
(1, 2, 3) (10, 20, 30)
```

```python
tp3 = tuple((10, 20, 30))
tp4 = tuple(['py', 'R'])
print(tp3, tp4)
```
**출력:**
```
(10, 20, 30) ('py', 'R')
```

튜플은 불변성을 가지고 있기 때문에, 요소의 추가, 삭제, 수정 등의 연산은 지원되지 않는다. 하지만 튜플 간의 연결이나 반복 등은 가능하다.

```python
tuple1 = (1, 2, 3)
tuple2 = (4, 5, 6)
concatenated_tuple = tuple1 + tuple2
concatenated_tuple
```
**출력:**
```
(1, 2, 3, 4, 5, 6)
```
