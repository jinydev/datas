## 03. 변수 이름 조건

프로그래밍 언어에서 프로그래머가 직접 만드는 변수나 함수 이름을 식별자(identifier)라 한다. 파이썬의 식별자는 다음 문자로 구성해야 한다. 영어 알파벳, 숫자, 밑줄(`_`)을 사용하며 알파벳 대문자와 소문자는 구별된다. 첫 문자는 반드시 알파벳이나 밑줄(`_`)로 시작해야 하며 첫 문자로 숫자는 올 수 없다.

마침표 `.`를 사용할 수 없다. 객체지향 언어인 파이썬은 마침표 `.`를 연산자로 사용하기 때문이다. 또한, 문법에 사용되는 예약어인 키워드는 사용할 수 없다.

- 영어 알파벳, 숫자, `_`(밑줄)을 사용
- 첫 글자는 반드시 알파벳이나 밑줄(`_`)로 시작
- 다양한 특수 문자(`%`, `$`, `#` 등)를 사용할 수 없음
- 문법에 사용하는 `if`, `while` 등의 키워드는 사용할 수 없음

다음은 적합한 변수인 `age`, `my_age`, `_height`에 정수나 실수를 저장하는 코드이다.

```python
# var name
age = 19
age

my_age = 20
my_age

_height = 172.3
_height
```

또한, 첫 밑줄 `_` 이후에 바로 숫자가 나와도 문제없다.

```python
_26asian = 2026 # 첫 _ 이후에 바로 숫자 사용 가능
_26asian
```
**출력:**
```
2026
```

변수 이름 `26worldcup`과 같이 숫자가 이름의 가장 앞에 위치할 수 없다. 또한, `my.age`와 같이 마침표 `.`은 식별자에 사용할 수 없다. 모두 에러가 발생한다.

```python
# 잘못된 변수
26worldcup = 2026 # 수가 맨 먼저 위치하면 에러
my.age = 21       # .은 식별자에 사용하면 에러
```

공백 문자(whitespace)나 `$`, `&`, `#` 등의 다른 특수 문자도 변수에 사용할 수 없다.

```python
my credits = 18   # 공백 문자 사용하면 에러
```
