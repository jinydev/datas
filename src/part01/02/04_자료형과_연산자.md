## 04. 자료형과 연산자

### 기본 자료형

프로그래밍 언어에서 자료의 형태를 자료형(data types)이라 한다. 파이썬에서 사용하는 다양한 기본 자료형은 다음과 같다.

| 자료형  | 설명                            | 예시                | 비고                         |
| ------- | ------------------------------- | ------------------- | ---------------------------- |
| float   | 실수                            | -3.89, 4.26         |                              |
| int     | 정수                            | 10, 15, -5          |                              |
| complex | 복소수                          | 3 - 4j, -10 + 7j    | 허수부에서 `j` 사용          |
| str     | 문자 하나 또는 여러 개의 문자열 | 'Python', "VS Code" | 큰 따옴표나 작은 따옴표 가능 |
| bool    | 논리 자료                       | True, False         | 대소문자 주의                |

`a`에 정수나 실수를 저장해 값과 자료형을 확인하는 코드이다. 함수 `print(a)`로도 변수 `a`에 저장된 값을 출력할 수 있다.

```python
# int
a = 10
print(a)
```
**출력:**
```
10
```

함수 `type(a)`로 변수 `a`의 자료형 `int`를 확인할 수 있다. 파이썬의 모든 자료형은 클래스이므로 `<class 'int'>`로 표시된다.

```python
type(a)
```
**출력:**
```
<class 'int'>
```

변수 `x`에 실수 2.718을 저장해 값과 자료형 `float`을 확인하는 코드이다.

```python
# float
x = 2.718
print(x)
type(x)
```
**출력:**
```
2.718
<class 'float'>
```

다음은 복소수를 변수 `c`에 저장해 그 내용 값과 자료형 `complex`를 출력하는 코드이다.

```python
# complex
c = 3 - 4j
print(c)
type(c)
```
**출력:**
```
(3-4j)
<class 'complex'>
```

파이썬에서 문자 하나 또는 문자 여러 개 모인 것을 문자열(string) 자료형이다. 다음은 변수 `s`에 문자를 저장해 그 내용 값과 자료형 `str`을 출력하는 코드이다.

```python
# string
s = "VS Code"
print(s)
type(s)
```
**출력:**
```
VS Code
<class 'str'>
```

다음은 참과 거짓을 표현하는 `True`, `False`의 논리형을 변수 `b`에 저장해 그 내용 값과 자료형 `bool`을 출력하는 코드이다. 파이썬의 `True`, `False`는 대소문자가 다르다. 파이썬에서 `true`로, `false`로 줄여 쓸 수 없다.

```python
# logical
b = True
print(b)
type(b)
```
**출력:**
```
True
<class 'bool'>
```

### 파이썬의 날짜 자료형

파이썬에서 날짜 정보는 기본 자료형이 아니다. 파이썬은 일반적인 프로그래밍 언어로 추가적으로 날짜 정보를 지원한다. 코딩을 위해 먼저 모듈 `datetime`을 메모리에 올려(load)야 한다. 이를 위해 구문 `import datetime as dt`를 사용한다. 여기서 `dt`를 별명(alias)이라 부르며 모듈의 이름 `datetime`을 짧게 `dt`로 만들어서 코문을 간결하게 만들기 위해서 사용한다.

함수 `dt.datetime.now()`를 사용해 시스템의 현재 시간 정보를 생성해 정보를 사용할 수 있다. 다음의 날짜 정보 자료형은 모듈 `datetime`의 클래스 `datetime`인 `<class 'datetime.datetime'>`인 것을 알 수 있다. 객체지향 언어인 파이썬은 마침표 `.`을 '~에 속해 있는'의 연산자로 사용하므로 식별자의 문자로 사용할 수 없다.

- `dt.datetime.now()`: 모듈 `dt`에 속해 있는 클래스 `datetime`의 메소드 `now()`를 호출

```python
import datetime as dt

c = dt.datetime.now() # datetime.now()는 현재 시스템 시간을 반환한다.
print(c.year, c.month, c.day)
type(c)
```
**출력 (예시):**
```
2025 5 3
<class 'datetime.datetime'>
```

`2026-03-01` 날짜 객체를 생성자 `dt.datetime(2026, 3, 1)`를 사용하여 지정한 날짜와 시간 객체를 생성할 수 있다. 변수 `d`에 저장된 날짜와 시간을 출력하고, 그 날짜의 요일을 `d.weekday()`로 출력한다. 메서드 `weekday()`는 날짜가 무슨 요일인지를 숫자로 반환한다. 0은 월요일, 1은 화요일, ..., 6은 일요일에 해당한다. 다음 결과로 2026년 3월 1일이 일요일인 것을 알 수 있다.

```python
import datetime as dt

d = dt.datetime(2026, 3, 1)
print(d, '요일:', d.weekday())
```
**출력:**
```
2026-03-01 00:00:00 요일: 6
```
