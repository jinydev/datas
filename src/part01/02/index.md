# 파이썬으로 배우는 데이터 분석 입문 - Part 2

문서가 각각의 주제별로 분리되었습니다. 아래의 링크를 클릭하여 해당 학습 내용으로 이동하세요.

## 학습목표
- **기초 문법 구사**: 파이썬의 핵심 기본 문법(변수, 자료형, 연산자)을 이해하고 활용할 수 있다.
- **함수와 모듈 활용**: 내장 함수와 `math` 모듈을 이용해 수학 연산 및 필수 기능을 수행한다.
- **분석 도구 입문**: 데이터 분석과 시각화를 위한 핵심 패키지(`pandas`, `numpy`, `matplotlib`, `seaborn`)의 개념과 설치법을 익힌다.
- **흐름 제어**: 조건문(`if`, `match`)과 반복문(`for`, `while`)을 바탕으로 논리적인 제어 흐름을 설계할 수 있다.
- **자료 구조 이해**: 리스트, 딕셔너리, 튜플, 세트의 특성과 메서드들을 숙지하고 목적에 맞게 사용할 수 있다.

## SECTION 01. 파이썬 언어 기초

### [01. 주석과 상수](./01_주석과_상수.md)
파이썬의 기본 문법인 주석(Comment) 작성법과 정수, 실수 등 다양한 상수(Literal) 값의 개념을 알아봅니다. 
파이썬에서 코드를 설명하는 방법론을 익힙니다.

### [02. 변수와 대입](./02_변수와_대입.md)
데이터를 임시로 저장하는 변수에 값을 담는 대입 연산자 `=`를 사용하는 방법을 배웁니다. 
더불어 여러 변수에 한 번에 값을 넣거나 교환하는 다중 대입 기능을 살펴봅니다.

### [03. 변수 이름 조건](./03_변수_이름_조건.md)
파이썬에서 변수나 함수 등 식별자(Identifier) 이름을 지을 때 지켜야 할 규칙과 제약사항을 정리합니다. 
알아보기 쉽고 문법에 맞는 변수명 짓는 팁을 다룹니다.

### [04. 자료형과 연산자](./04_자료형과_연산자.md)
정수, 실수, 논리, 문자열, 복소수 등 파이썬이 제공하는 기본 자료형들을 살펴봅니다. 
데이터 타입의 종류를 파악하고 `datetime`을 활용한 시간 처리 방법도 함께 알아봅니다.

### [05. 다양한 연산자](./05_다양한_연산자.md)
프로그램에서 값을 계산하기 위해 쓰이는 사칙연산, 몫, 나머지 등 연산자의 사용법을 학습합니다. 
또한 관계 연산과 두 값을 비교/조합하는 논리 연산의 원리를 배웁니다.

### [06. 내장 함수 활용](./06_내장_함수_활용.md)
파이썬 설치 시 기본으로 제공되는 `print`, `input`, `max`, `min` 등 유용한 내장 함수들의 기능을 파악합니다. 
별도 패키지 없이도 강력한 기능을 구현하는 법을 익힙니다.

### [07. 내장 모듈 math 활용](./07_내장_모듈_math_활용.md)
수학적인 연산이 필요할 때 기본 탑재된 `math` 모듈을 불러들여 사용하는 방법을 배웁니다. 
제곱근, 올림/내림, 삼각함수, 상수(파이, 자연상수) 등을 계산하는 방법을 학습합니다.

### [08. 패키지 판다스와 시각화 라이브러리 설치](./08_패키지_판다스와_시각화_라이브러리_설치.md)
데이터 분석에서 가장 많이 활용되는 핵심 라이브러리인 '판다스(Pandas)'와 배열 처리 특화 패키지 '넘파이(Numpy)'의 역할에 대해 설명합니다. 
시각화를 위한 준비 단계로써 기본 라이브러리 설치법을 짚어봅니다.

### [09. 패키지 matplotlib, seaborn, pydataset 개요와 설치](./09_패키지_matplotlib_seaborn_pydataset_개요와_설치.md)
데이터를 차트와 그래프로 시각화하는 `matplotlib`과 `seaborn` 라이브러리의 설치/구동 방법을 알아봅니다. 
추가적으로 학습용 토이 데이터셋 제공 패키지인 `pydataset`을 소개합니다.

### [10. 토이 데이터와 데이터프레임 활용](./10_토이_데이터와_데이터프레임_활용.md)
`pydataset` 패키지에서 제공하는 샘플 데이터를 불러와 데이터프레임(DataFrame) 형태로 조작하는 법을 익힙니다. 
실제 코드에 시각화 라이브러리를 결합해 분포를 차트로 그리는 연습을 진행합니다.

## SECTION 02. 제어 흐름과 함수

### [11. 조건문 if](./11_조건문_if.md)
특정 조건일 때만 코드를 실행하게 하는 `if`, `elif`, `else` 구문의 작성법을 학습합니다. 
상황에 따른 분기 및 한 줄 조건식을 활용해 제어 흐름을 구현하는 방법을 익힙니다.

### [12. match 구문](./12_match_구문.md)
최신 파이썬에 도입된 패턴 매칭 구문인 `match` 문의 사용법을 살펴봅니다. 
자바나 C 언어 등의 `switch` 문과 비슷한 이 구문으로 어떻게 다중 조건을 처리하는지 배웁니다.

### [13. 반복문 for와 while](./13_반복문_for와_while.md)
리스트 등의 요소들을 차례로 순회하며 동작을 취하는 `for` 문과 특정 조건이 참인 동안 반복하는 `while` 문의 구조를 이해합니다. 
이를 활용해 반복적인 작업을 효율적으로 프로그래밍합니다.

### [14. 사용자 정의 함수](./14_사용자_정의_함수.md)
특정한 기능을 수행하는 코드를 하나로 묶어 `def` 키워드로 나만의 함수를 만드는 과정을 배웁니다. 
매개변수와 반환값 세팅, 람다(Lambda) 익명 함수 개념까지 종합적으로 알아봅니다.

## SECTION 03. 표준 파이썬 기본 자료 구조

### [15. 리스트](./15_리스트.md)
여러 데이터를 연속적으로 묶어서 관리하는 파이썬의 대표적 자료구조인 리스트(List)의 생성 기술을 다룹니다. 
항목 조작 및 '리스트 컴프리헨션(List Comprehension)' 기법을 학습합니다.

### [16. 사전](./16_사전.md)
키(Key)와 값(Value)의 쌍으로 이루어진 사전(Dictionary) 자료형의 특징과 생성법을 익힙니다. 
데이터 조직화 및 빠른 검색에 유용한 딕셔너리의 각종 메서드를 활용하는 방법을 다룹니다.

### [17. 튜플](./17_튜플.md)
리스트와 유사하지만 한 번 생성된 후에는 값을 변경(수정/삭제)할 수 없는 특성을 지닌 튜플(Tuple) 자료구조를 설명합니다. 
이 튜플의 불변성이 주는 안전성과 그 활용도를 알아봅니다.

### [18. 집합](./18_집합.md)
중복된 원소가 없고 순서가 없는 값을 다룰 수 있는 집합(Set) 구조를 배웁니다. 
합집합, 교집합, 차집합 등의 논리 연산을 수월하게 처리하여 고유한 데이터를 안전하게 다루는 법을 배웁니다.

### [19. 수정 불가능(immutable)과 함수 hash() id()](./19_수정_불가능immutable과_함수_hash_id.md)
파이썬에 존재하는 객체들의 불변(Immutable) 특성과 딕셔너리 키로 사용될 수 있는 해시(Hash) 속성의 원리를 이해합니다. 
더불어 데이터의 고유 식별값을 반환하는 `id()` 함수의 역할을 학습합니다.
