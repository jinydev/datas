## 02. 변수와 대입

다음 코드에서 다양하게 변하는 값을 저장하는 변수(variable) `a`에 값 3을 저장해 출력한다. 변수에 값을 저장하려면 대입 연산자 `=`을 사용한다.

```python
# variable
a = 3 # assign value to variable a
a
```
**출력:**
```
3
```

파이썬은 다른 언어에서는 지원하지 않는 다중 대입을 지원한다. 대입연산자 `=` 왼쪽에 `m, n`처럼 저장할 변수를 쉼표로 구분해 나열한다. 오른쪽에는 저장할 값을 마찬가지로 나열한다. 즉, `m, n = 10, 20`으로 `m` 변수에 10을 할당하고 `n` 변수에 20을 할당한다. 그러므로 두 변수는 각각 값 10과 20을 갖게 된다. 함수 `print(m, n)`으로 변수 `m`과 `n`의 값을 빈 칸을 띄어 출력한다.

```python
m, n = 10, 20
print(m, n)
```
**출력:**
```
10 20
```

다음 코드 `m, n = n, m`으로 `m`과 `n`의 값을 서로 교환한다. 우측은 튜플 `(n, m)`의 값 `(20, 10)`이 되고, 이 값은 왼쪽 변수 `(m, n)`에 할당된다. 따라서 `m` 변수에는 20이, `n` 변수에는 10이 저장된다.

```python
m, n = n, m
print(m, n)
```
**출력:**
```
20 10
```
