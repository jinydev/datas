## 04. 사용자 정의 함수

특정한 기능을 수행하는 함수를 직접 정의해 보자. 파이썬에서 사용자 정의 함수를 선언하려면 키워드 `def`를 사용한 다음 구문을 활용한다.

```python
def function_name(parameters):
    function_body
    return value
```

- `function_name`: 이는 함수 정의 후 해당 함수를 호출하는 데 사용되는 함수 개체의 이름
- `parameters`: 매개변수(parameters)라 부르며 함수를 호출할 때마다 괄호 안에 배치되고 쉼표로 구분 된 함수 정의의 입력을 인수(arguments)라고 불리는 실제 값으로 설정
- `function_body`: 함수 구현 부분으로 함수의 기능을 수행, 함수 몸체라 부름
- `return value`: 옵션으로 함수를 수행한 후 반환되는 값

함수 정의에서 항상 함수이름과 매개변수 이후에는 콜론 `:`이 반드시 나와야 하며 다음은 들여쓰기로 사용해 함수 내용을 구현해야 한다. 다음은 매개변수가 없는 함수 `hello`의 선언 구문이다.

```python
def hello():
    print('Hello, Python!')
```

선언된 함수 `hello`는 `hello()`로 호출해 실행할 수 있다.

```python
hello()
```
**출력:**
```
Hello, Python!
```

함수 이름 이후 괄호가 없으면 단지 메모리에 저장된 함수 객체를 참조한다. 다음 16진수 값은 객체가 저장된 메모리 주소이다.

```python
hello
```
**출력 (예시):**
```
<function hello at 0x000001C014EE1DA0>
```

다음 코드는 `hello`의 자료형이 `function`임을 확인할 수 있다.

```python
type(hello)
```
**출력:**
```
<class 'function'>
```

다음 함수 `hi` 선언은 매개변수 `one`이 하나 있는 함수 선언이다.

```python
def hi(one):
    print("안녕,", one)
```

다음 `hi('영희')`로 함수를 호출해 실행한다. 반드시 인자가 필요하다.

```python
hi('영희')
```
**출력:**
```
안녕, 영희
```

### 함수 매개변수 기본 값

함수 선언시 매개변수에 값을 대입하면 인자 없이 함수를 호출할 수 있다. 함수 선언에서 인자에 초기 값을 저장하는 경우, 반드시 `one = '친구들'`처럼 `=` 연산자를 사용해야 한다.

```python
def hi(one = '친구들'):
    print("안녕,", one)
```

인자없이 다음 함수 `hi()` 호출로 기본 인자 값으로 함수가 실행된다.

```python
hi()
```
**출력:**
```
안녕, 친구들
```

다음 함수 `hi('철수')` 호출로 기본 인자 값은 무시되고 인자로 지정된 철수로 함수가 실행된다.

```python
hi('철수')
```
**출력:**
```
안녕, 철수
```

다음 함수 `circle_area`는 매개변수 기본 값으로 반지름 `r`을 1로 지정해 원의 면적을 구하는 함수이다.

```python
import math as m

def circle_area(r = 1):
    print(m.pi * r**2)
```

인자 없이 함수 `circle_area()` 호출은 반지름 기본 값 1인 원의 면적을 구한다.

```python
circle_area()
```
**출력:**
```
3.141592653589793
```

다음은 반지름을 4.5로 지정해 원 면적을 구한다.

```python
circle_area(4.5)
```
**출력:**
```
63.61725123519331
```

함수 호출 시, 다음처럼 인자 이름을 직접 지정하는 키워드 인자 지정 방식인 `r = 5.6`을 사용할 수 있다.

```python
circle_area(r = 5.6)
```
**출력:**
```
98.5203456165759
```

다음처럼 `return` 문장이 없는 함수를 호출해 그 결과를 출력하면 `None`이 출력되는 것을 볼 수 있다.

```python
print(circle_area(r = 7.2))
```
**출력:**
```
162.8601631620949 # 함수 몸체에서 출력한 결과가 표시
None
```

### return 구문과 함수 매개변수 지정

다음은 함수 몸체에 `return` 구문으로 함수 결과를 반환할 수 있다. 다음 `print(circle_area(r = 7.2))` 호출로 반지름이 7.2인 원의 면적을 반환 받아 출력할 수 있다.

```python
import math as m

def circle_area(r = 1):
    return m.pi * r**2

print(circle_area(r = 7.2))
```
**출력:**
```
162.8601631620949
```

다음 함수 선언으로 두 인자 `x`와 `y`의 합과 차를 계산해 두 결과를 함께 반환한다. 반환 값을 반환하는 `return` 함수는 쉼표로 구분해 여러 값을 반환할 수 있는 데, 이 경우 반환 자료형은 튜플(tuple)이다.

```python
def sum_sub(x, y):
    hap = x + y
    cha = x - y
    return hap, cha
```

다음 함수 호출로 합인 13, 차인 7이 괄호가 있는 튜플 형태로 출력된다.

```python
sum_sub(10, 3)
```
**출력:**
```
(13, 7)
```

다음처럼 인자를 직접 지정한 함수 호출도 합인 13, 차인 7이 출력된다.

```python
sum_sub(x = 10, y = 3)
```
**출력:**
```
(13, 7)
```

다음처럼 인자 순서를 바꾸어 직접 지정한 `(y = 3, x = 10)` 함수 호출도 합인 13, 차인 7이 출력된다.

```python
sum_sub(y = 3, x = 10)
```
**출력:**
```
(13, 7)
```

### 람다(lambda) 익명 함수

다음 코드처럼 람다 함수를 선언하면서 이후에 인자를 `(3, 4)`를 적용해 실행할 수 있다. 다음은 함수 이름을 지정하지 않은 익명 함수(anonymous function)로 다시 사용할 수 없다. 파이썬은 익명 함수로 `lambda` 함수를 지원한다.

```python
import math as m

(lambda a, b: m.sqrt(a**2 + b**2))(3, 4)
```
**출력:**
```
5.0
```
