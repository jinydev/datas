# SECTION 02. 제어 흐름과 함수

## 01. 조건문 if

### 블록 문장 조건문 if

다음 조건문은 조건이 `True`이면 들여쓰기 된 블록을 실행하는 조건 문장이다. 다음 코드로 `n`에 저장된 20이 짝수임을 출력한다. 조건에 괄호는 없어도 되며 조건 이후에 콜론 `:`은 반드시 필요하다. 들여쓰기는 빈칸으로 2개 또는 4개를 일관되게 사용해야 한다. 파이썬에서는 들여쓰기가 블록을 구성하는 문법으로 잘못 사용하면 오류가 발생하니 주의하자.

```python
n = 20

if n % 2 == 0:
    print("짝수")
```
**출력:**
```
짝수
```

다음 코드에서 `n`이 21로 홀수이므로 아무것도 출력되지 않는다.

```python
n = 21
if n % 2 == 0:
    print("짝수")
```

### 조건문 if else

다음 `if 조건: 블록1 else: 블록2` 구문은 조건이 `True`이면 `블록1`을, `False`이면 `블록2`를 실행한다. `else:` 처럼 `else` 이후에도 콜론 `:`이 반드시 필요하다. 다음 왼쪽 코드로 나이 18세는 조건 `age >= 20`가 거짓이므로 `else:` 이후 블록인 '미성년자'를 출력한다. 우측 코드로 21세는 조건이 `True`이므로 '성인'을 출력한다.

```python
age = 18
if age >= 20:
    print('성인')
else:
    print('미성년자')
```
**출력:**
```
미성년자
```

### 한 줄 조건문 if else

다음 한 줄 조건 `반환1 if 조건 else 반환2` 구문은 조건이 `True`이면 `반환1`을, 거짓이면 `반환2`를 반환한다.

```python
age = 21

if age >= 20:
    print('성인')
else:
    print('미성년자')
```

다음 코드로 조건 `age >= 20`이 만족되면 '성인'을 아니면 '미성년자'를 변수 `s`에 저장한다.

```python
age = 18
s = '성인' if age >= 20 else '미성년자'
print(s)
# 출력: 미성년자

age = 20
s = '성인' if age >= 20 else '미성년자'
print(s)
# 출력: 성인
```

### 반복된 조건문 if elif

다음 `if 조건1: 블록1 elif 조건2: 블록2 ...` 구문은 여러 개의 반복되는 문장으로 활용될 수 있다. 다만 `else if`가 아니라 `elif`를 쓰며 마지막에 `else:`를 사용할 수 있다. 왼쪽 코드로 성적 92는 학점 A가 출력되며, 오른쪽 코드로 성적 62는 학점 D가 출력된다.

```python
point = 92
if 90 <= point:
    print('A')
elif 80 <= point:
    print('B')
elif 70 <= point:
    print('C')
elif 60 <= point:
    print('D')
else:
    print('F')
```
**출력:**
```
A
```
