## 07. 내장 모듈 math 활용

파이썬의 모듈은 프로그램의 기능을 확장하기 위해 만드는 파이썬 소스 파일이다. 내장 모듈은 따로 설치 없이 사용할 수 있는 모듈이다. 대표적인 파이썬 내장 모듈이 `math`로 다음의 수학 관련 상수와 함수가 정의되어 있다.

| 분류                     | 연산자                             |
| ------------------------ | ---------------------------------- |
| 상수                     | `pi` `e`                           |
| 제곱근과 지수, 로그 함수 | `sqrt()` `exp()` `log()` `log10()` |
| 천정과 바닥값, 내림      | `ceil()` `floor()` `trunc()`       |
| 삼각 함수                | `sin()` `cos()` `tan()`            |

이미 보았듯이 먼저 모듈 `math`를 메모리에 적재(load)하기 위해 구문 `import math as m`을 사용한다. 여기서 `m`을 별명(alias)으로 지정해 `m.함수()`를 사용한다. 다음 `m.pi`, `m.e`로 원주율과 자연수를 참조한다.

```python
import math as m

m.pi, m.e
```
**출력:**
```
(3.141592653589793, 2.718281828459045)
```

함수 `m.sqrt(4)` 호출은 4의 제곱근인 2를 반환한다.

```python
m.sqrt(4)
```

함수 `m.ceil(3.7)` 호출은 3.7 이상의 가장 가까운 정수인 4를 반환한다. 함수 `ceil(a)`는 `a`의 천정 값이라고 한다.

```python
m.ceil(3.7)
m.ceil(3.1)
m.ceil(-3.7)
m.ceil(-3.1)
```

함수 `m.floor(3.7)` 호출은 3.7 이하인 가장 가까운 정수인 3을 반환한다. 함수 `floor(a)`는 `a`의 바닥 값이라고 한다.

```python
m.floor(3.7)
m.floor(3.1)
m.floor(-3.7)
m.floor(-3.1)
```

함수 `m.trunc(3.1415)` 호출은 소수점 이하을 버린다.

```python
m.trunc(3.1415)
```
