## 05. 다양한 연산자

다음은 대부분의 프로그래밍 언어에서 지원하는 사칙 연산자 `+`, `-`, `*`, `/`이다.

```python
10 + 4.3
5.4 - 1.2
4 * 2.7
10 / 4
```

연산자 `%`은 나머지 연산자이며, `//`은 정수 나누기 연산자이다. 연산자 `%`와 `//` 모두 실수에도 사용이 가능하다.

```python
10 % 3 # 나머지 연산자
10 // 3 # 몫 연산자 integer division
```

연산자 `**`은 지수 연산자이다.

```python
3 ** 3 # 지수 연산자
2 ** 2.8 # 지수 연산자
```

여러 연산자가 있는 연산식에서 먼저 계산을 원하는 연산에 괄호를 사용할 수 있다.

```python
3 * (4 - 2)
(10 - 3) ** 2
```

논리 값 `True`와 `False`에 사용되는 논리 연산자 `&`와 `and`는 AND 연산자이다. 연산자 `|`와 `or`는 OR 연산자이다. AND는 모두 `True`여야 결과가 `True`이고, OR는 하나만 `True`여도 `True`가 결과이다.

```python
True & True
True and False
True | True
False or False
```

파이썬에서 논리 값을 바꾸는 연산자는 `not`이다.

```python
not True
not False
```

파이썬의 주요 연산자를 정리하면 다음과 같다.

| 분류                                      | 연산자                           | 비고                                                                                                                             |
| ----------------------------------------- | -------------------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| 사칙연산                                  | `+`, `-`, `*`, `/`               | C 언어와 일치                                                                                                                    |
| 나머지 연산자 (modulo)                    | `%`                              | R 언어, `%%`                                                                                                                     |
| 정수 나누기, 몫 연산자 (integer division) | `//`                             | R 언어, `%/%`                                                                                                                    |
| 지수 연산자                               | `**`                             | R 언어, `^`                                                                                                                      |
| 관계 연산자                               | `>`, `<`, `>=`, `<=`, `==`, `!=` | C 언어와 일치                                                                                                                    |
| 논리 연산자                               | `&`, `and`, `                    | `, `or`, `not`                                                                                                                   | C 언어, `&`, `&&`, ` | `, ` |  | `, `!` |
| 대입 연산자                               | `=`                              | 파이썬과 자바 등의 대부분의 언어에서는 `=`을 사용하며, R은 대부분 `<-`을 사용하며, 함수 선언에서 패러미터 초기 지정은 `=`을 사용 |
