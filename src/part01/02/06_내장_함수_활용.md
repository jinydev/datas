## 06. 내장 함수 활용

다양한 함수를 기본으로 제공한다. 파이썬을 설치하고 추가 패키지 설치 없이 사용할 수 내장 함수(built-in function)로는 `input()`, `print()`, `range()` 등 매우 다양하다. 기본적으로 사용되는 파이썬의 주요 내장 함수를 정리하면 다음과 같다.

| 연산자    |                           |                         |
| --------- | ------------------------- | ----------------------- |
| `input()` | `print()`                 |                         |
| `type()`  | `int()` `float()` `str()` | `bin()` `oct()` `hex()` |
| `list()`  | `tuple()` `set()`         | `dict()`                |
| `len()`   |                           |                         |
| `range()` |                           |                         |
| `min()`   | `max()`                   | `abs()` `pow()`         |

```python
help(print)
```

### 함수 input() 출력 활용

함수 `input()`은 사용자로부터 키보드 입력을 받는 함수이다. 입력된 값은 기본적으로 문자열 형태로 반환되며 이를 적절히 형 변환하여 다른 타입으로 사용할 수 있다. 다음 코드로 사용자에게 년도를 입력하라는 프롬프트(prompt) '당신이 태어난 년도는 ? '가 표시되고, 사용자가 입력한 년도 2007을 변수 `year`에 저장한다.

```python
year = input('당신이 태어난 년도는 ? ')
# 당신이 태어난 년도는 ? 2007 # 직접 입력한 후 엔터를 누른다
```

위 코드로 변수 `year`에 값 2007이 문자열 형태로 저장된다. 다음으로 확인할 수 있다. `print()` 함수는 인자로 전달된 값을 화면에 출력한다.

```python
print(year)
type(year)
```
**출력:**
```
2007
<class 'str'>
```

즉, `year`가 문자열이어서 다음 코드의 빼기 연산에서 오류가 발생한다. 주의가 필요하다.

```python
age = 2026 - year
```
**오류:**
```
TypeError: unsupported operand type(s) for -: 'int' and 'str'
```

다음처럼 `int(year)` 함수를 사용하여 정수로 변환해 나이를 구할 수 있다.

```python
age = 2026 - int(year)
print('나이', age)
```

함수 `int(input(...))`을 사용하면 변수 `year`에 정수 값이 저장되어 간편히 나이를 구할 수 있다. 함수 `print()`에서 여러 개의 값을 출력할 때는 콤마 `,`로 구분한다. 각 값은 기본적으로 공백으로 구분되어 출력된다.

```python
year = int(input('당신이 태어난 년도는 ? '))
# 당신이 태어난 년도는 ? 2005
print('나이', 2026 - year)
```

함수 `abs(-3)` 호출은 인자인 -3의 절대값인 3을 반환한다.

```python
abs(-3)
```

함수 `max(a, b, c, ...)` 호출은 여러 인자에서 최대 값을 반환한다. 마찬가지로 함수 `min(a, b, c, ...)` 호출은 최소 값을 반환한다.

```python
min(10, 20, 1)
max(10, 100, 1)
```

다음으로 리스트 중에서 최소와 최대를 얻을 수 있다. 대괄호로 묶어 표현하는 `[10, 11, 12]`를 리스트라 한다.

```python
min([10, 11, 12])
max([10, 11, 12])
```

함수 `round(3.1415)` 호출은 반올림한 정수인 3을 반환한다. `round(3.1415, 3)` 호출은 두 번째 인자인 3에 의해 소수점이 이하 3자리까지 반올림해 표시하는 값을 반환한다.

```python
round(3.1415)
round(3.1415, 3) # 소수점 셋째자리까지 표기하기 위해 3.1415에서 5를 반올림해 3.142가 반환
```
