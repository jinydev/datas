# 파이썬으로 배우는 데이터 분석 입문 - Part 2

# SECTION 01. 파이썬 언어 기초

## 01. 주석과 상수

파이썬도 코드에서 `#` 이후부터 줄 마지막까지 주석(comment)이다.

```python
# comments
```

일상에서 사용하는 정수나 실수를 콘솔에 입력하고 Enter를 누르면 그 값을 바로 출력해 준다. 10은 정수(int) 상수이며, 3.4는 실수(float) 상수(literal)이다.

```python
3.4 # numeric literals
```
**출력:**
```
3.4
```

정수 뒤에 `L`을 붙이면 에러(error, 오류)가 발생한다. 오류는 발생 위치를 `^`로 표시하고 다음 줄에 오류 이름 `SyntaxError`와 오류 메시지 `invalid decimal literal`을 보여준다.

```python
7L
```
**출력:**
```
  File "<stdin>", line 1
    7L
     ^
SyntaxError: invalid decimal literal
```

다음처럼 여러 값을 나열해도 에러가 발생한다.

```python
-5 4.5
```
**출력:**
```
  File "<stdin>", line 1
    -5 4.5
         ^
SyntaxError: invalid syntax
```

다음처럼 상수를 쉼표(또는 콤마) `,`로 구분해 나열하면 값을 둘러싼 수의 나열이 표시된다. 이를 파이썬에서 튜플(tuple)이라 한다.

```python
10, -3
```
**출력:**
```
(10, -3)
```
