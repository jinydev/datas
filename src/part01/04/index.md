# 파이썬으로 배우는 데이터 분석 입문 - Part 4

문서가 각각의 주제별로 분리되었습니다. 아래의 링크를 클릭하여 해당 학습 내용으로 이동하세요.

## 학습목표
- **배열 결합**: `np.hstack`, `np.vstack`, `np.concatenate`, `np.stack` 등 다양한 함수를 활용해 배열을 수평 또는 수직으로 결합할 수 있다.
- **배열 분할**: `np.split`, `np.hsplit`, `np.vsplit` 등을 이용하여 다차원 배열을 원하는 축을 기준으로 여러 하위 배열로 나눌 수 있다.
- **난수 생성**: `numpy.random` 모듈의 제너레이터를 사용하여 원하는 형태의 균등 분포 난수나 표준 정규 분포 난수를 생성할 수 있다.

## SECTION 01. 배열 결합 개요와 유용한 함수

### [01. 배열 결합 개요](./01_배열_결합_개요)
배열을 결합할 때 기본적으로 사용되는 `vstack`과 `hstack`의 원리와 차원 확장에 대해 소개합니다.

### [02. numpy.vstack()](./02_numpy_vstack)
다차원 배열이나 벡터를 수직(vertically)으로 쌓아 합치는 `numpy.vstack()` 함수의 사용법과 조건을 배웁니다.

### [03. numpy.hstack()](./03_numpy_hstack)
다차원 배열이나 벡터를 수평(horizontally)으로 합쳐 확장하는 `numpy.hstack()` 함수의 사용법과 조건을 배웁니다.

### [04. 배열 결합 연산 column_stack()과 row_stack()](./04_배열_결합_연산_column_stack과_row_stack)
열을 기준으로 쌓는 `column_stack()`과 행 단위로 쌓는 `row_stack()`의 원리와, 1차원 배열 연산 시 `hstack`, `vstack`과의 결과 차이를 비교합니다.

### [05. 배열 결합 연산 concatenate()와 stack()](./05_배열_결합_연산_concatenate와_stack)
지정한 기준 축(`axis`)에 따라 범용적으로 배열을 결합하는 `concatenate()`와 새로운 축을 차원으로 추가하며 배열을 쌓는 `stack()`을 학습합니다.

### [06. column_stack()과 hstack()](./06_column_stack과_hstack)
1차원 배열과 2차원 배열을 함께 활용해 `column_stack`과 `hstack`이 배열을 어떻게 확장하여 결합하는지 직관적으로 알아봅니다.

### [07. row_stack()과 vstack()](./07_row_stack과_vstack)
완전히 동일한 기능과 역할을 수행하는 두 함수 `row_stack()`과 `vstack()`의 활용 예제를 간단히 살펴봅니다.

### [08. np.c_[]와 np.column_stack()](./08_npc_와_npcolumn_stack)
간편한 대괄호 연산자 객체 `np.c_[]`를 이용해 여러 배열이나 스칼라 값들을 1축을 기준으로 수평 결합하는 팁을 다룹니다.

### [09. np.r_[]](./09_npr_)
문자열 인자(`'0, 3, 0'` 등)와 등차 수열 생성을 결합해 0축을 기준으로 다차원 배열로 쌓거나 차원을 확장하는 강력한 결합 객체 `np.r_[]`을 익힙니다.

### [10. 1차원 배열, 벡터의 다양한 결합 방법](./10_1차원_배열_벡터의_다양한_결합_방법)
지금까지 배운 벡터의 요소 단위 결합과 1축 확장에 해당하는 여러 가지 함수 조합들의 동작 원리를 구체적인 코드 예제로 재확인합니다.

### [11. 1차원 배열의 다양한 결합 방법 정리](./11_1차원_배열의_다양한_결합_방법_정리)
1차원 배열을 수평/수직/열 방향으로 다채롭게 나누고 확장하는 결합 기법과 사용 함수들을 한 장의 도형 그림으로 요약하여 이해합니다.

## SECTION 02. 배열의 분할과 복사

### [12. 유용한 함수 split()](./12_유용한_함수_split)
결합과 반대로 특정 다차원 배열을 원하는 축(`axis`)을 기준으로 `n`등분하거나 지정된 첨자 구간별로 나누어 쪼개는 함수 `split()`을 배웁니다.

### [13. 분할 함수 hsplit()과 vsplit()](./13_분할_함수_hsplit과_vsplit)
가로 방향(수평)으로 분할하는 `hsplit()`과 세로 방향(수직)으로 분리하는 `vsplit()`의 차이 및 제약사항(동등 분할 필수 조건)을 살펴봅니다.

### [14. 분할 함수 array_split()](./14_분할_함수_array_split)
배열 요소의 수가 분할 개수로 정확히 나누어떨어지지 않을 때도 유연하게 배열을 쪼개는 등분 유도 함수 `array_split()`의 사용법과 `dsplit()`을 다룹니다.
