# 파이썬으로 배우는 데이터 분석 입문 - Part 7

문서가 각각의 주제별로 분리되었습니다. 아래의 링크를 클릭하여 해당 학습 내용으로 이동하세요.

## 학습목표
- **데이터프레임 정보 분석 및 처리**: Pandas DataFrame의 전반적인 구조 및 결측치 구성, 각 열의 데이터 타입을 알려주는 `info()`와 기초 정보 및 산술 속성을 이해한다.
- **다양한 통계 지표와 수정 대입 적용**: `mean()`, `sum()`, `max()` 등의 기본 함수로 행과 축을 넘나들며 집계 연산을 수행하고 그 값을 새롭게 열이나 행으로 수정, 대입하는 방식을 파악한다.
- **고급 조건 검색 및 레이블 기반 정렬**: 불리언 인덱싱(조건 참조)과 `isin()`, `str.contains()` 메서드를 비롯한 검색 꿀팁, 그리고 정렬 축과 오름·내림차순 조건에 입각한 정렬(`sort_index`, `sort_values`)을 정복한다.

## SECTION 01. 데이터프레임 정보와 조건 검색

### [01. 데이터프레임 정보와 수정](./01_데이터프레임_정보와_수정)
외부 파일 포맷인 CSV 자료를 불러와 `read_csv`로 DataFrame을 생성, `dtypes`, `info()`, `describe()`를 통해 테이블 전반의 메타 정보를 신속하게 요약해서 파악하는 기법을 배웁니다.

### [02. 데이터프레임 정보 추출과 대입](./02_데이터프레임_정보_추출과_대입)
sum, mean, max, min, cumsum 등 축(axis)에 따라 이동하면서 합계나 평균과 같은 유용한 통계 함수를 산출해 내는 여러 방식을 집중적으로 다룹니다.

### [03. 데이터프레임 수정 대입](./03_데이터프레임_수정_대입)
행 및 열 간의 산술적인 연산을 통해 새로운 지표('재해자수', '총계' 등)를 계산하고, 이 결과를 바로 기존 프레임의 새로운 Series(열, 행 단위) 값으로 추가하는 템플릿 코드 작성법을 익힙니다.

### [04. 데이터프레임 조건 검색](./04_데이터프레임_조건_검색)
기본적인 조건문(>, >=, <, ==, & 등)뿐만 아니라 `Series.isin`이나 문자열 관련 함수 `str.contains()`를 조합하여 아주 유연하고 강력하게 DataFrame 내부 데이터를 걸러내는(Filter) 필터링 방법을 알아봅니다.

## SECTION 02. 데이터프레임 정렬과 수정

### [05. 데이터프레임 정렬](./05_데이터프레임_정렬)
인덱스를 기준으로 하는 레이블 정렬(`sort_index`)과 컬럼 등의 값 크기 자체를 기준으로 삼는 값 기반 정렬(`sort_values`) 및, 복수 조건 하의 오름차순/내림차순 변형법을 설명합니다.

### [06. 데이터프레임의 다양한 수정](./06_데이터프레임의_다양한_수정)
하나의 요소나 선택된 공간(부분집합)에 대해서 `loc`, `at`, `iloc`, `iat`으로 접근하여 정확하게 수정하는 방법뿐만 아니라 `np.nan` 적용이나 `isna()`, `count()` 등 결측치 관련 검증 활용 방법까지 다룹니다.
