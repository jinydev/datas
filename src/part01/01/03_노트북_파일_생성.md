## 03. 노트북 파일 생성

노트북의 셀은 코드 셀(Code Cell)과 텍스트 셀(Text Cell)로 구분된다. 코드 셀에는 파이썬 코드를 담고, 텍스트 셀에는 마크다운(Markdown) 형식의 문서가 저장된다. 셀에서 위쪽 부분의 '+ 코드'를 선택하면 위에 셀이 생성되며 아래 부분을 선택하면 아래에 셀이 생성된다. 텍스트 셀은 왼쪽의 마크다운 입력 부분과 오른쪽의 표시 부분으로 구성된다.

다음은 두 코드 셀의 실행 화면이다. 좌측 상단의 `[1]`, `[2]`는 실행 순서이다. 셀의 순서가 아니라 실행된 순서이다. 다음 실행에서 함수 `site.getsitepackages()`로 출력되는 내용이 일반 지역 컴퓨터와 다르다는 것을 알 수 있다.

```python
import sys
sys.version
```

**출력 (예시):**
```
'3.10.12 (main, Nov 20 2023, 15:14:05) [GCC 11.4.0]'
```

```python
import site
site.getsitepackages()
```

**출력 (예시):**
```
['/usr/local/lib/python3.10/dist-packages',
 '/usr/lib/python3/dist-packages',
 '/usr/lib/python3.10/dist-packages']
```
