# SECTION 04. 구글 코랩 개요

## 01. Colab 개요

`구글` 코랩(Google Colab)은 구글이 제공하는 `클라우드` 기반의 `주피터 노트북`(Jupyter Notebook) 환경이다. 

주로 `파이썬`을 기반으로 한 `데이터 분석`, `머신러닝`, `딥러닝` 등의 작업을 수행할 때 그 진가를 발휘한다. 

### 

코랩은 `브라우저`에서 실행되며, `무료`로 사용할 수 있는 가상 머신을 제공하여 사용자가 별도로 서버를 구축하지 않고도 고성능의 하드웨어 자원을 활용할 수 있게 해준다. 물론, 고성능 하드웨어를 오랜 기간 사용하려면 유료이나 단기간 사용한다면 무료이다.

### 코랩의 주요특징

- **무료 GPU 지원**: 무료로 사용할 수 있는 GPU를 제공하여 딥러닝 모델 학습 등에 유용
- **클라우드 기반**: 모든 작업은 구글의 클라우드 서버에서 이루어지기 때문에 사용자의 로컬 환경에 구애받지 않음
- **주피터 노트북 통합**: 기존 주피터 노트북과 호환되어 있어 코드와 문서를 함께 작성하고 시각화 가능
- **라이브러 지원**: 주요 데이터 과학 및 머신러닝 라이브러리들을 기본적으로 지원하며, 필요 시 추가 라이브러리 설치 가능
- **구글 드라이브 연동**: 구글 드라이브와 연동되어 코드 및 데이터를 손쉽게 저장하고 공유 가능
- **다중 언어 지원**: 주로 파이썬을 사용하지만, 다양한 언어를 지원하며 특정 셀에서 다른 언어 실행 가능
- **데이터 시각화**: Matplotlib, Seaborn, Plotly 등 다양한 시각화 라이브러리를 활용하여 데이터를 시각적으로 분석 가능

코랩을 사용하면 사용자는 개인적인 환경설정이나 고성능 하드웨어의 필요성에 구애받지 않고, 즉시 클라우드 기반의 머신러닝 또는 데이터 분석 프로젝트를 시작할 수 있는 장점으로 활용도가 높다.
