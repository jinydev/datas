# SECTION 02. 비주얼 스튜디오 코드

## 01. VS Code 개요

비주얼 스튜디오 코드(Visual Studio Code, 이하 VS Code)는 마이크로소프트에서 제공하는 대부분의 프로그래밍 언어를 위한 개발환경을 지원하고 가장 확장 가능한 통합개발환경(IDE)이다. 주로 다양한 소프트웨어 개발환경으로 사용되며, 많은 언어와 프레임워크(framework)를 지원한다. 전문 텍스트 에디터로 2016년에 정식판이 처음으로 발표된 이후, 매우 빠른 버전 개발로 현재 호평받는 범용 코드 편집기이자 통합개발환경으로 발전되었다.

- **무료 및 오픈 소스**: VS Code는 무료로 사용할 수 있으며 오픈 소스로 개발되어 커뮤니티에서 지속적으로 발전
- **가볍고 빠른 실행**: 다른 대부분의 IDE에 비해 가벼우면서 빠르게 실행
- **다양한 언어 지원**: 다양한 프로그래밍 언어를 지원하며, 확장 기능을 통해 필요한 언어나 프레임워크의 지원 추가 가능
- **인텔리센스(IntelliSense)**: 코드 작성 중 자동 완성 및 코드 힌트를 제공하여 개발 생산성을 높일 수 있음
- **통합 터미널**: 내장된 터미널을 통해 프로젝트를 관리하고 명령을 실행할 수 있음
- **Git 통합**: 내장된 Git 지원으로 소스 코드의 버전 관리 및 협업 용이
- **테마 및 아이콘 설정**: 다양한 테마와 아이콘을 지원하여 개발자의 취향에 맞게 환경을 설정할 수 있음
- **확장성**: 확장 기능을 통해 플러그인을 추가하고, 필요한 도구와 기능을 손쉽게 확장할 수 있음
- **통합 디버깅**: 내장된 디버깅 기능으로 코드를 효과적으로 디버깅할 수 있음
- **다중 플랫폼 지원**: Windows, macOS, Linux 등 다양한 플랫폼에서 동작

데이터 사이언스 및 머신러닝용 기능에 대한 지원도 강력하며, 주피터 노트북 클라이언트를 자체 내장하고 있어 바로 이용할 수 있으며, 변수 탐색기(Variable Explorer) 등 편리한 기능을 다수 제공한다. 개발자들은 자신의 환경에 맞게 플러그인과 설정을 추가하여 보다 효율적인 작업을 수행할 수 있다. 특히, 파이썬 개발환경으로 사용한다면 일반 파이썬 파일 `*.py`과 노트북 파일 `*.ipynb` 활용이 가능하며 대화형 셸(interactive shell)로도 사용할 수 있는 장점이 있다.

VS Code는 간단한 코드 에디터에서부터 복잡한 프로젝트를 지원하는 풍부한 기능을 제공하여 최근 많은 개발자에게 인기를 얻고 있다.
