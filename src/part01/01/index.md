# 파이썬으로 배우는 데이터 분석 입문 - Part 1

개발 환경 구성을 위해 작성된 문서들이 주제별로 분리되어 있습니다. 아래의 링크를 클릭하여 세부 학습 내용으로 이동하세요.

## 학습목표
- **클라우드 환경 이해**: 구글 코랩(Google Colab)의 특징을 이해하고 기본적인 사용법을 익힌다.
- **데이터 활용 준비**: 코랩 환경에서 파일 데이터를 업로드하고 시각화를 위한 준비 과정을 마친다.
- **로컬 에디터 구성**: 로컬 개발 환경인 비주얼 스튜디오 코드(VS Code)를 설치하고 파이썬 확장을 구성한다.
- **스크립트 실행**: VS Code에서 파이썬 코드를 작성하고 실행하는 기본 과정을 익힌다.

## 학습 목차

### [01. Colab 개요](./01_colab_개요.md)
데이터 분석을 위한 파이썬 클라우드 환경인 구글 코랩(Google Colab)에 대해 소개합니다. 
브라우저 환경에서 별도 설정 없이 무료로 파이썬을 실행할 수 있는 장점을 다룹니다.

### [02. 코랩 접속](./02_코랩_접속.md)
구글 계정을 이용해 코랩에 로그인하고 접속하는 과정과 기본 화면 구성을 살펴봅니다. 
첫 파이썬 실습을 위한 클라우드 워크스페이스 준비 방법을 배웁니다.

### [03. 노트북 파일 생성](./03_노트북_파일_생성.md)
코랩 내에서 코드를 작성하고 실행할 수 있는 주피터 기반 인터랙티브 노트북을 생성하는 방법을 익힙니다. 
코드 셀(Cell)과 텍스트 셀을 다루는 기초적인 사용법을 학습합니다.

### [04. 데이터 시각화 준비](./04_데이터_시각화_준비.md)
분석된 결과물을 시각적으로 표현하기 위해 필요한 환경 설정과 단계를 알아봅니다. 
기본적인 시각화 라이브러리 사용을 위한 준비 사항과 한글 폰트 적용 등을 체크합니다.

### [05. 파일 데이터 처리](./05_파일_데이터_처리.md)
외부 데이터를 코랩 메모리로 불러오고 활용하기 위해 파일 데이터를 업로드하고 읽어오는 방법을 학습합니다. 
구글 드라이브와 연동하여 데이터 파일을 다루는 기초 기술을 배웁니다.

### [06. 노트북 파일 저장](./06_노트북_파일_저장.md)
코딩 실습한 코랩 노트북의 작업 내역을 구글 드라이브나 로컬(PC) 임시 저장소에 안전하게 유지하는 방법을 배웁니다. 
파일을 잃어버리지 않고 체계적으로 관리하는 법을 다룹니다.

### [07. VS Code 개요](./07_vscode_개요.md)
로컬 환경 최고의 파이썬 개발 에디터 중 하나인 비주얼 스튜디오 코드(VS Code)의 특징과 장점을 알아봅니다. 
코랩과 대비되는 로컬 개발 툴의 강력한 확장성과 편의성을 살펴봅니다.

### [08. VS Code 설정](./08_vscode_설정.md)
원활하고 편안한 파이썬 코딩을 위해 VS Code 내부 환경, 테마, 단축키 등을 사용자 맞춤형으로 세팅하는 과정을 배웁니다. 
개발 작업 집중력을 높이는 기초 인터페이스 설정을 진행합니다.

### [09. 파이썬 확장 설치와 소스 파일 실행](./09_파이썬_확장_설치와_소스_파일_실행.md)
VS Code 내에 필수 파이썬 플러그인(Extension)을 설치하여 프로그래밍 환경을 완성하고 실제 코드를 구동하는 과정을 익힙니다. 
첫 파이썬 소스 파일을 성공적으로 실행하여 로컬 디버깅 환경을 구축합니다.
