### 인구 데이터 전처리

키워드 인자 `encoding='cp949'`, `index_col=0`, `thousands=','`를 사용해 첫 열을 index로, 콤마가 들어간 값은 정수로 파일을 읽어 `pop`에 저장한다.

```python
import pandas as pd

pop = pd.read_csv("202311popmonth.csv", encoding='cp949', index_col=0, thousands=',')
pop
```

**출력 (예시):**
```
              총인구수      세대수   세대당인구    남자인구수    여자인구수   남여비율
행정구역
전국          51337076  23904793    2.15   25572695  25764381   0.99
서울특별시        9390925   4469374    2.10    4543055   4847870   0.94
부산광역시        3295496   1564199    2.11    1606680   1688816   0.95
...
```

다음 코드로 데이터프레임 `pop`의 컬럼명을 수정한다. 첫 5행을 표시해 보자.

```python
pop.columns = ['총인구수', '세대수', '세대당인구', '남자인구수', '여자인구수', '남여비율']
pop.head()
```

위 결과에서 우측 그래프 아이콘을 누르면 추천하는 그래프를 볼 수 있다.

다음으로 `pop`의 index를 수정하자. 행정구역 코드 등을 제거하고 지역명만 남기기 위함이다.

```python
pop.index = [pop.index[i].split()[0] for i in range(len(pop))]
pop
```

**출력 (예시):**
```
             총인구수      세대수   세대당인구    남자인구수    여자인구수   남여비율
전국          51337076  23904793    2.15   25572695  25764381   0.99
서울특별시        9390925   4469374    2.10    4543055   4847870   0.94
...
```

### 인구 데이터 시각화

다음은 `seaborn`의 `sns.scatterplot()`으로 '총인구수'의 산점도를 그린 코드이다.

```python
import matplotlib.pyplot as plt
import seaborn as sns

sns.scatterplot(pop['총인구수'])
plt.xticks(rotation=90)
plt.show()
```
