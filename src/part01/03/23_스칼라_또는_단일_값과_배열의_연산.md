## 23. 스칼라 또는 단일 값과 배열의 연산

단일 값 또는 스칼라(scalar)는 1차원 배열의 산술 연산에 사용될 수 있다. 다음은 배열 `a`이다.

```python
from numpy import array

a = array([3, 4, 5])
a
```
**출력:**
```
array([3, 4, 5])
```

이제, 스칼라 2가 저장된 변수 `b`가 있다. 배열 `a`와 스칼라 `b`는 산술 연산이 가능하다. 이전에 이미 살펴보았다.

```python
b = 2
```

내부적으로 스칼라 `b`가 브로드캐스트되어 연산이 수행되는 `a`와 같이 `b` 값이 그대로 3개의 원소가 된 배열이 된다. 즉 내부적으로 `b = [2, 2, 2]`로 브로드캐스트되어 연산이 수행된다. 이러한 간단한 예처럼 브로드캐스트란 스칼라 `b`를 배열 `a`와 같은 모양의 배열이 되도록 늘리는(stretch) 작업이다.

```python
a + b
```
**출력:**
```
array([5, 6, 7])
```

스칼라 값은 1차원 배열뿐 아니라 2차원 이상의 배열과의 연산에서도 다른 배열과 같은 모양으로 브로드캐스트되어 연산이 수행된다.

```python
A = np.array([[1, 2, 3], [1, 2, 3]])
A
```
**출력:**
```
array([[1, 2, 3],
       [1, 2, 3]])
```

다음 코드처럼 2차원 배열의 모든 원소에 2가 더해진 결과가 반환된다.

```python
A + 2
```
**출력:**
```
array([[3, 4, 5],
       [3, 4, 5]])
```
