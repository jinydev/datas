# SECTION 06. 브로드캐스팅

## 22. 브로드캐스팅 개요

브로드캐스팅(broadcasting)이라는 용어는 NumPy가 산술 연산을 할 때 서로 다른 모양의 배열을 연산이 수행되도록 처리하는 방법이다. 배열 연산 시, 작은 배열은 서로 호환되는 모양이 되도록 특정한 제약조건에 따라 현재의 작은 배열을 크게 전파("브로드캐스트")해 더 큰 배열로 만들어 연산을 수행한다. 즉, 브로드캐스팅은 배열에 대한 요소별 산술을 수행할 수 있도록 모양이 다른 배열의 크기를 일치시키는 작업이다.

이미 보았듯이 NumPy 연산은 일반적으로 요소 단위로 배열 쌍에 대해 수행된다. 일반적으로 두 배열에서 산술 연산을 하려면 가장 간단한 경우, 두 배열의 모양은 다음 예제와 같이 정확히 같아야 한다.

```python
from numpy import array

a = array([3, 4, 5])
b = array([8, 7, 6])
a + b
```
**출력:**
```
array([11, 11, 11])
```

두 배열의 모양이 애매하게 다르면 다음 코드처럼 `ValueError`가 발생한다.

```python
a = array([3, 4, 5, 7])
b = array([8, 7, 6])
a * b
```
**오류:**
```
ValueError: operands could not be broadcast together with shapes (4,) (3,)
```
