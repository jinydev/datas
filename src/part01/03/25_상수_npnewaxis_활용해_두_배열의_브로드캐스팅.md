## 25. 상수 np.newaxis 활용해 두 배열의 브로드캐스팅

다음은 모양 (4, )의 1차원 배열 `x`가 있다.

```python
from numpy import array

x = array([0, 10, 20, 30])
x
```
**출력:**
```
array([ 0, 10, 20, 30])
```

다음 코드처럼 `x[None, :]`로 행이 추가된 모양 (1, 4)의 2차원 배열의 보기(view)가 반환된다. 즉, 1차원에서 행이 추가되어 2차원으로 차원이 증가했다.

```python
a = x[None, :]
a
```
**출력:**
```
array([[ 0, 10, 20, 30]])
```

위 구문에서 `None` 상수인 `np.newaxis`로 대체해도 같은 결과를 얻는다.

```python
a = x[np.newaxis, :]
a
```
**출력:**
```
array([[ 0, 10, 20, 30]])
```

이제 다음 코드 `a = x[:, np.newaxis]`는 모양 (4, )이 모양 (4, 1)로 열이 추가된 2차원 배열의 보기(view)가 반환된다. 즉, 1차원에서 열이 추가되어 2차원으로 차원이 증가했다.

```python
a = x[:, np.newaxis]
a
```
**출력:**
```
array([[ 0],
       [10],
       [20],
       [30]])
```

다음 배열 `b`는 모양 (3, )인 1차원 배열이다.

```python
from numpy import array

b = array([1, 2, 3])
b
```
**출력:**
```
array([1, 2, 3])
```

위에서 만든 `a`와 `b`를 사용해 산술 곱하기 연산 `a + b`를 수행해 보자. 배열 `a`는 열이 1이며 배열 `b`는 행이 1이다. 배열 `a`를 2차원으로 그대로 오른쪽으로 브로드캐스트(전파)해 늘리고 배열 `b`를 2차원으로 그대로 아래쪽으로 브로드캐스트(전파)해 늘리면 배열 `a`와 `b`는 모양이 (4, 3)으로 같아 산술 연산이 가능해진다. 즉, 여기서 `np.newaxis`(새로운 축) 인덱스 연산자는 `a`에 새로운 축(axis 1)을 삽입하여 2차원 4x1 배열을 만든다. 4x1 배열에 (3, ) 모양을 가진 `b`를 결합해 연산을 수행하면 4x3 배열이 생성된다.

다음 코드는 `a + b`의 결과로 더하기 연산이 잘 수행된다.

```python
a + b
```
**출력:**
```
array([[ 1,  2,  3],
       [11, 12, 13],
       [21, 22, 23],
       [31, 32, 33]])
```

물론 곱하기도 가능하다.

```python
x = np.array([0, 10, 20, 30])
x = x[:, np.newaxis]
y = np.array([1, 2, 3])
x * y
```
**출력:**
```
array([[ 0,  0,  0],
       [10, 20, 30],
       [20, 40, 60],
       [30, 60, 90]])
```

브로드캐스팅은 완전히 같은 모양이 아니어도 배열의 보편적인 연산을 수행할 수 있게 해준다. 다음의 브로드캐스트 규칙 적용 후 모든 배열의 모양이 일치해야 오류 없이 연산이 수행된다.

- 브로드캐스팅의 첫 번째 규칙은 연산에 참여하는 배열이 동일한 수의 차원을 갖지 않으면 모든 배열이 동일한 수의 차원을 가질 때까지 작은 배열의 모양에 반복적으로 '1'이 추가된다는 것이다.
- 브로드캐스팅의 두 번째 규칙은 1의 크기를 갖는 배열들이 그 차원을 따라 가장 큰 모양을 갖는 배열의 크기로 늘어날 수 있다는 것이다.
- "브로드캐스트" 배열에 대해 배열 요소의 값은 그 차원을 따라 그대로 동일하게 확장된다고 가정한다.
