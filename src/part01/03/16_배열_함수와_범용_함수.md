## 16. 배열 함수와 범용 함수

### ① 배열 함수

다음 코드로 2차원 리스트를 인자로 배열을 생성하자.

```python
a = np.array([[10, 20], [5, 8], [1, 2]])
a
```
**출력:**
```
array([[10, 20],
       [ 5,  8],
       [ 1,  2]])
```

`a.sum()`은 배열의 모든 원소 합이 반환된다.

```python
a.sum()
```
**출력:**
```
46
```

`a.sum(axis=0)`은 배열의 동일 열에서 행을 순회해 세로로 더한 합을 구한 배열을 반환한다.

```python
a.sum(axis=0)
```
**출력:**
```
array([16, 30])
```

`a.sum(axis=1)`은 배열의 동일 행에서 열을 순회해 가로로 더한 합을 구한 배열을 반환한다.

```python
a.sum(axis=1)
```
**출력:**
```
array([30, 13,  3])
```

다음처럼 `np.sum(a, ...)`으로도 가능하다.

```python
import numpy as np

print(np.sum(a))
print(np.sum(a, axis=0))
print(np.sum(a, axis=1))
```
**출력:**
```
46
[16 30]
[30 13  3]
```

함수 `ndarray.cumsum()`은 모든 원소를 순서대로 누적 합계를 반환한다. n차원도 일차원 배열의 결과가 된다.

```python
a = np.array([[3, 5, 8], [4, 7, 11]])
a
```
**출력:**
```
array([[ 3,  5,  8],
       [ 4,  7, 11]])
```

```python
a.cumsum()
```
**출력:**
```
array([ 3,  8, 16, 20, 27, 38])
```

함수 `ndarray.cumsum(axis=0)`은 주어진 축인 열에 따라 요소의 누적 합계를 반환한다. 차원은 동일한 배열 모양의 결과가 된다. 함수 `ndarray.cumsum(axis=1)`은 주어진 축인 행에 따라 요소의 누적 합계를 반환한다. 차원은 동일한 배열 모양의 결과가 된다.

```python
a.cumsum(axis=0)
```
**출력:**
```
array([[ 3,  5,  8],
       [ 7, 12, 19]])
```

```python
a.cumsum(axis=1)
```
**출력:**
```
array([[ 3,  8, 16],
       [ 4, 11, 22]])
```

### ② 범용 함수

`numpy`는 `sin()`와 `cos()`, `exp()`와 같은 친숙한 수학 함수인 범용 함수(universal functions, 간단히 ufunc)를 제공한다. 범용 함수는 다양한 수학 및 통계적 연산을 빠르게 수행할 수 있도록 도와주는 핵심적인 기능 중 하나이다. NumPy 내에서 범용 함수는 배열에서 요소별로 작동하여 결과를 출력으로 생성한다. NumPy의 UFuncs는 배열 요소별로 연산을 적용하는 함수로, 벡터화된 연산을 지원하여 성능을 향상시킨다.

다음은 `numpy`의 주요 범용 함수 몇 가지를 표로 나타낸 것이다.

| 함수                  | 설명                       | 예제                                                     |
| :-------------------- | :------------------------- | :------------------------------------------------------- |
| `np.add(x1, x2)`      | 요소별 덧셈                | `np.add([1, 2], [3, 4])` -> `[4, 6]`                     |
| `np.subtract(x1, x2)` | 요소별 뺄셈                | `np.subtract([3, 4], [1, 2])` -> `[2, 2]`                |
| `np.multiply(x1, x2)` | 요소별 곱셈                | `np.multiply([1, 2], [3, 4])` -> `[3, 8]`                |
| `np.divide(x1, x2)`   | 요소별 나눗셈              | `np.divide([3, 4], [1, 2])` -> `[3., 2.]`                |
| `np.power(x1, x2)`    | 요소별 거듭제곱            | `np.power([1, 2], [3, 4])` -> `[1, 16]`                  |
| `np.sqrt(x)`          | 요소별 제곱근              | `np.sqrt([1, 4, 9])` -> `[1., 2., 3.]`                   |
| `np.sin(x)`           | 요소별 삼각 함수(sin)      | `np.sin([0, np.pi/2, np.pi])` -> `[0., 1., 0.]`          |
| `np.cos(x)`           | 요소별 삼각 함수(cos)      | `np.cos([0, np.pi/2, np.pi])` -> `[1., 0., -1.]`         |
| `np.exp(x)`           | 요소별 지수 함수(exp)      | `np.exp([1, 2, 3])` -> `[2.718..., 7.389..., 20.085...]` |
| `np.log(x)`           | 요소별 자연 로그 함수(log) | `np.log([1, np.e, np.e**2])` -> `[0., 1., 2.]`           |

다음 코드는 인자가 하나인 `np.sqrt(a)`의 결과를 반환한다.

```python
import numpy as np

a = np.array([1, 4, 9])
np.sqrt(a)
```
**출력:**
```
array([1., 2., 3.])
```

다음 코드는 인자가 하나인 `np.log(a)`의 결과를 반환한다.

```python
np.log(a)
```
**출력:**
```
array([0.        , 1.38629436, 2.19722458])
```

다음 원소 3개의 1차원 배열을 변수 `a`에 저장한다. 마찬가지로 동일 모양으로 원소가 모두 2인 배열을 변수 `b`에 저장한다.

```python
import numpy as np
a = np.array([1, 4, 9])
a
```
**출력:**
```
array([1, 4, 9])
```

```python
b = np.full_like(a, 2)
b
```
**출력:**
```
array([2, 2, 2])
```

모양이 같은 두 배열을 사용해 범용 함수를 활용해 보자. 다음 인자가 2개인 범용 함수 `np.add(a, b)`와 `np.divide(a, b)`, `np.power(a, b)`의 사용 예이다.

```python
np.add(a, b)
```
**출력:**
```
array([ 3,  6, 11])
```

```python
np.divide(a, b)
```
**출력:**
```
array([0.5, 2. , 4.5])
```

```python
np.power(a, b)
```
**출력:**
```
array([ 1, 16, 81], dtype=int32)
```
