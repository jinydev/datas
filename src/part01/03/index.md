# 03. 배열 수치 계산을 위한 numpy

## 학습목표
- **다차원 배열의 이해**: numpy의 핵심 자료구조인 ndarray의 개념과 속성을 이해하고, 파이썬 기본 리스트와의 차이점을 파악한다.
- **배열 생성 및 초기화**: `arange`, `linspace`, `zeros`, `ones`, `eye` 등 다양한 내장 함수를 활용하여 목적에 맞는 배열을 효율적으로 생성한다.
- **배열 연산과 브로드캐스팅**: 배열 간의 사칙연산, 범용 함수(ufunc) 특성을 이해하고, 형태가 다른 배열 간의 연산을 가능하게 하는 브로드캐스팅(Broadcasting) 원리를 숙지한다.
- **인덱싱과 슬라이싱**: 1차원을 넘어 2차원, 3차원 다차원 배열에서 원하는 데이터를 정확하게 추출하는 슬라이싱 및 고급 인덱싱(팬시 인덱싱) 기법을 다룬다.
- **배열 형태 변환 및 메모리 관리**: `reshape`, `transpose` 등을 활용해 차원을 능수능란하게 다루며, 데이터의 복사(copy)와 참조(view)의 메커니즘을 정확히 구별한다.

- [01. 과학용 컴퓨팅 패키지 numpy 개요](01_과학용_컴퓨팅_패키지_numpy_개요)  
  넘파이(numpy)는 파이썬에서 고성능 다차원 배열 및 행렬 연산을 지원하는 핵심 라이브러리로, 브로드캐스팅 및 다양한 수학 함수들을 제공하여 수치 계산과 데이터 분석에 필수적입니다.
- [02. 다차원 배열 자료형 ndarray](02_다차원_배열_자료형_ndarray)  
  numpy의 대표적인 자료형인 ndarray는 동일한 데이터 타입의 다차원 배열을 지원하며, 파이썬 기본 자료형보다 더 세분화되고 다양한 수치 자료형(int, float, complex 등)을 제공합니다.
- [03. 내장함수 array()로 ndarray 생성](03_내장함수_array로_ndarray_생성)  
  리스트나 튜플을 인자로 받아 ndarray 객체를 생성하는 np.array() 함수의 사용법과, `shape`, `ndim`, `dtype` 등의 주요 속성을 통해 배열 정보를 확인하는 방법을 설명합니다.
- [04. numpy의 내장함수 arange() 개요](04_numpy의_내장함수_arange_개요)  
  파이썬 표준 내장함수 range()처럼 일정한 간격의 숫자를 반환하지만 정수뿐 아니라 실수 간격도 지원하며, 범위를 정해 빠르고 쉽게 배열을 생성하는 np.arange()를 다룹니다.
- [05. 배열의 모양을 바꾸는 reshape()](05_배열의_모양을_바꾸는_reshape)  
  이미 만들어진 다차원 배열의 원소 개수를 유지하면서 배열의 행과 열 구조(shape)를 자유롭게 변경하는 reshape() 함수의 원리와 '-1' 인자의 의미를 소개합니다.
- [06. 일정한 간격의 값으로 배열을 만드는 linspace()](06_일정한_간격의_값으로_배열을_만드는_linspace)  
  시작값과 끝값을 지정하고 그 사이를 균등한 간격으로 나누어 원하는 개수(num)만큼의 값을 갖는 배열을 쉽게 생성해주는 np.linspace()의 기능을 학습합니다.
- [07. 내장함수 arange()와 linspace() 비교](07_내장함수_arange와_linspace_비교)  
  간격(step)을 기준으로 배열을 생성하는 arange()와 샘플 수(num)를 기준으로 구간을 분할하는 linspace()의 차이점과 실수 처리 시 주의사항을 비교하여 보여줍니다.
- [08. 원소가 모두 0인 배열을 생성하는 zeros()](08_원소가_모두_0인_배열을_생성하는_zeros)  
  지정한 모양(shape)과 데이터 타입(dtype)에 맞게 모든 원소가 0으로 채워진 안전한 다차원 배열을 초기화하여 생성하는 np.zeros() 함수를 배웁니다.
- [09. 원소가 모두 1인 배열을 생성하는 ones()](09_원소가_모두_1인_배열을_생성하는_ones)  
  np.zeros()와 유사하게, 인자로 전달한 형상(shape)에 맞춰 모든 원소값이 1.0(또는 지정 타입)으로 초기화된 배열을 단숨에 만들어주는 np.ones() 활용법을 알아봅니다.
- [10. 대각선 원소가 모두 1인 2차원 배열을 만드는 eye()](10_대각선_원소가_모두_1인_2차원_배열을_만드는_eye)  
  선형대수학의 단위행렬(Identity matrix)처럼 주 대각선의 원소는 모두 1이고 나머지는 0인 정방행렬 형태의 2차원 배열을 반환하는 np.eye() 함수를 설명합니다.
- [11. 원소가 모두 특정 값인 배열을 만드는 full()](11_원소가_모두_특정_값인_배열을_만드는_full)  
  원하는 배열의 크기(shape)와 함께 임의의 스칼라 또는 배열을 전달하여, 해당 동일한 값들로만 꽉 채워지는 다차원 배열을 생성하는 np.full() 함수에 대해 익힙니다.
- [12. 초기화되지 않은 배열을 생성하는 empty()](12_초기화되지_않은_배열을_생성하는_empty)  
  메모리 공간만 할당하여 매우 빠르게 형태(shape)를 갖춘 배열을 생성하지만, 초기화를 거치지 않아 예측 불가능한 쓰레기값을 가지게 되는 np.empty()를 다룹니다.
- [13. 내장함수 zeros_like()와 ones_like(), full_like()](13_내장함수_zeros_like와_ones_like_full_like)  
  기존에 있는 배열과 동일한 형태(shape)를 유지하되, 모든 성분을 0, 1, 혹은 특정 값으로 대체한 새로운 배열을 반환하는 유용한 `_like` 접미사 함수들을 묶어 설명합니다.
- [14. 배열과 스칼라와의 연산](14_배열과_스칼라와의_연산)  
  numpy 배열 전체에 단일 스칼라 숫자 값을 더하거나 곱하게 되면, 배열의 각 독립된 원소마다 해당 연산이 병렬로 적용되어 반환되는 편리한 특성을 알아봅니다.
- [15. 배열과 배열의 연산](15_배열과_배열의_연산)  
  형태(shape)가 동일한 두 다차원 배열끼리 사칙연산을 수행할 때 같은 인덱스 위치의 원소 단위(element-wise)로 계산되는 과정과 행렬 곱 연산자(`@`)를 학습합니다.
- [16. 배열 함수와 범용 함수](16_배열_함수와_범용_함수)  
  축(axis)을 기준으로 배열 전체 합을 구하는 배열 전용 메서드(`sum()`, `cumsum()`)와, 요소 단위의 복잡한 수학 연산을 전역적으로 쉽고 빠르게 처리하는 유니버설 함수(ufunc)를 소개합니다.
- [17. 1차원 배열의 첨자와 슬라이싱](17_1차원_배열의_첨자와_슬라이싱)  
  1차원 벡터에서 양/음수 인덱싱을 통해 개별 값에 접근하고, `start:stop:step` 등 파이썬 리스트와 유사한 슬라이싱 기법으로 부분값 추출 및 구간 일괄 할당 방법을 배워봅니다.
- [18. 2차원 배열의 첨자와 슬라이싱](18_2차원_배열의_첨자와_슬라이싱)  
  2차원 구조(행렬) 행과 열에 독립적인 인덱스와 슬라이스를 다중으로 적용하고 파이썬 기본 리스트 방식을 넘어선 `a[i, j]` 형태의 직관적인 다차원 값 추출법을 연습합니다.
- [19. 3차원 배열의 첨자와 슬라이싱](19_3차원_배열의_첨자와_슬라이싱)  
  깊이, 행, 열의 여러 겹 구조를 가진 3차원 이상 배열의 축(axis) 개념을 파악하고, 여러 `:`이나 `...`(Ellipsis) 연산자를 활용하여 세밀하게 차원을 넘나드는 데이터 탐색을 시연합니다.
- [20. 배열 형태 수정](20_배열_형태_수정)  
  메모리 구조 변경 없이 다차원을 1차원으로 펴주는 `ravel()`과, 선형 대수학 관점에서 행과 열의 축을 교환하는 전치행렬 기능 `transpose()`, `swapaxes()` 함수 사용법을 학습합니다.
- [21. 복사(copy)와 보기(뷰, view)](21_복사copy와_보기뷰_view)  
  메모리 효율을 위해 원본 데이터를 참조 형태로 공유하는 뷰(view)방식과 얕은 복사, 그리고 완전히 새로운 독립 배열 객체를 창출하는 깊은 복사 `copy()`의 차이를 명확히 합니다.
- [22. 브로드캐스팅 개요](22_브로드캐스팅_개요)  
  서로 구조(shape)가 완전히 일치하지 않는 배열 간에도 규칙에 따라 작은 배열을 큰 배열의 크기로 확장(broadcast)하여, 요소 간의 산술 연산을 가능하게 하는 개념을 설명합니다.
- [23. 스칼라 또는 단일 값과 배열의 연산](23_스칼라_또는_단일_값과_배열의_연산)  
  하나의 실수/정수값이 마치 배열의 전체 사이즈만큼 길이가 확장되어 자동으로 맞춰진 뒤 사칙 연산이 수행되는 원리 등, 가장 기본적인 형태의 브로드캐스트 현상을 되짚습니다.
- [24. 2차원 배열의 브로드캐스팅](24_2차원_배열의_브로드캐스팅)  
  1차원 배열이 2차원 배열의 행이나 열과 대응하여 논리적으로 면적을 확장하고 연산을 허용하는 경우, 두 배열이 교차로 확장하여 동작하는 다차원 브로드캐스팅의 복잡한 예시를 다룹니다.
- [25. 상수 np.newaxis 활용해 두 배열의 브로드캐스팅](25_상수_npnewaxis_활용해_두_배열의_브로드캐스팅)  
  기존의 배열 차원을 1단계 높임으로서 브로드캐스트 조건을 강제로 성립되게 유도하는 `np.newaxis` 또는 파이썬의 `None` 키워드 활용과 축 삽입 기법을 집중적으로 살펴봅니다.
- [26. 1차원 배열의 색인](26_1차원_배열의_색인)  
  numpy만의 강력한 특징인, 단순 정수 하나가 아닌 인덱스 숫자들이 나열된 또 다른 배열이나 리스트를 인덱스로 대입하여 데이터 군을 한 번에 가져오는 고급 색인 기능을 소개합니다.
- [27. 2차원 배열의 색인](27_2차원_배열의_색인)  
  2차원 배열의 행과 열 인덱스를 각각 다중 배열로 입력하여 결합된 좌표에 위치한 요소들을 추출하거나, 차원/크기 불일치시에도 허용되는 규칙적 브로드캐스팅 인덱스 방식들을 배웁니다.
- [28. 다양한 슬라이싱과 배열 색인](28_다양한_슬라이싱과_배열_색인)  
  기존에 배운 일반 슬라이싱(Slicing) 구문과 팬시 인덱싱(정수 배열, boolean)을 복합적으로 병행 사용하는 다양한 예제를 통해 실무에서 직면할 데이터 부분집합 추출을 마스터합니다.
