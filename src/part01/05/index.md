# 파이썬으로 배우는 데이터 분석 입문 - Part 5

문서가 각각의 주제별로 분리되었습니다. 아래의 링크를 클릭하여 해당 학습 내용으로 이동하세요.

## 학습목표
- **데이터 시각화 기초**: Matplotlib의 특징과 기본 설정 방법(한글 폰트 등)을 익히고, 간단한 직선 그래프 작성법을 실습합니다.
- **다양한 그래프와 서브플롯**: 선 플롯 외에도 산점도, 막대그래프, 히스토그램 등 여러 형태의 그래프를 작성해 보고, 한 화면에 여러 그래프를 그리는 서브플롯(`subplot()`, `subplots()`)의 개념을 이해합니다.
- **맞춤형 레이아웃 (GridSpec)**: GridSpec을 활용해 가로/세로 비율을 조절하거나, 중첩된 그리드 등 복잡하고 세밀한 화면 분할 레이아웃을 구성하는 방법을 익힙니다.

## SECTION 01. 데이터 시각화 기초

### [01. 데이터 시각화 개요](./01_데이터_시각화_개요)
Matplotlib 라이브러리의 특징을 알아보고, 주피터 노트북에서의 사용을 위한 한글 처리 등 기본 설정을 준비합니다.

### [02. 간단한 직선 그래프 그리기](./02_간단한_직선_그래프_그리기)
`plot()`을 이용한 선 그래프뿐만 아니라 속성 지정 방법과 산점도(`scatter()`), 막대그래프(`bar()`), 히스토그램(`hist()`)을 그리는 법을 살펴봅니다.

### [03. figure와 서브플롯](./03_figure와_서브플롯)
전체 그림 영역인 Figure와 실제 그래프가 그려지는 영역인 Axes의 관계를 이해하고, `subplot`, `subplots`, `add_subplot`으로 화면을 분할하는 기본 방법을 학습합니다.

## SECTION 02. 맞춤형 서브플롯

### [04. GridSpec을 사용한 맞춤형 figure 레이아웃](./04_GridSpec을_사용한_맞춤형_figure_레이아웃)
`GridSpec` 클래스를 통해 행/열의 여백을 상세히 나누고 병합하거나 중첩시켜 다채롭고 복잡한 그래프 배치 레이아웃을 구성하는 방법을 심도 있게 다룹니다.
