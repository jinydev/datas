# 파이썬으로 배우는 데이터 분석 입문 - Part 8

문서가 각각의 주제별로 분리되었습니다. 아래의 링크를 클릭하여 해당 학습 내용으로 이동하세요.

## 학습목표
- **테스트 데이터 활용(pydataset)**: 샘플 데이터셋 제공 패키지인 `pydataset`의 설치 및 사용법을 이해하고, 타이타닉(`titanic`), 자동차 연비(`mpg`)와 같은 유명한 내장 데이터프레임을 직접 로드하여 실습의 기반을 다진다.
- **데이터 시각화 응용(Seaborn)**: 통계적 시각화 도구인 `Seaborn` 패키지를 설치하고, `pydataset`에서 불러온 데이터를 기반으로 막대그래프(`barplot`), 히스토그램(`histplot`) 등 직관적인 분석 결과를 출력하는 방법을 익힌다.
- **외부 데이터 파일 연동**: `csv`, `excel` 등 외부에서 수집한 공공데이터 및 파일(예: 시도별 교통사고 데이터)을 판다스 `read_csv`, `read_excel`로 직접 불러들이고, 오류 제어 및 전처리(`skiprows`, `thousands`) 과정을 실습한다.

## SECTION 01. 패키지 pydataset 활용

### [01. pydataset 소개](./01_pydataset_소개)
데이터 분석 연습을 위해 파이썬에서 즉시 로드해 활용할 수 있는 다양한 Toy Dataset을 제공하는 패키지인 `pydataset`의 개념과 주요 특징에 대해 간략히 알아봅니다.

### [02. 패키지 pydataset 설치](./02_패키지_pydataset_설치)
명령어를 통해 `pydataset` 모듈을 설치하고, `data()` 함수를 호출하여 750여 개가 넘는 샘플 데이터셋 목록과 전체적인 정보를 탐색하는 방법을 살펴봅니다.

### [03. pydataset 주요 자료 가져오기](./03_pydataset_주요_자료_가져오기)
실제 분석에서 자주 쓰이는 대표적인 샘플 데이터인 `titanic`(타이타닉) 데이터와 `mpg`(자동차 연비) 데이터를 불러와 `info`, `describe`, `value_counts` 등으로 요약 정보를 뽑아내는 실습을 진행합니다.

### [04. seaborn으로 그래프 그리기](./04_seaborn으로_그래프_그리기)
아름답고 강력한 통계 그래픽 라이브러리인 `seaborn`을 설치하고, 앞서 불러온 `mpg` 데이터를 사용해 제조사별, 엔진 배기량, 연도별 분포를 `histplot`으로 시각화하는 방법을 학습합니다.
