# 2.3 리눅스(Linux) 및 서버 파이썬 설치 가이드

> **학습목표**: 클라우드 서버스(AWS, Google Cloud)나 우분투(Ubuntu) 같은 리눅스 운영체제(OS)에서 파이썬 최신 버전을 설치하는 방법을 배워봅니다.

보통 리눅스는 관리자 전용 서버 컴퓨터로 쓰이는 경우가 많아서 마우스로 누르는 예쁜 그래픽 화면보다, 까만색 터미널 화면에서 '텍스트 명령어'로 파이썬을 직접 설치합니다. 여기서는 세계에서 가장 많이 쓰이는 **Ubuntu(우분투)**를 기준으로 설명하겠습니다. (초보자들은 윈도우나 맥으로 공부하는 것이 일반적이지만, 나중에 내가 만든 인공지능을 세상에 배포하기 위해서는 서버 지식도 아주 유용하답니다!)

## 1. 시스템 업데이트
새로운 프로그램을 설치하기 전에 먼저 시스템의 패키지 설치 목록을 최신 상태로 만들어 줍니다.
터미널 창에 이렇게 입력하세요:
```bash
sudo apt update
sudo apt upgrade -y
```
- `sudo`: "최고 관리자 권한으로 명령어를 실행할게" 라는 뜻입니다. 누르면 서버 비밀번호를 물어봅니다.
- `apt`: 우분투의 스마트폰 앱스토어 같이 프로그램들을 다운받아주는 시스템 친구입니다.

## 2. 파이썬 설치하기
최신 우분투에는 파이썬이 이미 기본으로 깔려 있는 경우가 아주 많습니다. 
먼저 내 서버에 있는지 확인해 볼까요?
```bash
python3 --version
```
짠 하고 버전 번호가 나오면 이미 설치된 것입니다! 

하지만 만약 너무 옛날 버전이거나 아예 없다면 다음과 같이 명령어를 쳐서 설치하세요.
```bash
sudo apt install python3 -y
```

파이썬 외에 코딩할 때 필수로 필요한 다른 도구들(pip 패키지 매니저 등)도 함께 설치해 줘야 고생을 덜 합니다.
```bash
sudo apt install python3-pip python3-venv -y
```

## 3. 제대로 깔렸는지 최종 확인하기
모든 과정이 끝나면 다시 버전을 확인하여 파이썬과 pip가 제대로 돌아가는지 점검합니다.
```bash
python3 --version
pip3 --version
```

성공적으로 두 개의 버전 번호가 다 보인다면, 멋진 리눅스 서버에서도 파이썬 데이터 분석을 할 모든 준비가 완료된 것입니다! 🚀
