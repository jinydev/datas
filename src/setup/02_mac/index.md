# 2.2 맥(macOS) 파이썬 설치 가이드

> **학습목표**: MacBook이나 iMac 등 macOS 환경에서 파이썬을 설치하고 확인하는 방법을 알아봅니다.

최근의 맥북(특히 실리콘 M1, M2, M3 칩이 들어간 맥)에서는 파이썬을 설치하는 방법이 윈도우와 비슷하면서도 약간 다릅니다. 공식 홈페이지 설치 파일(pkg)을 이용해 보겠습니다.

## 1. 파이썬 다운로드
1. 파이썬 홈페이지에 접속합니다: [https://www.python.org/downloads/mac-osx/](https://www.python.org/downloads/mac-osx/)
2. `Latest Python 3 Release` 버튼을 누르거나 화면 중앙의 노란색 **Download Python 3.x.x** 버튼을 눌러 macOS용 `.pkg` 파일을 다운로드합니다.

## 2. 파이썬 설치하기
1. 다운로드된 `.pkg` 파이썬 설치 파일을 두 번 클릭해서 엽니다.
2. 설치 화면의 지시에 따라 **계속(Continue)** 버튼을 연타하고 맨 마지막에 내 맥북의 잠금해제용 **비밀번호(또는 Touch ID)**를 입력해 주면 설치가 쫙 진행됩니다.
3. 윈도우와 달리 별다른 복잡한 경로 체크박스 설정이 필요 없이 알아서 셋업이 완료됩니다. 편리하죠!

## 3. 설치 확인하기 (터미널 프로그램)
1. 바탕화면 오른쪽 위 돋보기 아이콘(Spotlight 검색)을 누르거나 `Command(⌘) + Space` 키를 동시에 눌러주세요.
2. 검색창에 **`터미널`** 또는 **`Terminal`** 이라고 치고 엔터를 쳐 터미널 앱을 엽니다.
3. 열린 까만색(또는 하얀색) 창 화면에 아래와 같이 명령어를 쳐봅시다:
   ```bash
   python3 --version
   ```
   *⚠️주의: 맥에서는 예전 시스템 기본 파이썬 프로그램들과 헷갈리지 않게 하기 위해서 `python` 대신 `python3`라고 입력해야 최신버전이 나오는 경우가 아주 많습니다.*
4. 앱 화면에 `Python 3.12.x` 등 버전 정보가 무사히 뜨면 설치 대성공입니다! 🎉
