# 2.1 윈도우(Windows) 파이썬 설치 가이드

> **학습목표**: 윈도우 환경에서 파이썬을 다운로드하고 설치하는 방법을 배우며, 설치가 잘 되었는지 터미널 창(명령 프롬프트)에서 확인해 봅니다.

데이터 분석의 첫 걸음은 내 컴퓨터에 파이썬을 설치하는 것입니다. 차근차근 따라해 보세요!

## 1. 파이썬 다운로드
1. 파이썬 공식 홈페이지의 다운로드 페이지에 들어갑니다: [https://www.python.org/downloads/](https://www.python.org/downloads/)
2. 화면 중앙에 있는 노란색 **'Download Python 3.x.x'** 버튼을 클릭하여 설치 파일을 다운로드합니다. (버전 숫자는 최신 버전에 따라 다를 수 있습니다.)

## 2. 파이썬 설치하기 (★ 매우 중요 ★)
설치 파일을 다운로드했다면 더블클릭하여 실행합니다. 여기서 **가장 중요한 설정** 하나가 있습니다.

1. 설치 창이 열리면 화면 맨 아래에 있는 **`Add python.exe to PATH`** 글씨 옆의 체크박스를 **반드시 체크**해 주세요.
   - *이 체크를 빼먹으면 터미널 창에서 파이썬 명령어를 알아듣지 못해서 처음부터 다시 설치해야 할 수 있습니다!*
2. 체크를 확인했다면, 바로 위에 있는 **`Install Now`**를 클릭합니다.
3. 설치가 쭉 진행되고 "Setup was successful" 메시지가 나타나면 **`Close`**를 눌러 창을 닫습니다.


## 3. 설치가 잘 되었는지 확인하기 (터미널)
이제 컴퓨터가 파이썬을 잘 설치했는지 테스트 해보겠습니다.

1. 키보드에서 `윈도우 키(❖) + R`을 동시에 누릅니다.
2. '실행' 창이 뜨면 `cmd` 라고 입력하고 엔터(Enter)를 누릅니다.
3. 까만색 터미널 창(명령 프롬프트)이 열릴 것입니다. 여기에 아래 명령어를 입력하고 엔터를 치세요!
   ```bash
   python --version
   ```
4. `Python 3.12.x` 처럼 파이썬 버전 숫자가 나타난다면 설치 대성공입니다! 🎉

> 윈도우 터미널 설정?
> 윈도우 터미널은 기존의 명령 프롬프트(cmd)를 대체하는 새로운 터미널 프로그램입니다. 윈도우 11에서는 기본적으로 설치되어 있으며, 윈도우 10에서는 Microsoft Store에서 'Windows Terminal'을 검색하여 설치할 수 있습니다.

> **💡 만약 에러가 난다면?**  
> `python은(는) 내부 또는 외부 명령, 실행할 수 있는 프로그램, 또는 배치 파일이 아닙니다` 라는 에러 메시지가 뜬다면, 설치할 때 `Add python.exe to PATH` 박스 체크를 깜빡하셨을 확률이 높습니다. 이럴 땐 설치 파일을 다시 실행해서 'Modify/Repair' 기능을 이용하거나, 삭제 후 다시 체크하고 깨끗하게 재설치 해주세요!
