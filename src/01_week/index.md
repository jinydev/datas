# 1주차: 데이터 분석 개요 및 환경 설정

> **학습 목표 (Learning Objectives)**
> 1. **데이터 분석의 이해**: 데이터 분석이 무엇이며, 왜 파이썬을 사용하는지 이해합니다.
> 2. **개발 환경 구축**: Google Colab과 VS Code(Antigravity)를 활용하여 코딩 환경을 설정합니다.
> 3. **파이썬 기초**: 변수, 자료형, 기본 연산 등 파이썬의 핵심 문법을 익힙니다.

## 1.3. 정리 (Summary)

이번 주차에는 데이터 분석의 큰 그림을 이해하고, 앞으로의 여행을 위한 베이스캠프(환경 설정)를 구축했습니다.

1.  **데이터 분석**: 원석(데이터)을 가공하여 보석(인사이트)을 만드는 과정입니다. DIKW 피라미드와 5단계 분석 과정을 배웠습니다.
2.  **환경 설정**:
    -   **Colab**: 설치가 필요 없는 간편한 웹 환경.
    -   **VS Code**: 현업에서 사용하는 강력한 로컬 에디터.
    -   **Antigravity (Vibe Coding)**: AI와 대화하며 코딩하는 새로운 패러다임.
3.  **파이썬 기초**: 프로그래밍의 첫걸음인 `print`, 변수, 기본 자료형(`int`, `str`)을 익혔습니다.
4.  **시각화 맛보기**: `Matplotlib`를 사용하여 데이터를 그래프로 표현하는 즐거움을 맛보았습니다.

다음 주차(**Week 2**)에는 파이썬의 더 강력한 기능(리스트, 제어문)을 배우고, 시각화 기술을 한 단계 업그레이드합니다. 준비되셨나요? 🚀
