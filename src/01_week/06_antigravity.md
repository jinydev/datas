# 1주차 환경설정: Vibe Coding & Antigravity

> **학습목표**: AI 코딩 도구인 Vibe Coding과 Antigravity의 개념을 이해하고, VS Code와 유사한 인터페이스를 제공하는 Antigravity의 활용 방법을 익힙니다.

## 바이브 코딩이란?

> 🎵 **Vibe Coding**: 
> AI와 자연스럽게 호흡하며(Vibe), 마치 음악을 연주하듯 리듬감 있게 코딩하는 새로운 개발 방식입니다.

과거의 코딩이 악보를 보고 하나하나 건반을 누르는 **"입력 중심"**이었다면,  
바이브 코딩은 AI라는 훌륭한 파트너와 함께 즉흥 연주를 하듯 **"흐름 중심"**으로 코드를 만들어가는 것입니다.

- **Think**: 여러분은 **"무엇을 만들까?"**에 집중하세요.
- **Implementation**: 복잡한 문법과 구현은 **AI**가 도와줍니다.
- **Rhythm**: 질문하고, 코드를 받고, 실행하는 과정이 끊김 없이 물 흐르듯 이어집니다.

### 대표적인 AI 코딩 도구 (Popular AI Coding Tools)
바이브 코딩을 도와주는 다양한 도구들이 있습니다.

- **GitHub Copilot**: 가장 널리 사용되는 AI 자동완성 도구로, 주석을 코드로 변환하거나 다음 코드를 제안합니다.
- **Cursor**: VS Code를 포크(Fork)하여 만든 AI 전용 에디터로, 코드베이스 전체를 이해하는 강력한 채팅 기능을 제공합니다.
- **Claude Code**: Anthropic에서 만든 강력한 코딩 능력의 AI로, 자연어 명령을 터미널에서 실행하고 코드를 수정합니다.
- **Antigravity**: 우리가 사용할 도구로, VS Code 내에서 강력한 Pair Programming 경험을 제공합니다.

## AI 와 함께 데이터 분석하기

데이터 분석 프로젝트에서 AI는 든든한 조수 역할을 합니다.

**① 복잡한 데이터 전처리 맡기기**
- *"날짜 컬럼이 섞여 있는데, 'YYYY-MM-DD' 형식으로 통일해 줘."*
- 귀찮은 엑셀 정리 작업, AI에게 맡기면 3초면 끝납니다.

**② 시각화 코드 생성하기**
- *"이 데이터를 이용해서 월별 매출 추이를 꺾은선 그래프로 그려줘. 색상은 파란색으로 해줘."*
- Matplotlib 코드를 일일이 기억할 필요가 없습니다.

**③ 코드 설명 및 에러 수정**
- *"이 코드가 무슨 뜻이야?"* (코드 해석)
- *"IndexError가 났는데 어떻게 고쳐야 해?"* (디버깅)

## 안티그래비티 설치 및 사용법

> 🤖 **Antigravity**: 
> 여러분의 바이브 코딩을 현실로 만들어줄 **AI 짝 프로그래밍(Pair Programming) 파트너**입니다. (VS Code 유사 인터페이스)

### 1. 설치 방법 (Installation)

**Windows / macOS 공통**

1.  **VS Code 실행**: Visual Studio Code를 켭니다.
2.  **확장 프로그램 탭 열기**:
    -   단축키: `Ctrl + Shift + X` (Win) / `Cmd + Shift + X` (Mac)
    -   또는 좌측 사이드바의 **네모 블록 아이콘**을 클릭합니다.
3.  **검색 및 설치**:
    -   검색창에 `Antigravity`를 입력합니다.
    -   Microsoft 또는 Google Deepmind의 `Antigravity` 항목에서 **Install** 버튼을 클릭합니다.
4.  **로그인**:
    -   설치 후 나타나는 `Sign in` 버튼을 눌러 구글 계정으로 로그인합니다.

### 2. 실행 및 활용 (Usage)

| 기능                    | 단축키 (Win/Mac)       | 설명                                                        |
| :---------------------- | :--------------------- | :---------------------------------------------------------- |
| **채팅 (Chat)**         | `Ctrl + L` / `Cmd + L` | AI에게 질문하거나 대화합니다. *"오늘 점심 뭐 먹을까?"*      |
| **코드 생성 (Gen)**     | `Ctrl + K` / `Cmd + K` | 에디터에서 바로 코드를 생성합니다. *"1부터 100까지 더해줘"* |
| **코드 설명 (Explain)** | `Ctrl + I` / `Cmd + I` | 선택한 코드가 무엇인지 물어봅니다. *"이게 무슨 코드야?"*    |
