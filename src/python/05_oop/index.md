# 3.5 객체지향 프로그래밍 (OOP)

현대 프로그래밍의 가장 중요한 패러다임인 객체지향 프로그래밍(Object-Oriented Programming)을 파이썬으로 구현하는 방법을 배웁니다. 클래스와 인스턴스의 개념을 숙지합니다.

## 학습목표
본 장에서는 현실 세계의 사물을 컴퓨터 세상에 붕어빵 찍어내듯 모델링하는 **객체지향 프로그래밍(OOP)**의 패러다임을 확립합니다. 코드를 깔끔하게 재사용하는 3대 기둥인 **캡슐화(클래스와 인스턴스), 상속 및 다형성, 추상화(인터페이스)**의 핵심 뼈대를 완벽히 이해합니다.

## 정리
수백 줄, 수천 줄로 길어지는 절차적 코드의 한계를 극복하기 위해 인류가 고안해 낸 위대한 발명품이 바로 객체지향(OOP)입니다. 서로 독립된 고유 상태를 가진 '인스턴스' 객체들이 생태계 안에서 각자의 '메서드'로 소통하고 동작하는 구조를 짤 수 있게 됨으로써, 여러분은 이제 진정한 소프트웨어 아키텍트의 넓은 시야를 갖추게 되었습니다.
