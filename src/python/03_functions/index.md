# 3.3 함수 (Functions)

코드를 재사용하고 모듈화하기 위한 핵심 개념인 함수에 대해 학습합니다. 논리적인 블록을 정의하고 매개변수와 반환값을 활용하는 방법을 익힐 수 있습니다.

## 학습목표
본 장에서는 마법 공간인 **'함수(Function)'**를 직접 설계하는 방법(`def`)부터 값을 넘겨받는 매개변수, 완제품을 뱉어내는 반환값(`return`)의 메커니즘을 완벽히 이해합니다. 또한 변수들의 생존 공간인 스코프(Scope)의 개념과 일회용 가벼운 껍데기인 **람다(Lambda) 함수**까지 폭넓게 배웁니다.

## 정리
함수를 만들고 사용하는 법을 깨우쳤다는 것은, 드디어 코드를 한 줄씩 엮다 마는 초보를 벗어나 '나만의 도구 상자'를 직접 설계하는 설계자 영역에 들어섰다는 뜻입니다. 앞으로 작성될 모든 긴 코드와 복잡한 알고리즘은 전부 함수 단위로 찢기고 조립되어 관리될 것입니다.
