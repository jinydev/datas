# 3.1.6 다양한 연산자

컴퓨터 프로그래밍에서 데이터를 조작하고 계산하기 위해 없어서는 안 될 요소가 바로 연산자(Operators)입니다. 파이썬에서 제공하는 기본적이면서도 강력한 연산자들을 학습합니다.

## 연산자의 탄색
우리가 일반적으로 알고 있는 사칙연산 `+`, `-`, `*`, `/` 는 수학의 탄생, 역사와도 관계가 있습니다. 

- `+` : 덧셈
- `-` : 뺄셈
- `*` : 곱셈
- `/` : 나눗셈

## 산술 연산자

가장 기본적인 수학 계산을 수행합니다. 

### 실습
실습을 통하여 산술 연산자 데이터 타입을 확인해 봅시다.
```python
a = 10
b = 3.3

print("a + b =", a + b)   # 덧셈 (13.3)
print("a - b =", a - b)   # 뺄셈 (6.7)
print("a * b =", a * b)   # 곱셈 (33.0)
print("a / b =", a / b)   # 나눗셈 (실수로 반환, 3.0303...)
print("a % b =", a % b)   # 나머지 연산 (modulo)
print("a // b =", a // b) # 몫 연산자 (integer division, 결과는 3.0)
print("a ** b =", a ** b) # 지수 연산자 (a의 b제곱)

print(3 * (4 - 2))        # 괄호를 이용한 순서 제어 (결과: 6)
```

## 비교 연산자

비교 연산자는 두 값의 관계를 평가하여 논리형 데이터인 `True` 또는 `False`를 반환합니다. 이 연산자들은 뒤에서 배울 조건문(if)과 반복문(while)에서 핵심적인 흐름 제어 역할을 수행합니다.

| 연산자 | 기호 | 예시 | 의미 |
| :--: | :--: | :--- | :--- |
| **같음** | `==` | `a == b` | a와 b가 정확히 같은가? |
| **다름** | `!=` | `a != b` | a와 b가 다른가? |
| **크다 / 작다** | `>`, `<` | `a > b` | a가 b보다 큰가? |
| **크거나 같다 / 작거나 같다** | `>=`, `<=` | `a >= b` | a가 b보다 이상인가? |

### 실습
실습을 통하여 비교 연산자 데이터 타입을 확인해 봅시다.

```python
a = 10
b = 20

print("a == b:", a == b) # 결과: False
print("a != b:", a != b) # 결과: True
print("a < b:", a < b)   # 결과: True
```
### 비교연산자 적용사례

비교 연산자는 사용자 입력의 검증 필터링 등 실무 논리 구조에 자주 쓰입니다.
```python
# 3.1.6 사용자 나이 검증 예제
user_age = 25
is_adult = user_age >= 18
print("성인입니까?", is_adult) # 결과: True
```

## 논리 연산자

### 논리 연산자의 개발
논리 연산은 1930년대에 수학자 조지 불이 개발한 불대수학에서 시작되었습니다. 불대수학은 논리학과 수학을 결합한 학문으로, 논리적인 명제를 수학적으로 표현하고 계산하는 방법을 다룹니다. 


### 논리 연산 처리
논리 연산자는 위에서 배운 불리언(Boolean) 값들을 재조합하여 더 복잡한 조건을 설계할 때 사용됩니다. AND, OR, NOT 세 가지의 핵심 연산자가 있습니다.

- **`and` 연산자**: 좌/우측의 평가 값이 **모두 참(`True`)일 때만** `True`를 반환합니다.
- **`or` 연산자**: 좌/우측의 평가 값 중 **하나라도 참(`True`)이면** `True`를 반환합니다.
- **`not` 연산자**: 원래의 불리언 값을 정반대로 뒤집습니다. (참을 거짓으로, 거짓을 참으로)

### 논리 연산자 실습
실습을 통하여 논리 연산자 데이터 타입을 확인해 봅시다.

```python
a = True
b = False

print("a and b:", a and b)   # 결과: False (둘 다 참이어야 하므로)
print("a or b:", a or b)     # 결과: True (하나라도 참이므로)
print("not a:", not a)       # 결과: False (a 값을 반전)

# 3.1.6 복합 논리 평가
price = 12000
quantity = 3
total_price = price * quantity

# 3.1.6 금액이 10000 이상, 50000 이하인 경우의 논리 평가
is_valid_range = (total_price >= 10000) and (total_price <= 50000)
print("허용 금액대 여부:", is_valid_range) # 결과: True (36000원이기 때문)
```


## 파이썬 주요 연산자 요약

| 분류 | 파이썬 연산자 | 타 언어(C/R) 비교 참고 |
| --- | --- | --- |
| 사칙연산 | `+`, `-`, `*`, `/` | C 언어와 일치 |
| 나머지 / 몫 / 지수 | `%`, `//`, `**` | R언어: `%%`, `%/%`, `^` |
| 관계 연산자 | `>`, `<`, `>=`, `<=`, `==`, `!=` | C 언어와 일치 |
| 논리 연산자 | `and`, `or`, `not` | C 언어: `&&`, `||`, `!` |
| 대입 연산자 | `=` | R언어: `<-` |


## 연산자 우선순위
연산자는 우선순위가 존재 합니다. 그리고 왼쪽에서 오른쪽으로 적용합니다. 
여러 연산자가 혼합되어 있을 때는 수학에서와 동일하게 소괄호 `()`를 사용하여 연산의 우선순위를 명확하게 강제할 수 있습니다.

일반적으로 우리가 알고 있는 연산자는 중위 표기법을 사용합니다. 하지만 컴퓨터에서는 후위 표기법을 사용하면 더 명확하게 연산의 우선순위를 표현할 수 있습니다.
