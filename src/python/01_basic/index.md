# 3.1 파이썬 언어 기초 (Python Basics)

> **학습 목표**
> 1. 파이썬의 기본 언어 구조와 실행 흐름을 명확히 이해합니다.
> 2. 데이터를 담는 **그릇인 변수(Variable)**와 데이터의 종류를 뜻하는 **자료형(Data Type)**의 개념을 완벽하게 체득합니다.
> 3. 컴퓨터의 본질인 **연산(Operation)**과 프로그래밍 입출력 방식, 그리고 데이터 분석의 핵심인 **문자열(String)** 처리 기법을 깊이 있게 다룹니다.

## 1. 파이썬의 시작 (Hello World)

프로그래밍의 오랜 전통에 따라 우리의 첫 번째 코드는 화면에 인사를 출력하는 것입니다.

```python
print("Hello, Python!")
```

- `print()`: 괄호 안의 내용을 화면(터미널)에 띄워주는 가장 기초적인 **함수(Function)**입니다. 코드의 실행 결과를 직접 눈으로 확인할 수 있게 해줍니다.
- `"Hello, Python!"`: 따옴표(`"` 또는 `'`)로 묶인 텍스트 데이터 덩어리를 **문자열(String)**이라고 부릅니다.

---

## 2. 변수와 기본 자료형 (Variables & Data Types)

데이터 분석은 곧 수많은 데이터를 분류하고 처리하는 과정입니다. 데이터를 다루기 위해서는 데이터를 담아둘 **이름표가 붙은 상자**가 필수적입니다.

### 2.1. 변수 (Variable)
변수는 여러분이 다룰 데이터를 임시로 보관하는 메모리 상의 **가상의 그릇**입니다. 직관적인 이름을 붙여 목적을 분명히 하는 것이 코딩의 첫걸음입니다.

```python
name = "Antigravity"
level = 1
hp = 100.5
```

- **할당 연산자 (`=`)**: 파이썬에서 `=`는 수학에서의 "같다"는 의미가 아닙니다. **"오른쪽의 값(데이터)을 왼쪽의 변수 이름표를 단 상자에 집어넣어라(할당하라)"**는 대입의 의미를 갖습니다.

### 2.2. 자료형 (Data Type)
그릇에 담기는 내용물이 물인지, 모래인지에 따라 활용법이 다르듯, 변수에 담기는 데이터에도 뚜렷한 종류가 있습니다. 이를 자료형이라고 합니다.

| 자료형 | 영문 명칭 | 설명 | 예시 |
| :--- | :--- | :--- | :--- |
| **int** | Integer | **정수**: 소수점이 없는 깔끔한 숫자입니다. 횟수나 개수 등을 셀 때 사용됩니다. | `1`, `-5`, `100` |
| **float** | Floating-point | **실수**: 소수점을 포함하는 정밀한 숫자입니다. 평균값, 비율 산출 등에 사용됩니다. | `3.14`, `100.5`, `-0.01` |
| **str** | String | **문자열**: 글자들의 집합입니다. 무조건 따옴표로 감싸야 합니다. | `"Hello"`, `'Python'` |
| **bool** | Boolean | **논리값**: 어떤 조건이 참(`True`)인지 거짓(`False`)인지를 판별합니다. | `True`, `False` |

```python
# 3.1 자료형 확인하기: type() 함수를 통해 변수의 내부 데이터 종류를 파악할 수 있습니다.
print(type(name))   # <class 'str'> (문자열)
print(type(level))  # <class 'int'> (정수)
```

## 3. 기본 사칙연산과 산술 (Basic Operations)

컴퓨터(Computer)의 본질은 '계산기(Compute-er)'입니다. 파이썬은 세상에서 가장 강력하고 다루기 쉬운 계산기입니다.

```python
a = 10
b = 3

print(a + b)  # 덧셈: 13
print(a - b)  # 뺄셈: 7
print(a * b)  # 곱셈: 30
print(a / b)  # 나눗셈: 3.3333... (일반적인 나눗셈, 결과는 항상 실수(float)로 반환됩니다)
print(a // b) # 몫: 3 (소수점을 버리고 정수 몫만 반환하는 정수 나눗셈입니다)
print(a % b)  # 나머지: 1 (매우 중요! 특정한 배수나 짝수/홀수를 판별할 때 핵심적으로 쓰입니다)
print(a ** b) # 거듭제곱: 1000 (10의 3승을 의미합니다)
```

## 4. 문자열 데이터 다루기 (String Manipulation)

데이터 분석 실무에서는 숫자를 다루는 것 이상으로 고객의 리뷰, 이름, 주소 등 텍스트 데이터를 처리하는 비중이 막대합니다. 따라서 문자열 조작 능력은 매우 중요합니다.

### 4.1. 문자열 연결 (Concatenation)
더하기 기호(`+`)를 숫자 사이에 쓰면 산술 덧셈이 되지만, 문자열 사이에 쓰면 글자들을 이어 붙입니다.

```python
first_name = "Harry"
last_name = "Potter"
full_name = first_name + " " + last_name
print(full_name) # Harry Potter (중간에 띄어쓰기 문자열 " "을 추가했습니다)
```

### 4.2. 포맷팅 (f-string)
파이썬 3.6부터 도입된 최강의 문자열 치환 기능입니다. 긴 문장 안에 변수들의 값을 매끄럽게 끼워 넣고자 할 때 사용합니다. 큰따옴표 앞에 `f`를 붙이고 조립할 변수를 `{}` 안에 넣으면 자동으로 문장이 완성됩니다.

```python
name = "Iron Man"
age = 45
print(f"I am {name}, and I am {age} years old.")
# 3.1 출력: I am Iron Man, and I am 45 years old.
```

### 4.3. 인덱싱과 슬라이싱 (Indexing & Slicing)
문자열은 여러 글자가 기차 객실처럼 줄지어 연결된 형태입니다. 특정 위치의 글자만 콕 집어내거나(인덱싱), 글자의 일부 구간을 잘라내는(슬라이싱) 강력한 기법입니다.

> ⚠️ **주의사항**: 파이썬을 포함한 대부분의 프로그래밍 언어에서 숫자는 1이 아닌 **0부터 셉니다.**

```python
text = "ABCDEFG"

# 3.1 인덱싱 (Indexing): 특정 '한 글자' 추출
print(text[0])   # 첫 번째 글자 (인덱스 0): 'A'
print(text[1])   # 두 번째 글자 (인덱스 1): 'B'
print(text[-1])  # 마지막 글자 (인덱스 순서를 거꾸로 세면 -1부터 시작): 'G'

# 3.1 슬라이싱 (Slicing): 특정 '구간' 추출 [시작위치 : 끝위치값(포함 안 됨)]
print(text[0:3]) # 0번 인덱스부터 3번 '직전'인 2번(0, 1, 2)까지만 잘라냅니다: 'ABC'
print(text[3:])  # 3번 인덱스부터 문자열의 끝까지 모두 가져옵니다: 'DEFG'
```
