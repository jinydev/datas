# 3.1.5 자료형(Data Types)의 이해

## 학습목표
본 장에서는 숫자형(정수, 실수, 복소수), 문자열형, 논리형 등 파이썬이 데이터를 어떻게 다른 형태로 기억하고 다루는지 기본 자료형의 종류를 체계적으로 파악합니다. 더 나아가, 단일 데이터를 넘어 여러 값을 묶어서 보관할 수 있는 컬렉션 구조(리스트, 튜플, 딕셔너리, 집합)에 대한 기본적인 감각을 미리 익혀봅니다.

파이썬은 데이터를 저장하고 처리하는 데 다양한 데이터 타입을 지원하여, 프로그래머가 데이터를 효율적으로 관리하고 조작할 수 있게 합니다. 

## 기본 자료형
파이썬에서 사용되는 주요 기본 자료형들과 각각의 특징을 정리하면 아래 표와 같습니다.

| 데이터 타입    | 키워드    | 설명                                 | 특징                   |
| :------------- | :-------- | :----------------------------------- | :--------------------- |
| **숫자형**     | `int`, `float`, `complex`   | 정수, 부동 소수점 수, 복소수 포함 | 변경 불가능(immutable) |
| **문자열형**   | `str`     | 텍스트 데이터를 저장                 | 변경 불가능(immutable) |
| **논리형**     | `bool`    | 참(True) 또는 거짓(False) 값         | 변경 불가능(immutable) |
| **컬렉션형**   | `list`, `tuple`, `dict`, `set` | 여러 값을 하나로 묶어 관리           | 변경 또는 변경 불가 혼재 |

`type()` 함수를 사용하면 변수에 할당된 값의 자료형 클래스를 실시간으로 확인할 수 있습니다.

## 수의 체계와 숫자형 데이터 타입

파이썬이 숫자를 다루는 방식을 이해하려면, 먼저 우리가 일상적으로 사용하는 **'수의 체계(Number System)'**를 짚고 넘어갈 필요가 있습니다.

### 1) 정수 (Integer, `int`)
정수는 소수점이 없는 딱 떨어지는 깔끔한 숫자입니다. 
- 양의 정수(`1, 2, 3...`), 영(`0`), 그리고 음의 정수(`-1, -2, -3...`)를 모두 포함합니다.
- 파이썬의 정수형(`int`)은 메모리가 허용하는 한 **무한대에 가까운 엄청나게 큰 숫자**도 오차 없이 정확하게 계산하고 저장할 수 있는 강력한 특징이 있습니다.

### 2) 실수 (Real Number, `float`)
실수는 정수와 정수 사이에 무한히 존재하는, 소수점을 포함하는 숫자입니다. (`3.14`, `-0.001` 등)
- 파이썬에서는 실수를 **부동 소수점(Floating Point)**이라는 방식으로 메모리에 저장하며, 데이터 타입 이름도 `float`를 사용합니다.

### 💡 심화: 부동 소수점(Floating Point)의 원리와 정확성의 한계

컴퓨터는 모든 데이터를 `0`과 `1`로 이루어진 이진수로 기억합니다. 인간이 쓰는 십진수 실수를 이진수로 완벽하게 변환하다 보면, 가끔 `1/3 = 0.3333...`처럼 무한 소수가 되어 끝이 나지 않는 경우가 발생합니다.

이때 컴퓨터는 한정된 메모리 조각(보통 64비트) 안에 숫자를 욱여넣기 위해 **소수점의 위치를 둥둥 떠다니게 이동시켜(부동, 浮動, Floating) 근사치로 잘라내어 저장**합니다. `1.234 × 10³` 같은 수학의 지수 표기법 원리와 같습니다.

이 '근사치 저장'이라는 태생적 한계 때문에 코딩 시 아주 유명하고 골치 아픈 **부동 소수점 오차**가 발생합니다.

```python
# 3.1.5 부동 소수점 오차 실습
print(0.1 + 0.2)
```
여러분의 직관상 위 계산은 무조건 `0.3`이 나와야 하지만, 실제 파이썬(그리고 대부분의 언어)에서 결과를 출력해 보면 `0.30000000000000004`라는 미세한 찌꺼기 오차가 발생합니다. 

따라서 실제 금융권의 돈 계산이나 우주 공학처럼 극도로 정밀한 소수점 처리가 필요할 때는 기본 `float` 대신 `decimal`이나 `fractions`와 같은 정밀 연산 전용 파이썬 외부 모듈을 사용해야 함을 꼭 기억해 두세요!

### 실습: 정수와 실수 확인하기
가장 흔히 사용하는 정수와 실수 타입을 실습 환경에서 확인해 보겠습니다.

```python
# 3.1.5 int: 정수형
a = 10
print(a, "는", type(a), "형식입니다.")

# 3.1.5 float: 부동 소수점형 (실수)
x = 2.718
print(x, "는", type(x), "형식입니다.")
```
**출력:**
```
10 는 <class 'int'> 형식입니다.
2.718 는 <class 'float'> 형식입니다.
```

## 복소수 (Complex Numbers)

파이썬은 데이터 과학과 공학 연산에 최적화된 언어답게, 다른 일반적인 언어들과 달리 **복소수(`complex`)**를 언어 차원석 기본 자료형으로 완벽하게 지원합니다.

### 1) 복소수란 무엇인가요?
복소수는 **실수부(Real Number)**와 상상 속의 숫자인 **허수부(Imaginary Number)**의 합으로 이루어진 수입니다. 중고등학교 수학 시간에 배운 `x² = -1`을 만족하는 해인 허수 단위 `i`를 포함하는 숫자입니다. 
(단, 파이썬과 공학 분야에서는 전류 기호 `i`와의 혼동을 피하기 위해 허수 단위를 **`j`** 또는 **`J`**로 표기합니다.)

*   표현식: `a + bj` (여기서 `a`는 실수부, `b`는 허수부)

### 2) 왜 복소수를 사용하나요?
일반적인 웹 개발이나 앱 개발에서는 복소수를 볼 일이 거의 없습니다. 하지만 **데이터 분석, 양자 컴퓨터, 전자파 통신, 신호 처리(푸리에 변환), 유체 역학** 등 고도의 과학/공학 시뮬레이션에서는 파동(Wave)이나 회전(Rotation), 주파수의 위상(Phase)을 동시에 2차원 평면에 깔끔하게 모델링할 수 있는 복소수가 필수적인 수학적 무기입니다.

### 3) 파이썬에서는 어떻게 적용하나요?

파이썬에서는 변수에 `숫자 + 문자 j`를 붙이는 것만으로 즉시 복소수 객체가 생성됩니다. 내장 메서드를 통해 실수부와 허수부를 따로 분리하거나, 켤레복소수(켤레)를 쉽게 뽑아낼 수 있습니다.

#### 실습
실습을 통하여 복소수 데이터 타입을 생성하고 내부 속성을 확인해 봅시다.

```python
# 3.1.5 complex: 복소수 생성
c1 = 3 + 4j
c2 = complex(1, -2) # 내장 함수를 사용한 생성 (1 - 2j)

print(c1, "는", type(c1), "형식입니다.")

# 복소수의 사칙연산 (파이썬이 알아서 처리함)
print("덧셈:", c1 + c2)
print("곱셈:", c1 * c2)

# 복소수 객체의 유용한 내장 속성과 메서드
print("실수부만 추출:", c1.real)
print("허수부만 추출:", c1.imag)
print("켤레복소수(허수부 부호 반전):", c1.conjugate())
```
**출력:**
```
(3+4j) 는 <class 'complex'> 형식입니다.
덧셈: (4+2j)
곱셈: (11-2j)
실수부만 추출: 3.0
허수부만 추출: 4.0
켤레복소수(허수부 부호 반전): (3-4j)
```

## 문자열형 데이터 타입 (`str`)

문자형 데이터는 단일 문자 또는 문자열을 저장하며, 작은 따옴표(`'`) 또는 큰 따옴표(`"`)로 묶어서 표현합니다. 여러 줄에 걸친 문자열을 생성할 때는 세 개의 따옴표(`'''` 또는 `"""`)를 사용합니다. 문자열은 변경 불가능한(immutable) 객체입니다.

### 실습
실습을 통하여 문자열형 데이터 타입을 확인해 봅시다.

```python
s = "VS Code"
ml_str = """이것은
여러 줄로 작성된
문자열입니다."""

print(s, type(s))
```
**출력:**
```
VS Code <class 'str'>
```

> **Tip:** 문자열은 `+` 연산자를 사용해 결합하고, `*` 연산자를 사용해 반복 출력할 수 있습니다. 
> 예: `"하" * 3` -> `하하하`

## 논리형 데이터 타입 (`bool`)

논리형은 참(`True`) 또는 거짓(`False`)만을 값으로 가지며, 조건문과 논리 연산에서 핵심적인 역할을 합니다. 파이썬의 `True`, `False`는 대문자로 시작하며, 내부적으로 `True`는 숫자 1, `False`는 숫자 0으로 처리될 수도 있습니다.

```python
b = True
print(b, type(b))

# 3.1.5 논리형 연산 활용 (True는 1로 처리됨)
print("True 더하기 True:", b + b)
```
**출력:**
```
True <class 'bool'>
True 더하기 True: 2
```

논리형 값은 주로 비교 연산(예: `10 > 5`)이나 논리 연산의 결과로 생성되어 `if`문 등 프로그램의 분기를 결정하게 됩니다.

## 간단한 컬렉션 타입 소개

단일 값이 아닌 여러 개의 값을 저장하는 데이터 구조를 컬렉션이라고 부릅니다. 
뒤의 [자료구조] 단원에서 아주 깊이 있게 다루게 됩니다.

- **리스트(`list`)**: `[1, 2, 3]` 형태. 순서가 있고, 나중에 요소를 수정/추가/삭제할 수 있는 동적 배열 형식 (Mutable).
- **튜플(`tuple`)**: `(1, 2, 3)` 형태. 리스트와 비슷하지만 생성 후에 값을 변경할 수 없는 불변 객체 (Immutable).
- **딕셔너리(`dict`)**: `{'이름': '파이썬', '나이': 30}` 형태. 키(Key)와 값(Value)의 쌍으로 데이터를 빠르게 찾는 구조 (Mutable).
- **집합(`set`)**: `{1, 2, 3}` 형태. 중복을 허용하지 않고 순서가 없는 수학적 집합과 동일한 자료구조 (Mutable).

## 부록: 파이썬의 날짜 자료형

파이썬에서 날짜 정보는 기본 자료형으로 내장되어 있지 않지만, 강력한 기본 모듈인 `datetime`을 불러와서(import) 해결할 수 있습니다.

```python
import datetime as dt

# 3.1.5 현재 시간을 가져오기
now = dt.datetime.now()
print("오늘:", now.year, now.month, now.day)
print("타입:", type(now))

# 3.1.5 특정 날짜 지정하기
d = dt.datetime(2026, 3, 1)
print(d, '요일:', d.weekday()) # 0:월요일 ~ 6:일요일
```
**출력 (예시):**
```
오늘: 2026 3 1
타입: <class 'datetime.datetime'>
2026-03-01 00:00:00 요일: 6
```

## 정리
이 장을 통해 우리는 파이썬이 데이터를 얼마나 정밀하고 다양한 형태로 분류해두었는지(숫자, 문자, 참/거짓)를 실습해 보았습니다. 코딩 시 `type()` 함수를 써서 현재 내가 다루고 있는 데이터의 근본적인 그릇 형태가 무엇인지 파악하는 습관을 기르는 것은 예상치 못한 치명적인 데이터 연산 에러를 막는 가장 든든한 방패가 됩니다.
