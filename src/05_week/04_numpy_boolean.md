# 5주차 4강: 조건으로 검색하기 (Boolean Indexing)

## 6.1.1. 마스크(Mask)의 개념

> **Boolean Indexing**: 배열에 **True/False 조건문**을 씌워서, True인 값만 걸러내는 기술입니다.


<br>

---

<br>

### 6.1.1.1. 시나리오: 고레벨 몬스터 찾기
몬스터들의 레벨 데이터가 있습니다. 레벨 50 이상인 '보스급' 몬스터만 찾고 싶습니다.

```python
import numpy as np

# 몬스터 레벨 데이터
levels = np.array([10, 55, 3, 70, 25, 90, 40])

# 1. 조건 마스크 생성 (50 이상인가?)
mask = levels >= 50
print("마스크:", mask)
# [False  True False  True False  True False]

# 2. 마스크를 씌워서 데이터 필터링
bossES = levels[mask]
print("보스 몬스터 레벨:", bossES)
# [55 70 90]
```

## 6.1.2. 다중 조건 사용하기

조건이 여러 개일 때는 비트 연산자를 사용합니다.
- `&` (AND): 둘 다 만족할 때
- `|` (OR): 둘 중 하나라도 만족할 때
- `~` (NOT): 반대일 때


<br>

---

<br>

### 6.1.2.1. 시나리오: 사냥 가능한 몬스터
레벨이 20 이상이고 50 이하인, "적당한" 사냥감을 찾아봅시다.

```python
# (레벨 >= 20) AND (레벨 <= 50)
target_mask = (levels >= 20) & (levels <= 50)

targets = levels[target_mask]
print("사냥 타겟:", targets)
# [25 40]
```

> **주의**: 파이썬의 `and`, `or` 예약어가 아니라, 기호 `&`, `|`를 써야 합니다! 또한 각 조건은 반드시 **괄호 `()`**로 묶어줘야 합니다.

## 정리 (Summary)

이 강의에서 배운 핵심 내용을 요약해 봅시다.

*   **[핵심 1]**: **불리언 인덱싱(Boolean Indexing)**은 `True`/`False`로 구성된 마스크를 사용해 데이터를 필터링합니다.
*   **[핵심 2]**: 비교 연산자(`>`, `<` 등)로 조건을 만들고, 비트 연산자(`&`, `|`, `~`)로 조건을 결합합니다.
*   **[핵심 3]**: 원본 데이터를 변경하지 않고 **복사본(Copy)**을 반환하므로 안전하게 데이터를 추출할 수 있습니다.
