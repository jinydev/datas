# 1.6.2 데이터 시각화와 통제

## 1.6.2 보이지 않는 데이터를 보이게 만드는 힘
데이터는 숫자의 나열이고 엑셀 시트 100만 줄입니다. 사람은 결코 이걸 눈으로 보고 깨달음을 얻을 수 없습니다.
![데이터 처리](../img/raw_data_traces.png)
이것을 가장 직관적으로, 한눈에 보이게 색을 입히고 선을 긋는 과정이 바로 '데이터 시각화(Data Visualization)'입니다.

## 1.6.2 데이터를 지휘하는 데이터 오케스트레이션
여러 팀에서 만들어진 데이터베이스를 한 곳으로 깔끔하게 모으고 파이프라인으로 연결하는 기술을 요즘 현업에서는 '데이터 오케스트레이션'이라고 부릅니다. 
![데이터 개념](../img/data_orchestration_conductor.png)
오케스트라 지휘자처럼 각각 다른 성격을 지닌 데이터를 한 곡의 멋진 음악으로 합주해 내는 것입니다.
