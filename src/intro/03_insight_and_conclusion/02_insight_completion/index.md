# 1.3.2 데이터 추출기와 컴퓨터, 그리고 통찰력

## 기계가 아닌 인간의 통찰력이 완성을 만듭니다.
파이썬 라이브러리가 자동으로 평균을 구해주고, AI가 멋지게 그래프를 그려준다고 해서 사람의 역할이 축소되는 것은 절대 아닙니다. 쏟아져 나온 그 화려한 그래프의 결과물을 보고, 궁극적으로 **"그래서 우리는 내일 어떤 전략적 행동을 취해야 하는가?"라는 지혜로운 통찰은 인간만의 고유한 영역**입니다.
