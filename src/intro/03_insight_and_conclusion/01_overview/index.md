# 1.3.1 데이터 추출기와 컴퓨터, 그리고 통찰력

## "데이터가 자원이라면 분석은 추출기다"
산업 전방위적으로 살펴보았듯, 데이터를 수집만 하고 가만히 내버려 두면 그저 컴퓨터 용량(하드디스크)만 차지하는 비싼 쓰레기일 뿐입니다. 데이터 분석 기술과 파이썬 코딩은 이 쓰레기를 진짜 돈과 생명이 되는 금괴로 제련해 내는 '추출기'의 역할을 합니다.
