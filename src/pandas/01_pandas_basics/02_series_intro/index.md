# 6.1.2 시리즈 개요

![시리즈 소개](img/series_intro_webtoon.png)

**[비유로 이해하기: 이름표가 붙은 1차원 데이터]**
시리즈(Series)는 데이터들이 한 줄로 나열되어 있는데, 각 데이터마다 식별 가능한 **꼬리표(Label Index)**가 붙어있는 구조입니다. 엑셀 표에서 딱 **하나의 열(Column)**만 뚝 떼어온 것과 완전히 같습니다.

시리즈(Series)는 numpy에서 제공하는 1차원 배열과 비슷하지만 각 데이터의 의미를 표시하는 인덱스(index)를 붙일 수 있다. 데이터 자체는 값(values)이라고 한다. 또한, 시리즈의 이름인 `name` 속성도 제공한다.

![시리즈 개요와 주요 속성 index, values, name](img/page_004_2.jpg)

시리즈는 테이블 형태의 데이터프레임(DataFrame)의 열에 해당하는 자료 구조이다. 즉 여러 시리즈가 모여 데이터프레임이 생성될 수 있다.

![시리즈와 데이터프레임](img/page_005.jpg)
