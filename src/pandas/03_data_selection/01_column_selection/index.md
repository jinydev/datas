# 6.3.1 DataFrame의 다양한 참조

## 6.3.1 열 참조

간단히 성적 처리를 위한 데이터프레임 `pf`를 정의하자.

```python
pf = pd.DataFrame(data = [[25, 35, 8, 18],
                          [18, 27, 10, 20],
                          [17, 17, 10, 19],
                          [12, 22, 9, 20],
                          [22, 34, 8, 16]],
                  index=['윤일형', '강수희', '홍소희', '유한빈', '신수빈'],
                  columns=['중간', '기말', '과제', '출석'])
print(pf)
```

`pf['중간']`으로 열 참조가 가능하다.

```python
pf['중간']
```
열이 1개인 반환 자료형은 `Series`이다.

`pf[['중간', '기말', '출석']]`으로 여러 열로 구성된 데이터프레임을 반환받을 수 있다.

```python
pf[['기말', '출석', '중간']]
```

다음처럼 열 1개도 목록으로 입력하면 반환 값은 데이터프레임이다.

```python
pf[['기말']]
```
