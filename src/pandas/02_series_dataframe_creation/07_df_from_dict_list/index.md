# 6.2.7 여러 딕셔너리의 목록 데이터 사용

다음 코드처럼 딕셔너리의 목록으로도 데이터프레임의 생성이 가능하다. 하나의 dict 항목이 하나의 행에 해당한다. 키에 해당하는 값이 없으면 결측값인 `NaN`이 입력된다.

```python
import pandas as pd
df = pd.DataFrame([{'국어': 70, '수학': 89, '영어': 90},
                   {'국어': 80, '영어': 99},
                   {'수학': 66, '영어': 78}])
print(df)
```
