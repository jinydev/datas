# 11주차: 머신러닝 개요 및 선형 회귀

## 학습 목표 (Learning Objectives)
1.  **머신러닝의 이해**: 기계가 데이터를 통해 학습한다는 것이 무엇인지, 지도학습과 비지도학습의 차이를 이해합니다.
2.  **데이터 분리 (Data Split)**: 모델을 믿을 수 있게 검증하기 위해 데이터를 학습용(Train)과 테스트용(Test)으로 나누는 이유를 배웁니다.
3.  **선형 회귀 (Linear Regression)**: 가장 기초적인 예측 모델인 선형 회귀를 실습하고 결과를 해석합니다.

## 강의 내용 (Course Content)

### [01. 기계가 학습을 한다고? (ML Overview)](./01_theory.html)
- 규칙 기반(Rule-based) vs 머신러닝(Machine Learning).
- 지도학습(타겟 있음) vs 비지도학습(타겟 없음).
- Scikit-learn 라이브러리 소개.

### [02. 미래를 예측해 보자 (Linear Regression)](./02_practice.html)
- 데이터 분리 (`train_test_split`).
- 모델 학습 (`fit`)과 예측 (`predict`).
- 시나리오: "공부 시간에 따른 시험 점수 예측".

### [03. 얼마나 잘 맞췄을까? (Evaluation)](./03_advanced.html)
- 회귀 모델의 평가지표 삼총사: MSE, RMSE, R2 Score.
- 과대적합(Overfitting)과 과소적합(Underfitting) 개념 맛보기.
