# 4.3.1 배열 연산

## 4.3.1 배열과 스칼라와의 연산

다음은 원소 3개로 구성된 1차원 배열 `a`이다.

```python
import numpy as np
a = np.arange(3)

a
```
**출력:**
```
array([0, 1, 2])
```

다음 코드처럼 배열과 스칼라 값과의 일반 연산은 모든 원소에 연산을 수행한 같은 모양의 배열을 반환한다.

```python
print(a + 2)
print(a - 2)
print(a * 2)
print(a / 2)
```
**출력:**
```
[2 3 4]
[-2 -1  0]
[0 2 4]
[0.  0.5 1. ]
```
