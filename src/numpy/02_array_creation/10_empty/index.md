# 4.2.10 초기화되지 않은 배열을 생성하는 empty()

함수 `empty(shape)`는 임의의 값을 반영한 모양 `shape`인 배열을 생성한다. `shape=[2, 2]`로 2행 2열을, `shape=3`으로 원소 3개의 1차원 배열을 생성한다. 원소값은 예측할 수 없다.

```python
import numpy as np

np.empty([2, 2])
```
**출력 (예시):**
```
array([[6.23042070e-307, 4.67296746e-307],
       [1.69121096e-306, 1.42176308e-312]])
```

```python
np.empty(3)
```
**출력 (예시):**
```
array([1.47466827e-311, 4.94065646e-324, 0.00000000e+000])
```

다음 코드는 정수인 2행 3열 2차원 배열을 생성한다.

```python
np.empty([2, 3], dtype=int)
```
**출력 (예시):**
```
array([[         0, 1072693248,          0],
       [1072693248,          0, 1072693248]])
```
