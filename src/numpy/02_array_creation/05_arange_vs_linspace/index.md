# 4.2.5 내장함수 arange()와 linspace() 비교

내장함수 `numpy.arange()`와 `numpy.linspace()`는 모두 시작과 종료 지점 사이에서 동일한 길이의 하위 간격으로 분할하는 수가 나열된 1차원 배열을 반환한다. `arange()`는 하위 간격의 크기를 매개변수 `step`에 명시한다. 반면, `linspace()`는 일정한 간격을 구성하는 값인 샘플 수를 매개변수 `num`에 지정한다. 함수 `arange()`의 `step` 기본값이 1이며, `linspace()`의 `num` 기본값은 50이다.

다음 `np.arange(0.1, 0.4, 0.1)` 코드로 0.4까지 배열이 구성되는 문제점이 있을 수 있다. 수학적 정의로는 `array([0.1, 0.2, 0.3])`이어야 한다. 컴퓨터에서 `float` 표현의 부정확성 때문에 발생한다.

```python
np.arange(0.1, 0.4, 0.1)
```
**출력:**
```
array([0.1, 0.2, 0.3, 0.4])
```

위의 경우, 정수로 계산해서 10으로 나누면 원하는 결과를 얻을 수 있다. 나중에 자세히 학습하겠지만 `numpy`의 배열과 실수값의 연산은 모든 배열 원소에 연산한 결과의 배열이 반환된다.

```python
np.arange(1, 4, 1) / 10.0
```
**출력:**
```
array([0.1, 0.2, 0.3])
```

또한, 다음처럼 `numpy.linspace()`를 사용해도 위와 같은 결과를 얻을 수 있다.

```python
np.linspace(0.1, 0.4, 3, endpoint=False)
```
**출력:**
```
array([0.1, 0.2, 0.3])
```

다음으로는 0.4를 포함해 4개의 수로 분할하는 배열을 반환한다.

```python
np.linspace(0.1, 0.4, 4)
```
**출력:**
```
array([0.1, 0.2, 0.3, 0.4])
```
