# 02. 데이터 컨테이너 (리스트의 세계)

## 1. 수집의 필요성
데이터 1개는 변수 1개로 충분합니다. 하지만 데이터가 100만 개라면? 변수 100만 개를 만들 수는 없습니다. 이를 담을 '컨테이너'가 필요합니다.

## 2. 리스트(List)의 도입
리스트는 순서가 있는 데이터의 집합입니다. 대괄호 `[]`로 감싸서 표현합니다.
`scores = [90, 80, 70]`

## 3. Python 문법 집중: 대괄호 `[]`
Python에서 대괄호는 리스트를 생성하거나, 리스트의 특정 요소에 접근할 때 사용됩니다.

## 4. 이종 데이터 허용
C언어의 배열(Array)과 달리, Python 리스트는 서로 다른 타입의 데이터를 함께 담을 수 있습니다.
`mixed = [1, "two", 3.0]` (하지만 분석용으로는 권장하지 않음).

## 5. 인덱싱 (Indexing)
각 데이터는 자신만의 주소(번지수)를 가집니다. 이를 인덱스라고 합니다.
`scores[0]`은 첫 번째 요소입니다.

## 6. 0-Based Indexing
컴퓨터 과학에서 숫자는 0부터 셉니다. 이는 메모리 오프셋(Offset) 개념에서 유래했습니다.

## 7. 음수 인덱싱 (Negative Indexing)
Python만의 강력한 기능입니다. `scores[-1]`은 뒤에서 첫 번째, 즉 마지막 요소를 의미합니다.

## 8. 슬라이싱 (Slicing)
데이터의 일부분을 잘라내는 기술입니다.
`scores[0:2]`는 0번부터 2번 **직전**까지(0, 1) 가져옵니다.

## 9. Python 문법 집중: `len()` 함수
리스트의 길이를 알려줍니다. 반복문 등에서 필수적으로 사용됩니다.

## 10. 가변성 (Mutability)
리스트의 내용은 변경 가능합니다.
`scores[1] = 100`.

## 11. 데이터 추가 (`append`)
`scores.append(85)`는 리스트의 **맨 끝**에 데이터를 덧붙입니다.

## 12. 데이터 삭제 (`remove`, `pop`)
특정 값을 찾아 지우거나, 특정 위치의 값을 꺼냅니다.

## 13. 리스트의 시각화
Matplotlib의 `plot` 함수는 리스트를 입력으로 받는 것을 가장 좋아합니다.

## 14. X와 Y 리스트
두 개의 리스트 `x`, `y`가 짝을 이루면 좌표평면에 점을 찍을 수 있습니다.

## 15. `range()` 함수
연속된 숫자를 자동으로 생성해줍니다. `list(range(10))`은 0부터 9까지의 리스트를 만듭니다.

## 16. Python 문법 집중: step 인자
`range(시작, 끝, 간격)`.
`range(0, 10, 2)`는 짝수 리스트를 만듭니다.

## 17. 리스트 병합 (`+`)
`list1 + list2`는 두 리스트를 이어 붙입니다.

## 18. 리스트 곱셈 (`*`)
`[0] * 5`는 `[0, 0, 0, 0, 0]`을 만듭니다. 초기화할 때 유용합니다.

## 19. 응용 예제: 성적 변화 추이 시각화
1학기 성적과 2학기 성적 리스트를 합쳐서 1년 간의 변화를 그려봅시다.

## 20. 결론
리스트는 가장 기초적이면서도 강력한 자료구조입니다. 데이터를 순서대로 관리하는 법을 익혔습니다.
