# 04. 자동화와 반복문 (Control Flow)

## 1. 자동화의 힘
컴퓨터의 가장 위대한 능력은 지치지 않고 단순 반복 작업을 수행하는 것입니다. "점 1000개를 찍어라"는 인간에겐 고역이지만 컴퓨터에겐 찰나의 순간입니다.

## 2. `for` 반복문의 구조
`for 아이템 in 집합:`
    `행동(아이템)`

## 3. Python 문법 집중: 들여쓰기 (Indentation)
Python은 중괄호 `{}` 대신 **들여쓰기(공백)**를 사용하여 코드의 블록을 구분합니다. 이는 강제적으로 가독성 좋은 코드를 작성하게 만듭니다.

## 4. 리스트 순회 (Iteration)
```python
fruits = ["사과", "바나나", "체리"]
for fruit in fruits:
    print(fruit)
```

## 5. `range()`와 함께 사용하기
특정 횟수만큼 반복하고 싶을 때 사용합니다.
`for i in range(10):` (10번 반복)

## 6. 변수 `i`의 의미
`i`는 Index 또는 Iterator의 약자입니다. 반복되는 수열의 현재 값을 담고 있는 변수입니다.

## 7. 누적 패턴 (Accumulation)
합계를 구하는 로직입니다.
1. `total = 0` (초기화)
2. `total = total + value` (반복문 안에서 누적)

## 8. Python 문법 집중: 복합 대입 연산자 `+=`
`total += value`는 `total = total + value`의 줄임말입니다.

## 9. 중첩 반복문 (Nested Loops)
반복문 안에 반복문이 들어갈 수 있습니다. 2차원 평면(행렬)이나 구구단을 출력할 때 사용합니다.
`for x in range(10): for y in range(10):`

## 10. 반복 중단 (`break`)
특정 조건에서 루프를 즉시 탈출합니다.

## 11. 건너뛰기 (`continue`)
이번 회차만 건너뛰고 다음 반복으로 넘어갑니다.

## 12. 리스트 내포 (List Comprehension)
반복문을 한 줄로 줄여 쓰는 파이썬스러운(Pythonic) 방식입니다.
`[x*2 for x in data]`

## 13. 응용 예제: 제너레이티브 아트
반복문을 사용하여 100개의 선을 미묘하게 다른 위치와 색상으로 그려 예술적인 패턴을 만들어 봅시다.

## 14. 알고리즘적 사고
반복문을 잘 쓰려면 패턴을 발견해야 합니다. "무엇이 변하고, 무엇이 그대로인가?"를 파악해야 합니다.

## 15. `zip()` 함수
두 개의 리스트를 동시에 묶어서 반복할 때 사용합니다.
`for x, y in zip(x_list, y_list):`

## 16. `enumerate()` 함수
값뿐만 아니라 순서(인덱스)도 함께 필요할 때 사용합니다.
`for i, value in enumerate(list):`

## 17. `while` 반복문
횟수가 아니라 **조건**이 만족될 때까지 반복합니다. `while count < 10:`.

## 18. 무한 루프의 위험
종료 조건이 영원히 충족되지 않으면 프로그램은 멈추지 않습니다(`Ctrl+C`로 강제 종료).

## 19. 반복문의 시각화
점이 하나씩 찍히는 과정을 상상해 보십시오. 우리는 그 최종 결과를 한 번에 플롯팅하는 것입니다.

## 20. 결론
반복문을 통해 우리는 1개의 로직을 100만 개의 데이터에 확장 적용할 수 있게 되었습니다. 이것이 스케일업(Scale-up)의 기초입니다.
