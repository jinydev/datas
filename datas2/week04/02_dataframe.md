# 4-2. 데이터프레임 만들기 (DataFrame Basic)

## 1. 데이터프레임 구조
엑셀 표와 똑같습니다.
*   **Column (열)**: 세로줄. 데이터의 종류 (예: 이름, 나이, 키)
*   **Row (행)**: 가로줄. 개별 데이터 (예: 철수 데이터)
*   **Index (인덱스)**: 행의 이름표 (주민번호 같은 것)

## 2. 표 만들기
딕셔너리(Dictionary)를 이용하면 쉽게 만들 수 있습니다.

```python
import pandas as pd # pd라는 별명 국룰

data = {
    '이름': ['아이언맨', '캡틴', '토르', '헐크'],
    '체력': [100, 120, 300, 500],
    '공격력': [200, 80, 150, 400]
}

# 표 생성!
df = pd.DataFrame(data)

print(df)
```

## 3. 시리즈(Series) vs 데이터프레임(DataFrame)
*   **Series**: 표에서 열 하나만 떼어낸 것 (1차원). Numpy 배열과 사촌.
*   **DataFrame**: Series가 여러 개 붙어있는 표 (2차원).

```python
# '이름' 열만 가져오면 시리즈가 됩니다.
print(df['이름'])
```

## 4. 핵심 정리
*   `pd.DataFrame(데이터)`: 표 만들기.
*   딕셔너리의 Key는 열 이름(Column)이 되고, Value는 데이터가 됩니다.
