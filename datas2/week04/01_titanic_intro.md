# 01. 비극의 데이터 (타이타닉 프로젝트 개요)

## 1. 역사상 가장 유명한 해난 사고
1912년 침몰한 타이타닉호는 데이터 사이언스 입문자들에게 가장 사랑받는 예제입니다. 우리는 생존자 데이터를 통해 "누가 살아남았는가?"를 예측하고 분석할 것입니다.

## 2. 프로젝트 목표
이 프로젝트의 목표는 단순히 생존율을 구하는 것이 아닙니다. 탐색적 데이터 분석(EDA)을 통해 숨겨진 패턴을 발견하고, 데이터에 기반한 가설을 검증하는 프로세스를 익히는 것입니다.

## 3. 데이터셋 구성 요소 (Data Dictionary)
*   **Survived**: 생존 여부 (0: 사망, 1: 생존) - Target Variable
*   **Pclass**: 티켓 등급 (1, 2, 3등석) - 사회적 지위의 대리 변수
*   **Sex**: 성별
*   **Age**: 나이
*   **SibSp**: 동승한 형제자매/배우자 수
*   **Parch**: 동승한 부모/자녀 수
*   **Fare**: 운임 요금
*   **Embarked**: 탑승 항구 (C, Q, S)

## 4. 분석 환경 준비
Seaborn 라이브러리에 내장된 타이타닉 데이터를 사용할 것입니다.
```python
import seaborn as sns
df = sns.load_dataset('titanic')
```

## 5. 데이터의 첫인상 확인 (`head`, `info`)
데이터를 로드하자마자 행 수, 결측치, 데이터 타입을 스캔합니다. `deck` 컬럼에 결측치가 많음을 발견하게 될 것입니다.

## 6. 결측치 지도 그리기
`missingno` 라이브러리나 히트맵을 이용해 결측치의 분포를 시각화합니다. 결측치 처리는 분석의 신뢰도를 결정합니다.

## 7. 가설 1: "여성과 아이가 먼저?"
영화 대사처럼 실제로 여성과 아이의 생존율이 높았는지 데이터로 증명해야 합니다.

## 8. 가설 2: "부자가 더 많이 살았을까?"
1등석 승객(Pclass 1)과 3등석 승객의 생존율 차이를 분석하여, 재력이 생존에 미친 영향을 파악합니다.

## 9. 범주형 데이터 분석 (Categorical Analysis)
성별, 등급, 탑승 항구 등은 범주형 데이터입니다. `value_counts()`와 `countplot`을 주력으로 사용합니다.

## 10. 수치형 데이터 분석 (Numerical Analysis)
나이, 요금은 수치형 데이터입니다. 히스토그램과 `distplot`으로 분포를 봅니다.

## 11. 상관관계 분석 (Correlation)
각 변수 간의 관계를 히트맵으로 그려 봅니다. 요금(Fare)과 등급(Pclass)은 강한 상관관계가 있을 것입니다.

## 12. 파생 변수 아이디어
`SibSp`와 `Parch`를 합쳐 `FamilySize`라는 새로운 변수를 만들면 어떨까요? 혼자 탄 사람(Solo)의 생존율은?

## 13. 시각화 전략: Seaborn
Matplotlib보다 더 통계적으로 특화된 Seaborn을 사용하여 세련된 그래프를 그립니다.

## 14. 데이터 전처리 계획

3.  인코딩 (문자를 숫자로)
