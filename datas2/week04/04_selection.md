# 4-4. 데이터 확인하고 선택하기 (Selection)

## 1. 원하는 부분만 콕 집기
엑셀에서는 마우스로 드래그하면 되지만, 코딩에서는 주소를 불러야 합니다.
두 가지 방법만 기억하세요. `loc`와 `iloc`.

## 2. 이름으로 찾기 (loc - Location)
"사람 이름(Index)과 열 이름(Column)으로 찾기"

```python
# df.loc[행이름, 열이름]
# 모든 행(:), '체력' 열만 가져오기
print(df.loc[:, '체력']) 
```

## 3. 번호로 찾기 (iloc - Integer Location)
"몇 번째 줄, 몇 번째 칸인지 숫자로 찾기" (Numpy랑 똑같습니다!)

```python
# 0번째 행, 1번째 열
print(df.iloc[0, 1])
```

## 4. 데이터 훑어보기
데이터를 불러오자마자 해야 할 건강검진 3종 세트입니다.

1.  `df.head()`: 앞부분 보기 (생김새 확인)
2.  `df.shape`: 크기 확인 (몇 행 몇 열?)
3.  `df.info()`: 정보 확인 (빈 값은 없는지, 숫자인지 문자인지)

```python
df.info()
```

## 5. 핵심 정리
*   `loc[행, 열]`: 이름으로 선택.
*   `iloc[행번호, 열번호]`: 숫자로 선택.
*   `df.info()`: 데이터 요약 정보 확인.
