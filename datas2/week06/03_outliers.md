# 6-3. 이상한 값 찾아내기 (Outliers)

## 1. 이상치(Outlier)란?
전체 패턴에서 벗어난 튀는 값입니다.
*   실수일 수도 있음 (입력 오류)
*   진짜일 수도 있음 (천재, 로또 당첨) -> 특이점 발견의 기회!

## 2. 눈으로 찾기 (Boxplot)
상자 그림(Boxplot)을 그리면 이상치를 점으로 찍어줍니다.

```python
import matplotlib.pyplot as plt

# 100점 만점인데 500점이 있다?
scores = [10, 20, 30, 40, 50, 60, 70, 80, 90, 500]

# 상자그림
plt.boxplot(scores)
plt.show()
```

## 3. 코드로 찾기 (조건문, 필터링)
"평균에서 너무 멀리 떨어진 애들은 이상하다"

```python
import pandas as pd
df = pd.DataFrame({'점수': scores})

# 100점 넘는 애들 찾기
print(df[df['점수'] > 100])
```

## 4. 어떻게 처리해요?
*   삭제한다. (오류인 경우)
*   최대값으로 깎는다. (Clipping, 500 -> 100)
*   그냥 둔다. (이상치 자체가 분석 대상인 경우)

## 5. 핵심 정리
*   **Boxplot**: 이상치를 시각적으로 찾는 최고의 도구.
*   무조건 지우는 게 답은 아닙니다. 왜 튀는지 고민해봐야 합니다.
