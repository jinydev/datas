# 5-4. 조건에 맞는 데이터 찾기 (Filtering)

## 1. 엑셀의 필터 기능
"점수가 90점 이상인 사람만 보여줘"
데이터 분석의 핵심은 원하는 데이터만 쏙 골라내는 능력입니다.
이것을 **필터링(Filtering)** 또는 **불리언 인덱싱(Boolean Indexing)**이라고 합니다.

## 2. 사용법 (매우 중요 ⭐⭐⭐)
문법이 조금 특이하니 잘 봐두세요.
`df[ 조건식 ]` 형태입니다.

```python
import pandas as pd

data = {'이름': ['철수', '영희', '민수', '지수'], '점수': [80, 95, 70, 100]}
df = pd.DataFrame(data)

# 1. 조건(질문) 만들기
# 점수가 90점 이상인가요?
condition = df['점수'] >= 90
print(condition) # [False, True, False, True]

# 2. 질문을 데이터프레임에 던지기
# True인 행만 살아남습니다.
elite = df[condition]
# 또는 한번에: df[df['점수'] >= 90]

print(elite)
```

## 3. 조건이 여러 개일 때
*   그리고(AND): `&`
*   또는(OR): `|`

```python
# 점수가 80점 이상이고 95점 이하인 사람
df[ (df['점수'] >= 80) & (df['점수'] <= 95) ]
```

## 4. 핵심 정리
*   `df[조건]`: 조건이 참(True)인 행만 뽑아낸다.
*   데이터 분석의 80%는 이 필터링 작업입니다.
