# 10-2. 상관계수 구하기 (Corr Method)

## 1. 판다스로 계산하기
데이터프레임에 `.corr()`만 붙이면 숫자형 변수들끼리의 상관계수를 알아서 다 구해줍니다.

```python
df.corr()
```

## 2. 결과 해석하기
*   자기 자신과의 상관계수는 항상 **1**입니다. (대각선)
*   절댓값이 클수록 관계가 깊습니다.
*   0.7 이상이면 "매우 강한 관계"라고 봅니다.
