# 9-4. 이동평균과 추세 (Rolling)

## 1. 데이터가 너무 튀어요
주식 차트를 보면 하루하루 등락이 심해서 전체적인 흐름이 잘 안 보일 때가 있습니다.
이럴 때 **"부드럽게 펴주는 다림질"**이 필요합니다.

## 2. 이동 평균 (Moving Average)
최근 N일간의 평균을 구해서 점을 찍습니다.
*   **5일 이동평균**: 단기 흐름
*   **60일 이동평균**: 장기 흐름

```python
# 20일치씩 밀면서 평균 구하기
df['ma20'] = df['Close'].rolling(window=20).mean()
```

## 3. 골든 크로스와 데드 크로스
단기 이평선이 장기 이평선을 뚫고 올라가면 '골든 크로스(매수 신호)', 내려가면 '데드 크로스(매도 신호)'라고 부르기도 합니다.

## 4. [응용 예제] 골든 크로스 찾기
애플 주가에서 20일 이평선이 60일 이평선을 돌파하는 순간을 찾아봅시다.
