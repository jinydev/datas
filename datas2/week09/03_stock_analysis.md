# 9-3. 주식 데이터 분석 (Finance Data)

## 1. 실제 데이터 가져오기
금융 데이터는 시계열 분석을 공부하기에 가장 좋은 재료입니다.
`yfinance` 라이브러리를 사용하면 야후 파이낸스에서 실제 주식 데이터를 무료로 받아올 수 있습니다.

> **주의**: 실습환경(Colab)에 `yfinance`가 없다면 `!pip install yfinance`로 설치해야 합니다.

## 2. 주가 차트 그리기
애플(AAPL), 삼성전자(005930.KS) 등의 티커(Ticker)만 알면 데이터를 불러올 수 있습니다.
