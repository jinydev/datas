# 03. 데이터 가공 (변수 파생과 삭제)

## 1. 있는 그대로의 데이터는 부족하다
분석가는 주어진 데이터에 만족하지 않고, 기존 데이터를 조합하여 새로운 통찰을 만들어내야 합니다. 이를 파생 변수(Derived Variable)라고 합니다.

## 2. 새로운 열 만들기 (기본)
`df['새컬럼'] = 값`
아주 간단합니다. 딕셔너리에 키를 추가하는 것과 같습니다. 모든 행에 동일한 값이 들어갑니다.

## 3. 사칙연산을 이용한 파생 변수
`df['총점'] = df['국어'] + df['수학']`
벡터화 연산(Vectorized Operation) 덕분에 반복문 없이도 열 단위로 더하기가 수행됩니다.

## 4. 논리 연산을 이용한 파생 변수
`df['합격여부'] = df['점수'] > 80`
True/False 값이 채워진 새로운 열을 만듭니다.

## 5. `apply()` 함수 - 판다스의 만능 칼
단순 사칙연산으로 해결되지 않는 복잡한 로직(if문 등)을 적용하고 싶을 때 사용합니다.
`df['컬럼'].apply(함수명)`

## 6. Python 문법 집중: 람다 함수 (Lambda)
한 줄짜리 간단한 함수를 만들 때 씁니다.
`df['점수'].apply(lambda x: x + 5)` (모든 점수에 5점 가산).

## 7. 조건에 따른 값 할당 (예: 학점 부여)
90점 이상은 A, 나머지는 B... 이런 로직은 함수로 정의하고 `apply`를 씁니다.

## 8. 컬럼 삭제하기 (`drop`)
불필요한 열은 메모리를 차지하고 분석을 방해합니다.
`df.drop('컬럼명', axis=1)`

## 9. axis의 의미 (0 vs 1)
판다스에서 가장 헷갈리는 부분입니다.
*   `axis=0`: 행(Row) 방향. (가로줄을 삭제)
*   `axis=1`: 열(Column) 방향. (세로줄을 삭제)

## 10. `inplace` 옵션
`df.drop(...)`은 삭제된 **사본(Copy)**을 반환합니다. 원본을 진짜로 지우려면 `inplace=True`를 주거나, 다시 변수에 할당해야 합니다.
`df = df.drop(...)`

## 11. 컬럼 이름 변경 (`rename`)
컬럼명이 'Unnamed: 0' 처럼 지저분하다면 바꿔줘야 합니다.
`df.rename(columns={'옛이름': '새이름'})`

## 12. 자료형 변환 (`astype`)
숫자가 문자로 저장되어 있다면 계산이 안 됩니다.
`df['가격'] = df['가격'].astype(int)`

## 13. 날짜형 변환 (`to_datetime`) - 매우 중요
'2024-01-01'은 문자열입니다. 이를 날짜 객체로 바꿔야 '1주일 뒤' 같은 시간 계산이 가능합니다.
`pd.to_datetime(df['날짜'])`

## 14. 날짜 정보 추출 (`dt` accessor)
날짜형으로 변환된 열에서는 `.dt`를 통해 연/월/요일을 쉽게 뽑을 수 있습니다.
`df['날짜'].dt.year`, `df['날짜'].dt.weekday_name`.

## 15. 문자열 처리 (`str` accessor) 복습
주소에서 '서울시'만 떼어내기.
`df['주소'].str.split().str[0]`

## 16. 구간 나누기 (`cut`)
연속형 변수(나이)를 범주형 변수(연령대: 10대, 20대...)로 바꿀 때 사용합니다. 히스토그램 그릴 때 유용합니다.

## 17. 더미 변수 (One-Hot Encoding)
머신러닝을 위해서는 '남자/여자' 같은 문자를 0과 1로 바꿔야 합니다.
`pd.get_dummies(df)`

## 18. 결측치 처리 (`fillna`)
비어있는 값을 삭제하지 않고 평균값이나 0으로 채우는 전략입니다.
`df.fillna(0)`

## 19. 응용 예제: 쇼핑몰 데이터 가공
주문 날짜에서 요일을 뽑고, 구매 금액에 따라 'VIP' 등급을 매기는 로직을 구현해 봅니다.

## 20. 결론
데이터 가공은 요리로 치면 재료 손질입니다. 잘 다듬어진 재료(Clean Data)가 맛있는 분석 결과를 만듭니다.
