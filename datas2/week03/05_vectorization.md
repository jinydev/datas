# 3-5. 한번에 계산하기 (Vectorization)

## 1. 브로드캐스팅 (Broadcasting)
작은 녀석을 큰 녀석 크기에 맞춰서 뿌려주는 기능입니다.

```python
import numpy as np

arr = np.array([1, 2, 3])

# 배열에 숫자 하나 더하기
# 원래는 크기가 안 맞지만, 알아서 [10, 10, 10]으로 생각해서 더해줍니다.
print(arr + 10) 
# 결과: [11, 12, 13]
```

## 2. 모든 데이터 필터링하기
데이터가 100만 개인데, 50보다 큰 숫자만 찾고 싶다면?
Numpy는 질문(조건)을 던지면 True/False로 답해줍니다.

```python
scores = np.array([40, 90, 80, 50, 65])

# 60점 이상인가?
print(scores >= 60)
# 결과: [False  True  True False  True]
```

## 3. 핵심 정리
*   **브로드캐스팅**: 모양이 다른 배열끼리 연산할 때 알아서 크기를 맞춰주는 기능.
*   반복문을 쓰지 않아도 알아서 모든 원소에 계산을 적용해주어 코드가 매우 짧아집니다.
