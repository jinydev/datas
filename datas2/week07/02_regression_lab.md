# 02. Lab: 물고기 무게 예측하기 (회귀 실습)

## 1. 실험실 목표
길이, 높이, 두께 정보를 이용하여 물고기의 무게(Weight)를 예측하는 회귀 모델을 만듭니다. 농어(Perch) 데이터를 사용합니다.

## 2. 데이터 준비
직접 측정한 농어 데이터를 넘파이 배열로 준비합니다.
`perch_length`, `perch_weight`.

## 3. 훈련 세트와 테스트 세트 분리
회귀에서도 일반화 성능 평가를 위해 데이터 분리는 필수입니다. `train_test_split`.

## 4. 단순 선형 회귀 도전 (길이 -> 무게)
가장 직관적인 변수 하나(길이)만 사용하여 무게를 예측해 봅니다.
결과 그래프를 그려보면 직선으로는 농어의 무게 증가(곡선 형태)를 설명하기 역부족임을 알게 됩니다. (과소적합)

## 5. 다항 회귀 (Polynomial Regression) 도입
길이의 제곱($L^2$) 항을 추가합니다.
무게는 길이의 제곱(혹은 세제곱)에 비례한다는 부피의 법칙을 반영합니다.
확연히 좋아진 $R^2$ 점수를 확인합니다.

## 6. 다중 회귀 (Multiple Regression)
이제 길이뿐만 아니라 높이와 두께까지 모든 변수를 투입합니다.
판다스로 데이터를 불러와서 `to_numpy()`로 변환합니다.

## 7. 특성 공학 (Feature Engineering)
변수 3개를 불려서 ($x_1^2, x_1x_2, ...$) 복잡한 상호작용을 모델링합니다. `PolynomialFeatures(degree=5)`.
변수가 55개로 늘어납니다.

## 8. 데이터 정규화 (StandardScaler)
특성이 많고 규제 모델을 쓸 때는 스케일링이 필수입니다. 릿지/라쏘 회귀를 위해 표준화를 진행합니다.

## 9. 릿지 회귀 (Ridge Regression)
너무 복잡해진 모델(과대적합)을 잡기 위해 L2 규제를 적용합니다. `alpha` 값을 조절하며 최적의 규제 강도를 찾습니다.

## 10. 라쏘 회귀 (Lasso Regression)
L1 규제를 적용하여 불필요한 변수의 가중치를 0으로 만듭니다. 어떤 변수가 살아남았는지 확인하여 가장 중요한 특성을 파악합니다.

## 11. 하이퍼파라미터 튜닝
`alpha` 값에 따른 훈련 세트 점수와 테스트 세트 점수의 변화를 그래프로 그려봅니다. 두 점수가 가장 가깝고 높은 지점을 찾습니다.

## 12. 과대적합 vs 과소적합 판단
훈련 점수는 높은데 테스트 점수가 낮다면? -> 과대적합 (규제 강화 필요).
둘 다 낮다면? -> 과소적합 (모델 복잡도 증가 필요).

## 13. 최종 모델 선정
가장 성능이 좋은 모델(예: 릿지, alpha=0.1)을 최종 선택하고 저장합니다.

## 14. 새로운 데이터 예측
"길이 50cm짜리 농어는 몇 kg일까?"
모델은 이전에 없던 거대한 농어의 무게도 합리적으로 예측해야 합니다.

## 15. 시각화: 실제값 vs 예측값
테스트 세트의 실제 무게와 모델이 예측한 무게를 산점도로 그려서 $y=x$ 선 위에 잘 모이는지 확인합니다.

## 16. 잔차(Residual) 패턴 확인
잔차의 등분산성(Homoscedasticity) 등을 체크하여 회귀 분석의 가정을 만족하는지 봅니다.

## 17. 다른 지표들 (MAE, RMSE)
$R^2$ 외에 실제 오차(kg)가 평균적으로 얼마나 나는지 `mean_absolute_error`로 확인합니다. "평균 20g 오차가 난다."

## 18. 실험: 선형 회귀로 분류를 할 수 있을까?
(심화) 생존 여부(0, 1)를 회귀로 예측하면 0.7, -0.2 같은 값이 나옵니다. 이것이 왜 문제가 되는지, 그래서 로지스틱 회귀가 필요한 이유를 상기합니다.

## 19. Python 문법 집중: `np.column_stack`
여러 1차원 배열을 나란히 붙여서 2차원 데이터셋을 만드는 넘파이 팁을 배웁니다.

## 20. 결론
회귀 분석은 입력 변수들의 조합으로 결과를 설명하는 가장 과학적인 방법입니다. 우리는 규제(Regularization)를 통해 복잡한 모델을 통제하는 법을 배웠습니다.
