# 01. 디지털 연구실 구축 (환경 설정 및 오리엔테이션)

## 1. 데이터 사이언스의 세계에 오신 것을 환영합니다
데이터 사이언스 입문 과정을 시작하게 된 예비 분석가 여러분을 환영합니다. 이 과정은 데이터 분석의 이론적 이해와 실무적 적용 사이의 간극을 줄이기 위해 설계되었습니다.

## 2. 데이터의 시대
우리는 데이터가 '새로운 석유'라고 불리는 시대에 살고 있습니다. 그러나 정제되지 않은 원유가 쓸모없듯이, 가공되지 않은 데이터는 단순한 숫자의 나열일 뿐입니다. 우리는 이를 가치 있는 정보로 변환하는 기술을 배울 것입니다.

## 3. 강의 철학: 시각화 우선 (Visual First)
우리는 지루한 문법부터 배우지 않습니다. 우리는 **보는 것**부터 시작합니다. 시각화는 인간의 인지 능력 중 가장 강력한 도구이기 때문입니다.

## 4. 우리의 도구: Python
Python(파이썬)은 데이터 과학의 공용어(Lingua Franca)입니다. 강력하고, 유연하며, 거대한 생태계가 이를 뒷받침하고 있습니다.

## 5. 실습 환경: Google Colab
우리는 클라우드 기반의 주피터 노트북 환경인 Google Colab을 사용합니다. 별도의 설치 없이 구글의 강력한 컴퓨팅 자원을 무료로 사용할 수 있습니다.

## 6. 셀(Cell)의 이해
노트북은 '셀'이라는 단위로 구성됩니다. 텍스트 셀(Markdown)은 설명을 위한 것이고, 코드 셀(Code)은 실제 명령을 실행하는 공간입니다.

## 7. Python 문법 집중: `print()` 함수
컴퓨터와 대화하기 위해서는 '함수'라는 명령어를 사용합니다. 가장 그 기초가 되는 것이 출력을 담당하는 `print()`입니다.

## 8. 코드 실행 실습
우리의 환경이 정상적인지 확인해 봅시다.
```python
print("Hello, Data Science!")
```

## 9. 문자열(String)의 이해
Python에서 글자는 **문자열(String)**이라고 부릅니다. 반드시 따옴표(`"` 또는 `'`)로 감싸야 합니다.

## 10. 흔한 실수: 따옴표 누락
`print(Hello)`라고 입력하면 `NameError`가 발생합니다. 파이썬은 따옴표가 없으면 이를 '변수'라고 착각하기 때문입니다.

## 11. `print` 내에서의 연산
`print()` 함수 내에서 직접 계산을 수행할 수도 있습니다.
```python
print(10 + 20)
```

## 12. 데이터 타입의 구분
`print("10 + 20")`과 `print(10 + 20)`은 무엇이 다를까요? 하나는 단순히 글자를 보여주는 것이고, 다른 하나는 수학적 계산을 수행하는 것입니다.

## 13. 라이브러리(Library)의 개념
Python의 진정한 힘은 방대한 **라이브러리**에서 나옵니다. 우리는 모든 기능을 처음부터 만들지 않고, 이미 만들어진 도구를 가져와서 사용합니다.

## 14. Python 문법 집중: `import` 문
라이브러리를 사용하기 위해서는 먼저 `import` 명령어로 불러와야 합니다.
```python
import matplotlib.pyplot as plt
```

## 15. 별칭(Alias) 시스템 (`as`)
매번 `matplotlib.pyplot`이라고 입력하는 것은 비효율적입니다. `as plt`를 붙여서 'plt'라는 별명으로 부르겠다고 선언합니다.

## 16. 첫 번째 그래프: `plot()` 함수
`plt.plot()`은 선 그래프를 그리는 가장 기본적인 명령어입니다.

## 17. 데이터 입력하기
그래프를 그리려면 데이터가 필요합니다. 리스트 형태의 숫자를 전달해 봅시다.
```python
plt.plot([1, 2, 3, 4])
```

## 18. `show()` 명령어의 역할
어떤 환경에서는 `plt.show()`를 호출하기 전까지 그래프가 보이지 않을 수 있습니다. 이는 "준비된 시각화 자료를 지금 화면에 출력하라"는 명령어입니다.

## 19. 응용 예제: 선형 추세 시각화
단순히 증가하는 선형 수열을 시각화해 봅시다.
```python
plt.plot([10, 20, 30, 40])
plt.show()
```
