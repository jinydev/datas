# 02. 데이터의 기하학 (도형 그리기 기초)

## 1. 선(Line) 그 이상을 향하여
선 그래프는 데이터의 흐름을 보여주는 가장 기본적인 형태입니다. 하지만 데이터 시각화를 위해서는 좌표 평면 위에 원하는 형태를 자유자재로 그릴 수 있어야 합니다.

## 2. 좌표계(Coordinate System)의 이해
우리가 사용하는 그래프는 데카르트 좌표계(Cartesian coordinate system)를 기반으로 합니다. 모든 점은 $(x, y)$의 튜플로 표현됩니다.

## 3. 다중 데이터 포인트
Matplotlib은 여러 개의 좌표를 리스트 형태로 한 번에 처리합니다.
`x = [1, 2, 3]`
`y = [1, 4, 9]`

## 4. Python 문법 집중: 주석(Comment) `#`
코드는 기계뿐만 아니라 사람도 읽을 수 있어야 합니다. `#` 기호 뒤에 오는 텍스트는 실행되지 않고 설명으로 남습니다.
```python
# 이것은 주석입니다. 컴퓨터는 이 줄을 무시합니다.
```

## 5. 삼각형 그리기
삼각형은 3개의 꼭짓점(Vertex)을 가집니다. 도형을 닫기(Close) 위해서는 시작점으로 다시 돌아와야 합니다.

## 6. 꼭짓점 정의하기
삼각형을 그려봅시다: $(1,1) \rightarrow (2,3) \rightarrow (3,1) \rightarrow (1,1)$.

## 7. Python 문법 집중: 다중 할당
한 줄에 여러 변수를 동시에 할당할 수 있습니다.
```python
x, y = [1, 2, 3, 1], [1, 3, 1, 1]
```

## 8. 삼각형 시각화
`plt.plot(x, y)`는 순서대로 점들을 연결하여 도형을 완성합니다.

## 9. 사각형 그리기
사각형은 4개의 꼭짓점을 가집니다. $(1,1) \rightarrow (1,2) \rightarrow (2,2) \rightarrow (2,1) \rightarrow (1,1)$.

## 10. 순서(Order)의 중요성
만약 꼭짓점의 순서가 뒤섞이면, 선이 꼬이면서 사각형이 아닌 복잡한 다각형이나 모래시계 모양이 그려질 것입니다.

## 11. 마커(Marker) 추가하기
데이터가 존재하는 정확한 지점을 표시하고 싶을 때 `marker` 옵션을 사용합니다.
`plt.plot(x, y, marker='o')`

## 12. 마커의 종류
*   `'o'`: 원 (Circle)
*   `'s'`: 사각형 (Square)
*   `'^'`: 삼각형 (Triangle)

## 13. 선 스타일(Linestyle)
점들을 잇는 선의 모양을 변경할 수 있습니다.
*   `'-'`: 실선 (기본값)
*   `'--'`: 파선 (Dashed)

## 14. 색상(Color)의 활용
색상은 정보를 구분하는 중요한 수단입니다. `color='red'` 또는 `c='r'`과 같이 지정합니다.

## 15. Python 문법 집중: 함수의 인자(Arguments)
`plot()` 함수는 여러 입력을 받습니다. `x`, `y`는 **위치 인자**로 순서가 중요하며, `color` 등은 **키워드 인자**로 이름을 명시해야 합니다.

## 16. 축 레이블(Label) 달기
축의 의미를 설명하지 않는 그래프는 모호합니다.
`plt.xlabel("Time")`, `plt.ylabel("Value")`

## 17. 제목(Title) 추가하기
`plt.title("기하학적 도형")`으로 그래프의 주제를 명시합니다.

## 18. 그리드(Grid) 추가
`plt.grid(True)`를 통해 격자무늬를 추가하면 값을 읽기가 훨씬 수월해집니다.

## 19. 응용 예제: 집(House) 그리기 
사각형과 삼각형을 조합하여 간단한 집 모양을 그려봅시다. 이는 좌표 감각을 익히는 데 매우 좋은 훈련입니다.

## 20. 결론
