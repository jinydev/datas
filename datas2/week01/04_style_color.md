# 04. 심미적 요소와 스타일 (Aesthetics and Style)

## 1. 심미성의 가치
데이터 시각화에서 아름다움은 사치가 아닙니다. 이는 **인지 부하(Cognitive Load)**를 줄여주는 핵심 요소입니다. 잘 디자인된 그래프는 데이터를 더 쉽게 이해하게 만듭니다.

## 2. 데이터와 색채 이론
색상은 범주(Category)를 구분하거나 강도(Intensity)를 표현하는 데 사용됩니다.

## 3. Matplotlib 색상 이름
우리는 이름을 사용하여 색을 지정할 수 있습니다: `'red'`, `'blue'`, `'green'`, `'cyan'`, `'black'`, `'white'` 등.

## 4. 16진수 색상 코드 (Hex Codes)
더 정밀한 색상이 필요할 땐 `#FF5733`과 같은 Hex 코드를 사용합니다. 수백만 가지 색상을 표현할 수 있습니다.

## 5. Python 문법 집중: 인수로서의 문자열
함수에 `'red'`를 전달할 때, 우리는 '문자열 객체'를 전달하는 것입니다.

## 6. 선 굵기 (Line Width)
`linewidth` (또는 `lw`)로 선의 두께를 조절합니다.
`plt.plot(..., linewidth=3)`

## 7. 선 스타일 (Line Styles)
`linestyle` (또는 `ls`)로 선의 패턴을 바꿉니다.
`:` (점선), `-.` (쇄선) 등이 있습니다.

## 8. 마커의 크기와 색상
`markersize` (또는 `ms`): 점의 크기.
`markerfacecolor` (또는 `mfc`): 점 내부의 색상.
`markeredgecolor` (또는 `mec`): 점 테두리의 색상.

## 9. Python 문법 집중: 이름 있는 인자 (Named Arguments)
`plot(x, y, a=1, b=2)` 처럼 이름을 지정해서 값을 넘기는 방식입니다. 이 경우 순서는 중요하지 않습니다.

## 10. 투명도 (Alpha)
`alpha` 값은 0(투명)에서 1(불투명) 사이로 조절합니다. 데이터 포인트가 겹칠 때 유용합니다.

## 11. 겹쳐 그리기
`plt.show()`를 호출하기 전에 `plt.plot()`을 여러 번 호출하면, 하나의 그래프 위에 여러 선이 겹쳐서 그려집니다.

## 12. 자동 색상 순환
색상을 지정하지 않으면 Matplotlib은 자동으로 파랑 -> 주황 -> 초록 순서로 색을 바꿔가며 그립니다.

## 13. 범례 (Legend)
`plt.legend()`는 그래프의 지도와 같습니다. 이를 표시하려면 `plot()` 함수 안에 `label='설명'`이 있어야 합니다.

## 14. 범례 위치 조절
`plt.legend(loc='upper right')`와 같이 위치를 수동으로 지정할 수 있습니다.

## 15. 그림 크기 (Figure Size)
`plt.figure(figsize=(10, 5))`로 도화지의 가로, 세로 비율을 설정합니다.

## 16. 축 범위 제한 (Axis Limits)
자동 스케일링이 실제 데이터를 왜곡할 때가 있습니다. `plt.xlim(0, 100)`으로 범위를 강제할 수 있습니다.

## 17. 축 레이블 스타일링
`plt.xlabel("Date", fontsize=14, color='gray')`처럼 폰트 크기와 색상도 변경 가능합니다.

## 18. Python 문법 집중: 메서드(Methods)
`plt`는 모듈(객체)입니다. `xlabel`은 그 안에 속한 **메서드(함수)**입니다. 점(`.`)을 찍어서 접근합니다.

## 19. 응용 예제: 하트 모양 스타일링
수학 공식을 이용해 하트 모양을 그리고, 핑크색과 투명도를 활용해 예쁘게 꾸며봅시다.

## 20. 결론
이제 우리는 데이터를 어떻게 보여줄지 완전히 제어할 수 있게 되었습니다. 시각적 명확성은 분석의 설득력을 높여줍니다.
