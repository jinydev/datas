# 06. Lab: 벡터 그래픽 엔지니어링 (이니셜 그리기)

## 1. 실험실 목표
이번 세션의 목표는 좌표 조작, 플롯팅, 스타일링 기술을 종합하여 여러분의 '이니셜'을 벡터 그래픽으로 구현하는 것입니다.

## 2. 이론적 배경: 벡터 vs 래스터
우리는 픽셀(Raster)이 아닌 수학적 지시어(Vector)로 그래픽을 만듭니다. 덕분에 우리의 이니셜은 아무리 확대해도 깨지지 않습니다.

## 3. 프로젝트 요구사항
1.  자신의 이름 첫 스펠링을 그립니다.
2.  두 번째 스펠링을 옆에 그립니다.
3.  각 글자는 서로 다른 색상을 가져야 합니다.
4.  반드시 좌표계(Coordinates)를 통해 정의되어야 합니다.

## 4. 글자 'M'의 설계 (예시)
'M'의 기하학적 구조를 분해해 봅시다.
좌표 흐름: 좌하단 $\rightarrow$ 좌상단 $\rightarrow$ 중간하강 $\rightarrow$ 우상단 $\rightarrow$ 우하단.

## 5. 좌표 매핑
*   P1: (0, 0)
*   P2: (0, 10)
*   P3: (5, 5)
*   P4: (10, 10)
*   P5: (10, 0)

## 6. Python 구조: 리스트 활용
두 개의 리스트 `mx`와 `my`가 필요합니다.
`mx = [0, 0, 5, 10, 10]`
`my = [0, 10, 5, 10, 0]`

## 7. Python 문법 집중: 오프셋 변수
두 번째 글자 'A'를 그릴 때, 좌표 숫자를 새로 계산하는 것은 하수입니다. `offset` 변수를 사용하여 위치를 이동시켜야 합니다.

## 8. 오프셋 정의
`offset = 12` (M의 너비 + 여백).

## 9. 글자 'A'의 설계
흐름: 좌하단 $\rightarrow$ 상단중앙 $\rightarrow$ 우하단 $\rightarrow$ (펜 들어올리기) $\rightarrow$ 중간왼쪽 $\rightarrow$ 중간오른쪽.

## 10. '펜 들어올리기' 개념
`plt.plot()`은 기본적으로 선을 이어 그립니다. 선을 끊으려면 별도의 plot 명령어를 사용해야 합니다.

## 11. 오프셋을 적용한 좌표 계산
*   A_P1: (0 + offset, 0)
*   A_P2: (5 + offset, 10)
*   A_P3: (10 + offset, 0)

## 12. 'M' 그리기
`plt.plot(mx, my, color='navy', linewidth=5, label='M')`

## 13. 'A' 그리기 (메인 획)
`plt.plot(ax_main, ay_main, color='crimson', linewidth=5, label='A')`

## 14. 'A' 그리기 (가로 획)
이 부분은 별도의 객체(Object)로 그려줍니다.
`plt.plot(ax_cross, ay_cross, color='crimson', linewidth=5)`

## 15. 심미적 개선
꼭짓점(Vertex)마다 마커를 추가하여, 점과 점을 잇는다는 구조적 특징을 강조해 봅시다.

## 16. 비율 설정 (Aspect Ratio)
글자가 찌그러지지 않으려면 `plt.axis('equal')`이 필수입니다.

## 17. 문맥 추가
`plt.title("Vectorized Initials: M.A.")`
`plt.grid(True, alpha=0.2)`

## 18. 범례 문제 해결
'A'를 그릴 때 선을 두 번 사용했으므로 범례에 'A'가 두 번 나올 수 있습니다. 두 번째 선에는 `label`을 달지 않거나 `_nolegend_`를 사용합니다.
