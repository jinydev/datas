# 05. 캔버스와 좌표계 (스케일과 레이블)

## 1. 캔버스의 은유
`plt.figure()`를 호출하는 것은 그림을 그릴 캔버스의 크기를 고르는 것과 같습니다. 좌표계는 그 위에 그려진 격자눈금입니다.

## 2. 자동 스케일링 (Auto-Scaling)
Matplotlib은 기본적으로 똑똑합니다. 입력된 데이터가 모두 화면에 꽉 차게 들어오도록 X축과 Y축의 `lim` (한계)을 자동으로 조절합니다.

## 3. 왜곡의 문제
만약 X 범위가 0-10이고 Y 범위가 0-1000이라면, 그래프 모양은 정사각형일지라도 시각적 스케일은 심각하게 왜곡된 것입니다.

## 4. 한계 설정 (`xlim`, `ylim`)
우리는 이를 수동으로 덮어쓸 수 있습니다. `plt.xlim(0, 100)`은 축이 강제로 이 범위를 보여주게 만듭니다.

## 5. 균등 비율 (Equal Aspect Ratio)
기하학적 도형(원, 정사각형)을 그릴 때는 X축의 1단위와 Y축의 1단위가 시각적으로 동일해야 합니다.
`plt.axis('equal')` 명령어를 사용합니다.

## 6. Python 문법 집중: 데이터로서의 문자열
텍스트 레이블은 문자열입니다. "Math"라는 글자는 사실 'M', 'a', 't', 'h'라는 문자의 배열입니다.

## 7. 눈금 (Ticks)
축 위에 표시되는 작은 표시를 눈금(Tick)이라고 합니다. `xticks`와 `yticks`로 이를 제어합니다.

## 8. 눈금 커스터마이징
`plt.xticks([0, 50, 100])`라고 쓰면 X축에는 오직 이 세 개의 숫자만 표시됩니다.

## 9. 텍스트 눈금
숫자 대신 글자를 넣을 수도 있습니다.
`plt.xticks([1, 2], ['1월', '2월'])`

## 10. 텍스트 주석 (Annotation) 추가
`plt.text(x, y, "Hello")`를 사용하면 특정 좌표에 글자를 적을 수 있습니다.

## 11. Python 문법 집중: 이스케이프 문자
문자열 안에 따옴표를 넣으려면? `print("그가 \"안녕\"이라고 말했다")`. 역슬래시(`\`)를 사용합니다. 줄바꿈은 `\n`입니다.

## 12. 그리드 커스터마이징
`plt.grid(color='gray', linestyle='--')`. 격자도 선처럼 스타일링이 가능합니다.

## 13. 서브플롯 (Subplots)
여러 개의 그래프를 나란히 비교해야 할 때가 있습니다. 화면을 분할하는 기술입니다.
`plt.subplot(행, 열, 순서)`

## 14. 1x2 레이아웃 만들기
`plt.subplot(1, 2, 1)`: 왼쪽 영역 활성화.
`plt.subplot(1, 2, 2)`: 오른쪽 영역 활성화.

## 15. 이미지 저장 (`savefig`)
분석 결과를 파일로 저장합니다. `plt.savefig("my_graph.png")`.

## 16. 해상도 (DPI)
인쇄용 고해상도 이미지가 필요하다면 `dpi=300` 옵션을 줍니다. Dots Per Inch를 의미합니다.

## 17. 축 숨기기
`plt.axis('off')`는 프레임과 숫자를 모두 숨깁니다. 예술적인 그래프를 그릴 때 유용합니다.

## 18. Python 문법 집중: 임포트의 활용
랜덤성을 추가하기 위해 `import random`을 사용할 수 있습니다.

## 19. 응용 예제: 모눈종이 만들기
반복문과 그리드 설정을 이용하여 나만의 커스텀 모눈종이를 그려봅시다.
