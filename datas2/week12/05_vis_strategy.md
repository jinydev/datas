# 12-5. 그래프 전략 짜기

## 1. 무엇을 말하고 싶은가?
그래프는 여러분의 주장을 뒷받침하는 무기입니다.
*   "치킨집이 **점점 늘어나고 있다**" -> Line Plot (추세)
*   "강남구 **치킨집이 제일 많다**" -> Bar Chart (비교)
*   "치킨 가격과 판매량은 **반비례한다**" -> Scatter Plot (관계)

## 2. 색상 예쁘게 쓰기
