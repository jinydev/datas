# 12-3. 데이터 탐색하기 (EDA)

## 1. EDA(Exploratory Data Analysis)란?
데이터를 분석하기 전에 "일단 한 번 뜯어보는" 과정입니다.
데이터가 어떻게 생겼는지, 이상한 점은 없는지 확인해야 합니다.

## 2. 필수 체크리스트
1.  **크기 확인**: `df.shape` (몇 행 몇 열이지?)
2.  **정보 확인**: `df.info()` (숫자인가 문자인가? 결측치는 없나?)
3.  **통계 확인**: `df.describe()` (평균은? 최댓값은?)
4.  **찍어 보기**: `df.head()` (눈으로 직접 보기)
