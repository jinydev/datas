# 12-6. [실습] 프로젝트 제안서 작성

## 1. 제안서에 들어갈 내용
1.  **주제**: 무엇을 분석할 것인가?
2.  **배경**: 왜 이걸 분석하고 싶은가? (궁금증)
3.  **데이터**: 어디서 데이터를 구할 것인가? (출처)
4.  **예상 결과**: 분석을 통해 무엇을 알고 싶은가?

## 2. 미션: 나만의 제안서 작성하기
아래 템플릿을 복사해서 여러분만의 계획을 채워보세요.

```markdown
## 프로젝트 제목: (예: 서울시 배달 음식 주문 분석)
### 1. 분석 배경
- 평소에 배달 음식을 자주 시켜 먹는데, 동네마다 인기 메뉴가 다를지 궁금했다.

### 2. 데이터 출처
- SK텔레콤 빅데이터 허브 (무료 공개 데이터)

### 3. 분석 계획
- 동별로 가장 주문이 많은 메뉴 top 3 뽑기
- 요일별 주문량 변화 (불금엔 치킨?)

### 4. 예상 결과
- 강남구는 파스타, 관악구는 치킨이 많을 것 같다.
```
