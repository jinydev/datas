# 8-4. 범주 데이터 시각화 (Bar, Countplot)

## 1. 범주형 데이터(Categorical Data)란?
숫자가 아니라 이름으로 구분되는 데이터입니다.
*   성별 (남/여)
*   요일 (월/화/수...)
*   등급 (A/B/C)

## 2. 막대 그래프 (Barplot)
"A반 평균 점수 vs B반 평균 점수" 처럼 그룹별 **평균**을 비교할 때 씁니다.
Seaborn은 자동으로 평균을 계산해주고, 검은색 막대(오차 막대)도 그려줍니다.

## 3. 빈도 그래프 (Countplot)
"A등급은 몇 명이고, B등급은 몇 명이지?" 처럼 **개수**를 셀 때 씁니다.
