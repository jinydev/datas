# 8-1. [이론] 정보 시각화의 원리

## 1. 예쁜게 다가 아니다
시각화의 목적은 "예쁜 그림"이 아니라 **"정확한 정보 전달"**입니다.
*   **간결성**: 불필요한 장식 제거
*   **명확성**: 축 이름, 제목, 단위 적기
*   **적합성**: 데이터 성격에 맞는 그래프 고르기

## 2. 그래프 선택 가이드
*   **변화**를 보고 싶다 -> **Line Plot**
*   **비교**를 하고 싶다 -> **Bar Chart**
*   **분포**를 보고 싶다 -> **Histogram, Boxplot**
*   **관계**를 보고 싶다 -> **Scatter Plot**

## 3. Matplotlib vs Seaborn
*   Matplotlib: 파이썬 시각화의 할아버지. 자유도가 높지만 코드가 길다.
*   Seaborn: Matplotlib을 기반으로 만든 "요즘 스타일". 코드가 짧고 색감이 예쁘다.
