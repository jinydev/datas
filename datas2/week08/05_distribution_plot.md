# 8-5. 데이터 분포 시각화 (Box, Violin)

## 1. 평균의 함정에서 벗어나기
평균은 데이터를 하나로 요약해주지만, 전체 그림을 보여주진 못합니다.
데이터가 어디부터 어디까지 퍼져있는지 보여주는 **Boxplot**이 필요합니다.

## 2. 상자 그림 (Boxplot)
*   가운데 선: 중앙값 (Median)
*   상자: 데이터의 50%가 모여있는 구간 (Q1 ~ Q3)
*   수염(Whiskers): 정상 데이터 범위
*   점(Points): 이상치 (Outliers)

## 3. 바이올린 그림 (Violinplot)
Boxplot과 비슷하지만, 데이터가 뚱뚱한지 홀쭉한지를 더 예쁘게 보여줍니다.

## 4. [응용 예제] 요일별 팁 금액 분포
팁 데이터에서 어느 요일에 사람들이 팁을 가장 후하게(또는 짜게) 주는지 분포를 확인해봅시다.
