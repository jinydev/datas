10주차 2강: 줄 세우기 (Sorting & Stats)

10.2.1. 데이터 정렬하기 (sort_values)

랭킹 시스템: 게임에서 점수가 높은 순서대로 랭킹을 보여주는 것과 같습니다.

# 데이터 준비
df = pd.DataFrame({
    'Name': ['A', 'B', 'C', 'D'],
    'Score': [70, 95, 80, 60]
})

# 점수 기준 오름차순 (낮은 순 -> 높은 순)
print(df.sort_values(by='Score'))

# 점수 기준 내림차순 (높은 순 -> 낮은 순)
# ascending=False (오름차순 아님 = 내림차순)
ranked = df.sort_values(by='Score', ascending=False)
print(ranked)



10.2.2. 상관계수 확인하기 (corr)

Correlation: 두 변수가 얼마나 강한 관계를 맺고 있는지 -1 ~ 1 사이의 숫자로 보여줍니다.

  • 1: 완벽한 양의 관계 (키가 크면 몸무게도 많이 나간다)
  • -1: 완벽한 음의 관계 (운동을 많이 하면 몸무게가 줄어든다)
  • 0: 관계 없음
# 상관계수 행렬 출력
print(df.corr())
