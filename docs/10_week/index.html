10주차: 데이터 처리와 시각화 (Preprocessing & Visualization)

학습목표:

  • 다양한 연습용 데이터셋을 자유자재로 불러와 실습 환경을 구축합니다.
  • Seaborn 라이브러리를 활용하여 통계적 의미가 담긴 아름다운 시각화를 구현합니다.
  • 웹 데이터나 클립보드 등 다양한 경로의 외부 데이터를 효과적으로 가져오는 방법을 익힙니다.

10.1. 데이터셋 준비 (Datasets)

  • 01. 다양한 예제 데이터 (PyDataset)
    • PyDataset 라이브러리 설치 및 사용법
    • 주요 데이터셋 소개 (titanic, iris, tips)

10.2. 시본 기초 시각화 (Seaborn Basics)

  • 02. 시본 기초 및 관계 시각화
    • Matplotlib vs Seaborn
    • 스타일 설정 (set_theme)
    • 관계 그래프: scatterplot, lineplot

10.3. 분포 및 범주형 시각화 (Distribution & Categorical)

  • 03. 분포와 범주형 시각화
    • 분포: histplot (히스토그램)
    • 범주: barplot, countplot, boxplot, violinplot
    • 상관관계: heatmap

10.4. 외부 데이터 활용 (External Data)

  • 04. 외부 데이터 활용하기
    • 인터넷 URL에서 바로 읽기
    • read_clipboard(): 복사 붙여넣기 마법
    • 한글 인코딩 해결 (cp949, euc-kr)

10.5. 종합 실습 (Comprehensive Practice)

  • 05. 시각화 종합 실습
    • ToothGrowth 데이터셋 분석
    • 히스토그램, 산점도, 상자 그림, 바이올린 플롯, 히트맵 실습

10.6. 파이썬 기본기 (Python Basics)

  • 06. 파이썬 GUI 프로그래밍
    • Tkinter로 윈도우 창 만들기
    • 위젯 사용법: Label, Button, Entry
    • 배치하기: pack(), grid()



정리 (Summary)

이번 주에는 데이터를 구하고 그려보는 실전 능력을 키웠습니다. 또한 파이썬의 Tkinter를 이용해 나만의 GUI 프로그램을 만드는 재미도 느껴보았습니다. 다음 주부터는 이제 기계학습(Machine Learning)의 세계로 진입하여, 데이터를 통해 미래를 예측하는 모델을 만들어 보겠습니다.
