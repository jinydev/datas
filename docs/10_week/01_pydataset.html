10주차 1강: 다양한 예제 데이터 (PyDataset)

학습목표: 데이터 분석을 연습할 때 필요한 질 좋은 예제 데이터셋을 PyDataset 라이브러리를 통해 손쉽게 불러오는 방법을 익힙니다.

10.1.1. PyDataset 소개 및 설치



1. PyDataset이란?

R 언어에는 데이터 분석 연습을 위한 수많은 데이터셋(예: Titanic, Iris)이 내장되어 있습니다. 파이썬에서도 이 데이터들을 똑같이 쓸 수 있게 해주는 고마운 라이브러리입니다.



2. 설치하기

터미널이나 주피터 노트북에서 설치합니다.

!pip install pydataset



10.1.2. 사용법

1. 데이터 목록 확인: data()

아무 인자 없이 data()를 호출하면 사용 가능한 모든 데이터셋의 목록을 보여줍니다.

from pydataset import data

# 데이터셋 목록 출력 (id와 설명 확인)
print(data())



2. 데이터 불러오기: data('id')

원하는 데이터셋의 ID를 문자열로 넣어주면 데이터프레임(DataFrame) 형태로 바로 돌려줍니다.

# 타이타닉 데이터 불러오기
titanic = data('titanic')

print(titanic.head())
#   class     age sex survived
# 1   1st  adults man      yes
# 2   1st  adults man      yes
# ...



10.1.3. 주요 예제 데이터 추천

연습할 때 자주 쓰이는 “국민 데이터셋”들입니다.

ID 설명 크기 용도
iris 붓꽃의 꽃잎/꽃받침 길이 데이터 150행 분류(Classification)
mtcars 1974년 자동차 디자인/성능 데이터 32행 회귀(Regression)
Titanic 타이타닉호 승객 생존 여부 1313행 분류/상관분석
tips 식당에서의 팁(Tip) 데이터 244행 시각화/그룹분석 
# 팁 데이터 불러와서 정보 확인하기
tips = data('tips')
tips.info()



정리 (Summary)

이 강의에서 배운 핵심 내용을 요약해 봅시다.

  • [핵심 1]: !pip install pydataset으로 설치하면 R의 방대한 예제 데이터를 파이썬에서 쓸 수 있습니다.
  • [핵심 2]: data()로 목록을 보고, data('이름')으로 데이터프레임을 생성합니다.
  • [핵심 3]: iris, mtcars, tips 등의 데이터는 앞으로의 실습에서 계속 활용될 것입니다.
