## 10.1.2. 사용법

### 1. 데이터 목록 확인: `data()`
아무 인자 없이 `data()`를 호출하면 사용 가능한 모든 데이터셋의 목록을 보여줍니다.

```python
from pydataset import data

# 데이터셋 목록 출력 (id와 설명 확인)
print(data()) 
```


<br>

---

<br>

### 2. 데이터 불러오기: `data('id')`
원하는 데이터셋의 ID를 문자열로 넣어주면 **데이터프레임(DataFrame)** 형태로 바로 돌려줍니다.

```python
# 타이타닉 데이터 불러오기
titanic = data('titanic')

print(titanic.head())
#   class     age sex survived
# 1   1st  adults man      yes
# 2   1st  adults man      yes
# ...
```

## 10.1.3. 주요 예제 데이터 추천

연습할 때 자주 쓰이는 "국민 데이터셋"들입니다.

| ID          | 설명                             | 크기   | 용도                 |
| :---------- | :------------------------------- | :----- | :------------------- |
| **iris**    | 붓꽃의 꽃잎/꽃받침 길이 데이터   | 150행  | 분류(Classification) |
| **mtcars**  | 1974년 자동차 디자인/성능 데이터 | 32행   | 회귀(Regression)     |
| **Titanic** | 타이타닉호 승객 생존 여부        | 1313행 | 분류/상관분석        |
| **tips**    | 식당에서의 팁(Tip) 데이터        | 244행  | 시각화/그룹분석      |

```python
# 팁 데이터 불러와서 정보 확인하기
tips = data('tips')
tips.info()
```
