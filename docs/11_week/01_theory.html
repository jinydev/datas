11주차 1강: 기계가 학습을 한다고? (ML Overview)

11.1.1. 머신러닝(Machine Learning)이란?

기존 프로그래밍 (Rule-based): 개발자가 규칙을 하나하나 입력합니다. “IF 레벨 < 10 THEN ‘초보자’”

머신러닝 (Data-driven): 기계에게 데이터(문제와 정답)를 주고 규칙을 스스로 찾게 합니다. “이 유저들의 데이터를 줄 테니까, 초보자와 고수를 구별하는 규칙을 찾아봐!”

11.1.2. 지도학습 vs 비지도학습

머신러닝은 크게 두 가지로 나뉩니다. 정답지(Target)가 있느냐 없느냐의 차이입니다.

  1. 지도학습 (Supervised Learning): 선생이 옆에서 지도해 줍니다.
    • 회귀 (Regression): 숫자를 예측합니다. (예: 내일 주식 가격, 아파트 가격)
    • 분류 (Classification): 카테고리를 맞춥니다. (예: 스팸 메일 여부, 개 vs 고양이)
  2. 비지도학습 (Unsupervised Learning): 선생 없이 스스로 공부합니다.
    • 군집화 (Clustering): 비슷한 것끼리 묶기. (예: 고객 세분화, 뉴스 토픽 분류)

11.1.3. Scikit-learn (사이킷런) 소개

파이썬 머신러닝의 표준 라이브러리입니다. 데이터 전처리부터 모델 학습, 평가까지 모든 도구가 들어있는 만능 공구함입니다.
