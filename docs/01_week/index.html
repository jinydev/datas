1주차: 데이터 분석 개요 및 환경 설정

학습 목표 (Learning Objectives)

  1. 데이터 분석의 이해: 데이터 분석이 무엇이며, 왜 파이썬을 사용하는지 이해합니다.
  2. 개발 환경 구축: Google Colab과 VS Code(Antigravity)를 활용하여 코딩 환경을 설정합니다.
  3. 파이썬 기초: 변수, 자료형, 기본 연산 등 파이썬의 핵심 문법을 익힙니다.

강의 내용 (Course Content)

01. 데이터 분석과 파이썬

1.1.1. 데이터 분석이란?

  • 데이터 분석의 정의와 필요성.
  • 통찰력(Insight)과 의사결정.

1.1.2. 왜 파이썬인가?

  • 파이썬의 장점 (쉬운 문법, 강력한 라이브러리).
  • 다른 언어(C, Java)와의 비교.

1.1.3. 데이터 분석의 과정

  • 문제 정의부터 평가까지의 5단계 흐름.
  • 데이터 전처리(요리 재료 손질)의 중요성.

02. 개발 환경 설정

1.2.1. Google Colab 설정

  • 별도의 설치 없이 웹 브라우저에서 바로 파이썬 실습 가능.
  • 간단한 실습이나 GPU가 필요할 때 추천.

1.2.2. VS Code 설정

  • 현업에서 가장 많이 쓰는 강력한 코드 에디터.
  • 로컬 환경에서 전문적인 개발을 원할 때 필수.

1.2.3. 바이브 코딩 & 안티그래비티

  • AI(Antigravity)와 함께하는 짝 프로그래밍(Pair Programming).
  • 최신 AI 코딩 트렌드인 Vibe Coding 경험하기.

03. 파이썬 기초 1 (Python Basics 1)

  • 파이썬의 시작: Hello World 출력하기.
  • 변수와 자료형: 값을 담는 그릇 이해하기 (int, float, str).
  • 기본 연산: 컴퓨터로 더하기, 빼기, 곱하기, 나누기.
  • 문자열 다루기: 문자를 연결하고 조작하는 방법.

04. 데이터 시각화 맛보기 (Matplotlib Teaser)

  • 변수와 리스트 (데이터 담는 상자와 가방).
  • 데이터 시각화 맛보기: 단 4줄로 그래프 그려보기.
  • 파이썬으로 할 수 있는 강력한 기능 체험.

1.3. 정리 (Summary)

이번 주차에는 데이터 분석의 큰 그림을 이해하고, 앞으로의 여행을 위한 베이스캠프(환경 설정)를 구축했습니다.

  1. 데이터 분석: 원석(데이터)을 가공하여 보석(인사이트)을 만드는 과정입니다. DIKW 피라미드와 5단계 분석 과정을 배웠습니다.
  2. 환경 설정:
    • Colab: 설치가 필요 없는 간편한 웹 환경.
    • VS Code: 현업에서 사용하는 강력한 로컬 에디터.
    • Antigravity (Vibe Coding): AI와 대화하며 코딩하는 새로운 패러다임.
  3. 파이썬 기초: 프로그래밍의 첫걸음인 print, 변수, 기본 자료형(int, str)을 익혔습니다.
  4. 시각화 맛보기: Matplotlib를 사용하여 데이터를 그래프로 표현하는 즐거움을 맛보았습니다.

다음 주차(Week 2)에는 파이썬의 더 강력한 기능(리스트, 제어문)을 배우고, 시각화 기술을 한 단계 업그레이드합니다. 준비되셨나요? 🚀
