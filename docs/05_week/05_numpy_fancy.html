5주차 5강: 원하는 것만 쏙쏙 (Fancy Indexing)

6.2.1. 인덱스 배열 사용하기

슬라이싱은 “어디부터 어디까지” 연속된 부분만 자를 수 있었습니다. 하지만 팬시 인덱싱을 쓰면 띄엄띄엄 있는 데이터도 마음대로 가져올 수 있습니다.

Fancy Indexing: 인덱스가 담긴 리스트나 배열을 전달하여 조회하는 방식입니다.



6.2.1.1. 시나리오: 즐겨찾는 아이템 선택

인벤토리에 아이템이 잔뜩 있지만, 내가 자주 쓰는 0번, 2번, 5번 아이템만 꺼내고 싶습니다.

import numpy as np

inventory = np.array(['물약', '검', '방패', '활', '지도', '열쇠', '보석'])

# 원하는 인덱스 목록
picks = [0, 2, 5]

# 팬시 인덱싱
my_items = inventory[picks]
print("선택한 아이템:", my_items)
# ['물약' '방패' '열쇠']



6.2.2. 값 수정하기

가져오는 것뿐만 아니라, 특정 위치의 값을 한 번에 수정할 수도 있습니다. 선택한 아이템을 모두 ‘빈 슬롯’으로 바꿔봅시다.

inventory[picks] = '빈 슬롯'
print(inventory)
# ['빈 슬롯' '검' '빈 슬롯' '활' '지도' '빈 슬롯' '보석']

이 기능을 활용하면 데이터에서 이상치(Outlier)나 결측치를 특정 값으로 한 번에 메우는 전처리 작업을 쉽게 할 수 있습니다.



정리 (Summary)

이 강의에서 배운 핵심 내용을 요약해 봅시다.

  • [핵심 1]: 팬시 인덱싱(Fancy Indexing)은 정수 배열(인덱스 리스트)을 사용해 원하는 위치의 데이터를 쏙쏙 뽑아옵니다.
  • [핵심 2]: 불연속적인 데이터를 모으거나, 데이터의 순서를 바꿀 때 유용합니다.
  • [핵심 3]: 인덱싱을 활용해 특정 위치의 값을 수정(할당)하는 것도 가능합니다.
