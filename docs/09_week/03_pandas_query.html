9주차 3강: 데이터 검색과 필터링 (Query & Filtering)

학습목표: 수많은 데이터 중에서 내가 원하는 조건에 맞는 행(Row)만 쏙쏙 골라내는 고급 필터링 기술을 익힙니다.

9.3.1. 불리언 인덱싱 (Boolean Indexing)

Numpy에서 배운 것과 똑같습니다. True/False 마스크를 만들어 데이터를 걸러냅니다.

import pandas as pd

data = {
    '이름': ['철수', '영희', '민수', '지수'],
    '나이': [15, 17, 15, 18],
    '성별': ['남', '여', '남', '여']
}
df = pd.DataFrame(data)

# 나이가 16살 이상인 학생만 뽑기
mask = df['나이'] >= 16
result = df[mask]

print(result)
#    이름  나이 성별
# 1  영희   17  여
# 3  지수   18  여



복합 조건 (&, |)

  • & (AND): 그리고 (둘 다 만족)
  • | (OR): 또는 (하나라도 만족)
# '여자'이면서 '나이가 18세 이상'인 사람
cond = (df['성별'] == '여') & (df['나이'] >= 18)
print(df[cond])



9.3.2. 쿼리 함수: query()

조건식이 복잡해지면 df[...] 문법은 괄호 때문에 읽기 어렵습니다. query()를 쓰면 SQL처럼 문자열로 조건을 깔끔하게 적을 수 있습니다.

# 나이가 16세 이상인 사람 (따옴표 안에 조건식!)
print(df.query('나이 >= 16'))

# 성별이 여자인 사람 (문자열 비교는 따옴표 주의)
print(df.query("성별 == '여'"))

# 변수 사용하기 (@ 붙이기)
min_age = 16
print(df.query('나이 >= @min_age'))



9.3.3. 포함 여부 확인: isin()

“이름이 철수거나 영희인 사람”을 찾을 때, OR 연산자를 여러 번 쓰면 힘들죠? isin()을 쓰면 리스트에 있는 값인지 한 번에 확인해 줍니다.

# OR 연산자 사용 (비효율적)
# df[(df['이름'] == '철수') | (df['이름'] == '영희')]

# isin 사용 (효율적)
target_names = ['철수', '영희']
print(df[df['이름'].isin(target_names)])



정리 (Summary)

이 강의에서 배운 핵심 내용을 요약해 봅시다.

  • [핵심 1]: 불리언 인덱싱은 조건식의 결과(True/False)를 이용해 데이터를 필터링하는 가장 기본적인 방법입니다.
  • [핵심 2]: query() 함수를 사용하면 조건식을 문자열 형태로 직관적이고 깔끔하게 작성할 수 있습니다.
  • [핵심 3]: 여러 개의 찾고 싶은 값이 있을 때는 isin() 함수를 사용하는 것이 훨씬 효율적입니다.
