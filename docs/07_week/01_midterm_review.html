7주차 1강: 전반기 총정리 (Review)

7.1.1. 파이썬과 분석 환경 (Week 1)

  • 파이썬: 쉽고 강력한 데이터 분석의 만능 도구(Universal Language).
  • 환경: Colab(클라우드), VS Code(로컬), Antigravity(AI 페어 프로그래밍).
  • 기초 문법: 변수(상자), 리스트(가방), 딕셔너리(이름표 달린 금고).



7.1.2. 데이터 시각화 (Week 2)

  • Matplotlib: 파이썬의 미술 도구.
  • 주요 그래프:
    • plot(): 선 그래프 (추세).
    • bar(): 막대 그래프 (비교).
    • scatter(): 산점도 (상관관계).



7.1.3. Numpy의 기초 (Week 3)

  • List vs Array: 리스트는 승용차(범용), 배열은 화물트럭(고속/대량).
  • 생성 함수: array(), zeros(), ones(), arange(), linspace().



7.1.4. 배열 형태 다루기 (Week 4)

  • Structure: reshape, flatten.
  • Combination: concatenate, vstack, hstack.
  • Splitting: split, vsplit, hsplit.
  • Slicing & Indexing: 데이터를 자르고 콕 집어내기.



7.1.5. 연산과 논리 (Week 5)

  • Operations: 사칙연산 (Element-wise).
  • Broadcasting: 크기가 달라도 OK.
  • Aggregation: sum, mean.
  • Filtering: Boolean Indexing (조건), Fancy Indexing (선택).



7.1.6. 고급 도구 (Week 6)

  • Exceptions: try-except로 에러 방어.
  • Random: rand, randint로 시뮬레이션.
  • Modules: math, numpy 등 외부 도구 활용.
  • Advanced Stacking: stack, column_stack.
