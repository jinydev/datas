# 8주차 3강: 데이터 정찰하기 (Inspection)

데이터를 불러왔으면, 가장 먼저 "어떤 데이터인지" 파악해야 합니다. 이를 **EDA(탐색적 데이터 분석)**의 시작이라고 합니다.

## 8.3.1. 앞뒤 데이터 확인하기 (`head`, `tail`)

데이터가 100만 줄인데 `print(df)`를 하면 화면이 꽉 차서 보기 힘듭니다.
`head()`는 상위 5개, `tail()`은 하위 5개만 보여줍니다.

```python
# 상위 5개 출력
print(df.head())

# 하위 2개만 출력
print(df.tail(2))
```

## 8.3.2. 요약 정보 확인하기 (`info`)

데이터의 크기, 컬럼명, 자료형(Type), 결측치(Null) 등을 한눈에 보여주는 **종합 상황판**입니다.

```python
print(df.info())
```

**[해석 포인트]**
- `Non-Null Count`: 결측치가 없는 데이터 개수. (이게 전체 개수보다 적으면 결측치가 있다는 뜻!)
- `Dtype`: `int`(정수), `float`(실수), `object`(문자열) 등 자료형 확인.

## 8.3.3. 기초 통계량 확인하기 (`describe`)

숫자 데이터들의 통계 정보를 요약해 줍니다. 엑셀의 "데이터 분석" 기능과 같습니다.

```python
print(df.describe())
```

- `count`: 개수
- `mean`: 평균 (레벨 평균은 얼마인가?)
- `std`: 표준편차
- `min`/`max`: 최솟값/최댓값
- `25%`, `50%`, `75%`: 4분위수 (하위 25%, 중간값, 상위 25%)

> **팁**: `describe()` 한 방이면 "우리 파티의 평균 레벨은 11.25이고, 최고 레벨은 15구나!"라고 1초 만에 파악할 수 있습니다.
