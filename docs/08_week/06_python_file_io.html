8주차 6강: 파이썬 파일 입출력 (File I/O)

학습목표: 판다스를 쓰지 않고도 파이썬의 기본 기능만으로 파일을 읽고 쓰는 방법을 익힙니다. 텍스트 로그를 남기거나 간단한 설정 파일을 다룰 때 필수적입니다.

8.6.1. 파일 열고 닫기 (open, close)

파일을 다룰 때는 ‘열기 -> 쓰기/읽기 -> 닫기’ 의 3단계를 꼭 지켜야 합니다.

  • open(파일명, 모드): 파일을 엽니다.
  • close(): 파일을 닫습니다. (안 닫으면 파일이 깨지거나 메모리 누수가 발생할 수 있어요!)

주요 모드 (Mode)

  • 'w' (Write): 쓰기 모드. 파일이 없으면 만들고, 있으면 내용을 다 지우고 새로 씁니다.
  • 'r' (Read): 읽기 모드. 파일이 없으면 에러가 납니다. (기본값)
  • 'a' (Append): 추가 모드. 기존 내용 뒤에 이어 씁니다.
# 1. 파일 열기 (쓰기 모드)
f = open("hello.txt", "w")

# 2. 파일 쓰기
f.write("Hello, Python!\n")
f.write("파일 입출력 연습 중입니다.")

# 3. 파일 닫기
f.close()



8.6.2. 자동으로 닫아주는 with 문

매번 close()를 쓰는 건 귀찮고 까먹기 쉽습니다. with 문을 쓰면 블록이 끝날 때 알아서 닫아줍니다. (권장 방법!)

# with 블록을 벗어나면 자동으로 close()가 호출됨
with open("hello.txt", "r") as f:
    text = f.read()
    print(text)



8.6.3. 파일 읽기 삼총사

1. read()

파일의 모든 내용을 하나의 문자열로 통째로 읽어옵니다.

2. readline()

한 줄만 읽어옵니다. 반복문과 함께 쓰면 메모리를 아끼며 큰 파일을 읽을 수 있습니다.

with open("hello.txt", "r") as f:
    while True:
        line = f.readline()
        if not line: break # 더 읽을 줄이 없으면 종료
        print(line.strip()) # 줄바꿈 문자 제거

3. readlines()

모든 줄을 읽어서 리스트(List)로 반환합니다.

with open("hello.txt", "r") as f:
    lines = f.readlines()
    print(lines) 
    # ['Hello, Python!\n', '파일 입출력 연습 중입니다.']



정리 (Summary)

이 강의에서 배운 핵심 내용을 요약해 봅시다.

  • [핵심 1]: 파일 입출력의 기본 모드인 'r'(읽기), 'w'(덮어쓰기), 'a'(이어쓰기)를 구분해야 합니다.
  • [핵심 2]: with open(...) as f: 구문을 사용하면 close()를 신경 쓰지 않아도 돼서 안전하고 편리합니다.
  • [핵심 3]: 파일 전체를 읽을 땐 read(), 줄 단위로 처리할 땐 readline()이나 readlines()를 사용합니다.
