2주차: 데이터 시각화 기초 (Matplotlib)

학습 목표 (Learning Objectives)

  1. 파이썬 기초 정복: 변수, 리스트, 딕셔너리, 제어문 등 데이터 분석에 필수적인 파이썬 문법을 마스터합니다.
  2. 시각화의 필요성: 데이터를 숫자로만 보지 않고, 그림으로 표현해야 하는 이유를 이해합니다.
  3. Matplotlib 기초: 파이썬의 대표적인 시각화 라이브러리인 Matplotlib의 기본 사용법을 익힙니다.
  4. 그래프 그리기: 선 그래프, 막대그래프, 산점도 등 다양한 형태의 그래프를 직접 그려봅니다.

강의 내용 (Course Content)

01. 파이썬 기초 2 (Python Basics 2)

  • 리스트와 딕셔너리: 여러 데이터를 한 번에 다루는 법.
  • 제어문(Control Flow): 조건문(if)과 반복문(for)으로 프로그램의 논리 만들기.
  • 실습: 간단한 텍스트 전투 시스템 만들기.

02. 데이터 시각화란? (Introduction)

  • 왜 코딩보다 시각화를 먼저 배울까?
  • Matplotlib 설치 및 기본 구조 이해.
  • Figure와 Axes의 개념 (도화지와 축).

03. 기본 그래프 그리기 (Basic Plots)

  • 선 그래프 (Line Plot): 변화의 추세 확인.
  • 막대그래프 (Bar Plot): 크기 비교.
  • 산점도 (Scatter Plot): 데이터 간의 관계 파악.

04. 그래프 스타일링 (Attributes & Styling)

  • 제목, 라벨, 범례 추가하기.
  • 선 색상, 스타일, 마커 변경하기.
  • 게임 캐릭터 스탯 비교 시각화 실습.

05. 산점도 (Scatter Plot)

  • 두 변수 간의 상관관계(Correlation) 파악하기.
  • 점의 색상과 크기(Bubble Chart) 활용하기.

06. 막대그래프 (Bar Plot)

  • 범주형 데이터의 비교(Comparison).
  • 세로형(Vertical) vs 가로형(Horizontal).

07. 히스토그램 (Histogram)

  • 수치형 데이터의 분포(Distribution) 확인하기.
  • 도수분포표(Frequency Table)와 구간(Bin) 설정.

08. 피규어와 서브플롯 (Figure & Subplots)

  • Figure(전체 창)와 Axes(그래프 영역)의 해부학.
  • 여러 그래프 한 번에 그리기 (subplot, subplots, add_subplot).
  • tight_layout()으로 예쁜 레이아웃 만들기.

2.3. 정리 (Summary)

이번 주차에는 파이썬의 핵심 문법을 다지고, 데이터 시각화의 기초를 닦았습니다.

  1. 파이썬 기초 2:
    • 리스트 & 딕셔너리: 데이터를 체계적으로 관리하는 법.
    • 제어문 (if, for): 프로그램의 흐름을 조종하는 마법.
  2. 데이터 시각화 (Matplotlib):
    • 백문이 불여일견: 데이터를 숫자가 아닌 그림으로 표현해야 하는 이유.
    • 기본 그래프: 선(Trend), 산점도(Relation), 막대(Comparison) 그래프 그리기.
    • 스타일링: 제목, 축 레이블, 색상, 한글 폰트 설정으로 완성도 높이기.
